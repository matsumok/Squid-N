## 2. T0: 等価1自由度化（容量曲線）

設計書 §11.1。プッシュオーバー（P5 `PushoverResult`）の各ステップ（層変位分布 δi、ベースシア Qb）を、
1次モード相当の**代表変位 Δ1・代表加速度 A1**へ縮約する。**閉形式（明示）:**

```
記号: mi=各層質量, δi=各層水平変位（その荷重ステップ）, Qb=ベースシア, g=重力加速度
有効質量    M1* = (Σ mi·δi)² / Σ mi·δi²
代表変位    Δ1  = (Σ mi·δi²) / (Σ mi·δi)
代表加速度  A1  = Qb / M1*            （加速度の次元。Sa と同じ土俵に乗る）
→ 各ステップを (Δ1, A1) に写すと「容量曲線」（容量スペクトル）になる。
```


**DoD（T0）:** 1質点系で `M1*=m, Δ1=δ, A1=Qb/m` に一致（自明系で検算）。多質点で有効質量比が妥当。

## 3. T1: 加速度応答スペクトル（告示1457）と ADRS 変換

設計書 §11.1。地震動を周期ごとの応答加速度 `Sa(T)` で与える（告示1457号）。

**スペクトルの形（告示1457。値は原典照合リスト対象＝専門家が1回照合）:**
```
Sa(T) = Z · Gs(T) · So(T)        （Z=地域係数, Gs=表層地盤増幅, So=工学的基盤の標準スペクトル）

So(T)（安全限界・h=5%・標準。[m/s²]。★告示値＝要原典照合）:
   T < 0.16      : So = 3.2 + 30·T
   0.16 ≤ T<0.64 : So = 8.0
   0.64 ≤ T      : So = 5.12 / T
（損傷限界はこの 1/5：0.64+6T / 1.6 / 1.024/T）
```

**ADRS（加速度–変位応答スペクトル）変換**（容量曲線と同じ土俵に乗せる）:
```
Sd(T) = Sa(T) · (T / 2π)²        （周期 T ごとに (Sd, Sa) の点列にする）
```

## 4. T2: 等価粘性減衰と応答低減 Fh

設計書 §11.1。建物が塑性化すると等価減衰 h が増え、応答スペクトルが下がる。

**閉形式（明示）:**
```
応答低減率   Fh = 1.5 / (1 + 10·h)      （告示。h=等価粘性減衰定数）
等価粘性減衰 h = h0 + γ·heq
   h0  = 弾性減衰（RC 0.05 等）
   heq = 1/(4π)·(消費エネルギー/弾性ひずみエネルギー)  ≈ (1/π)(1 − 1/√μ) 系（μ=塑性率）
   γ   = 履歴型による補正係数（外部設定）
低減後スペクトル: Sa_d = Fh · Sa,  Sd_d = Fh · Sd
```

**DoD（T2）:** `Fh = 1.5/(1+10h)`（h=0.05→1.0, h=0.15→0.6）。塑性率 μ 増で h 増・Fh 減。

## 5. T3: 性能点（容量スペクトル法）

設計書 §11.1。**容量曲線（§2、A1–Δ1）と、減衰低減した需要スペクトル（§3–§4、Sa_d–Sd_d）の交点＝
性能点（その地震に対する応答）。**

反復で解く（応答が減衰に依存するため）:
```
1. 仮の性能点 → 塑性率 μ → 等価減衰 h（§4）→ Fh
2. Fh で需要スペクトルを低減 → 容量曲線との交点を更新
3. 交点が安定するまで反復（割線周期 Teq での需要＝容量）。
交点の (Δ1, A1) が応答。代表変位 Δ1 → 各層変位 δi に戻して層間変形・部材応答を評価。
```

## 6. T4: 損傷限界・安全限界の判定

設計書 §11.1。

- **損傷限界（中地震・稀）**：損傷限界用スペクトル（§3 の 1/5）に対する性能点で、層間変形角・部材応力が
  損傷限界内（弾性〜軽微）であること。
- **安全限界（大地震・極稀）**：安全限界用スペクトルに対する性能点で、層間変形角・部材塑性率が
  安全限界内（終局に余裕）であること。

**DoD（T4）:** 両限界の性能点から層間変形角・部材応答を出し、限界値と比較・判定する。

## 8. 数式・補足（ジュニア向け）

### 8.1 容量スペクトル法の全体像（→ §2,§5）
建物の「押す力 vs 変形」（容量曲線）と、地震の「周期 vs 応答」（需要スペクトル）を**同じ ADRS 平面
（Sa–Sd）**に描き、その**交点＝この地震での応答（性能点）**とする。塑性化で減衰が増え需要が下がるので、
減衰を仮定→低減→交点更新を反復する。

### 8.2 等価1自由度化（→ §2）
多層建物の押し倒し挙動を、1次モード形で代表させた1質点（有効質量 M1*・代表変位 Δ1）に縮約する。
これで地震の応答スペクトル（1自由度の応答）と直接比較できる。

### 8.3 等価粘性減衰と Fh（→ §4）
塑性履歴ループの面積（消費エネルギー）が大きいほど等価減衰 h が大きく、応答が下がる。告示は
`Fh=1.5/(1+10h)` で低減する（h=5%で 1.0、増えるほど小さく）。

### 8.4 限界耐力計算と保有水平耐力ルートの違い（→ §0.1）
保有水平耐力（P7）は「終局耐力 ≥ 必要耐力」を Ds で確認する**耐力ベース**。限界耐力計算は
応答スペクトルで**変形（応答）ベース**に性能を評価する。本ソフトは両ルートを提供する。
