## 4. T1: 剛性率 Rs・層間変形角（P2 層間変形から）

設計書 §11.1。**剛性率は分岐なし**（P2 が層間変形を直接出すので計算可能）。

```
層 i の変形角  rs_i = δ_i / h_i        （δ_i＝層間変位, h_i＝階高。P2 §3/§6 出力）
層の剛さ指標   Ks_i = 1 / rs_i = h_i / δ_i
剛性率         Rs_i = Ks_i / mean(Ks)   （規定：Rs ≥ 0.6）
```

**規定値（告示1792／施行令。法令＝公有。下記で確定）:**
- **剛性率の規定:** `Rs ≥ 0.6`（下回ると §5.3 の Fs で割増）。
- **層間変形角の規定（令82条の2）:** 原則 `δ/h ≤ 1/200`。帳壁・仕上げ等に著しい損傷の恐れがなければ
  `1/120` まで緩和可。

**確定値の例（計算可能・テスト用）:** 3層で `δ/h = [1/200, 1/150, 1/250]`
→ `Ks = [200, 150, 250]`、`mean = 200` → `Rs = [1.0, 0.75, 1.25]`（全層 ≥0.6）。
- 層間変形角 `δ/h` を `1/200`（緩和時 `1/120`）と比較して出す。

**DoD（T1）:** Rs が手計算一致（厳密寄り）。層間変形角が規定値と比較される。

## 5. T2/T3: 偏心率 Re（剛心＝D値法）と Fes

設計書 §11.1。**偏心率は剛心が要る → v1 は D値法（略算）で剛心を出す**（規準例題に整合）。

### 5.1 剛心（D値法）

**D値（武藤）の閉形式（文献転記不要）:**
```
柱の標準水平剛性   Kc0 = 12·E·Ic / h³          （両端固定柱、h=階高、Ic=柱断面二次モーメント）
柱の剛比           kc  = Ic / h                 （部材剛比の基準）
梁との剛比          k̄  = (ΣKb) / (2·kc)         （ΣKb=その柱頭・柱脚に取り付く梁の剛比和）
剛比補正係数        a  = k̄ / (2 + k̄)            （一般階・中間柱。最下階・端柱は別式）
D値               D  = a · Kc0
剛心              Xs = Σ(Di·xi) / Σ Di,   Ys = Σ(Di·yi) / Σ Di
```
- 重心は質量分布から。`ex = |Xg − Xs|`, `ey = |Yg − Ys|`（§5.2 で使用）。
- 最下階柱・回転拘束のない柱脚等の `a` の別式は補正係数表（武藤の D値法。公知）に従う。

### 5.2 偏心率

**告示1792 の標準定義（下記で確定）。** `x̄i, ȳi` は剛心からの各構面の距離、`Kxi, Kyi` は各構面の
X/Y 方向水平剛性:

```
偏心距離     ex = |Xg − Xs|,   ey = |Yg − Ys|        （g=重心, s=剛心）
ねじり剛性   KR = Σ(Kxi · ȳi²) + Σ(Kyi · x̄i²)         （剛心まわり）
弾力半径     rex = √( KR / ΣKxi ),   rey = √( KR / ΣKyi )
偏心率       Rex = ey / rex,   Rey = ex / rey          （規定：Re ≤ 0.15）
```
（X方向加力時の偏心は直交方向距離 ey が効くので `Rex = ey/rex`。Y方向は対称。）

**確定値の例（X方向・計算可能・テスト用）:** X方向に構面2つ、水平剛性 `Kx=[100, 300]`、位置
`x=[0, 6000]`、Y偏心 `ey=1500` とする → 剛心 `Xs = (100·0 + 300·6000)/400 = 4500`、重心 `Xg=3000`、
`ex = 1500`。`rex = √(KR/ΣKx)` に代入して `Rex = ey/rex`。

### 5.3 Fes（形状係数）

**告示1792 の算式（下記で確定）:**
```
Fs（剛性率より）: Rs ≥ 0.6 → Fs = 1.0
                  Rs < 0.6 → Fs = 2.0 − Rs/0.6          （Rs=0.6 で 1.0、Rs=0.3 で 1.5）
Fe（偏心率より）: Re ≤ 0.15 → Fe = 1.0
                  Re > 0.15 → Fe = 1.0 + 0.5·(Re − 0.15)/0.15   （Re=0.30 で 1.5、以降 1.5 で頭打ち）
Fes = Fs · Fe                       （層ごと）
```

```rust
pub fn fs(rs: f64) -> f64 { if rs >= 0.6 { 1.0 } else { 2.0 - rs / 0.6 } }
pub fn fe(re: f64) -> f64 { if re <= 0.15 { 1.0 } else { (1.0 + 0.5*(re - 0.15)/0.15).min(1.5) } }
```

**DoD（T2/T3）:** 剛心・偏心距離 e が手計算一致（厳密寄り）。`fs(0.3)=1.5`, `fe(0.30)=1.5`,
`fes = fs·fe` が告示算式どおり（厳密 1e-9）。

## 9. 数式・理論補足（ジュニア向け）

### 9.1 二次設計の枠組み（→ §2,§3）
一次設計（許容応力度、P3）で中地震 OK を確認後、大地震に対し**建物が終局までに保有する水平耐力 Qu**が
**必要量 Qun を上回る**ことを確認するのが二次設計。`Qun = Ds·Fes·Qud`。Ds は「粘り（変形能力）が
あるほど小さくできる」低減係数、Fes は「剛性・重心のばらつきによる割増」係数。

### 9.2 剛性率 Rs（→ §4）
各層の「硬さ」を層間変形角の逆数で測り、全層平均との比をとる。ある層だけ柔らかい（Rs 小）と
そこに変形が集中して危険。規定は Rs ≥ 0.6（下回ると Fs で割増）。

### 9.3 偏心率 Re（→ §5）
重心（質量中心）と剛心（剛性中心）がずれると、地震時に建物がねじれる。ずれ e を弾力半径 re で
正規化したのが Re。規定は Re ≤ 0.15（超えると Fe で割増）。剛心は v1 では D値法（柱剛性の重み重心）で略算。

### 9.4 Ds（構造特性係数）（→ §7）
部材が粘り強く（曲げ降伏先行・幅厚比が小さい）全体崩壊する架構ほど Ds は小さく（必要耐力が小さくて
よい）、脆い破壊（せん断先行・幅厚比大）や層崩壊だと Ds は大きい。部材ランク（FA〜FD）と崩壊機構から
告示の表で決まる。自動分類は提案で、技術者が上書きできる。
