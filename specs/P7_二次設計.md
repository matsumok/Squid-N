## 2. T0: 三つのせん断と出力型（★最初に整理）

> **⚠ 3つの「せん断力」を取り違えない（ジュニア最頻ミス）。** 層ごとに `Qu ≥ Qun` を確認する:
> - **Qu（保有水平耐力＝耐力）** = **P5 PushoverResult** の崩壊機構形成時の層せん断。
> - **Qud（地震時層せん断＝需要）** = **P2 Ai分布の二次設計用層せん断**（`C0 = 1.0`。一次設計は 0.2。
>   P2 §5.2 の `c0` を 1.0 にして算定）。

## 4. T1: 剛性率 Rs・層間変形角（P2 層間変形から）

設計書 §11.1。**剛性率は分岐なし**（P2 が層間変形を直接出すので計算可能）。

```
層 i の変形角  rs_i = δ_i / h_i        （δ_i＝層間変位, h_i＝階高。P2 §3/§6 出力）
層の剛さ指標   Ks_i = 1 / rs_i = h_i / δ_i
剛性率         Rs_i = Ks_i / mean(Ks)   （規定：Rs ≥ 0.6）
```

> **⚠ 要検証:** 剛性率・偏心率・Fes・Ds の定義値・割増表は **昭和55年建設省告示第1792号**（**番号・年
> 要確認**）。表は外部データ化し出典明記。

**確定値の例（計算可能・テスト用）:** 3層で `δ/h = [1/200, 1/150, 1/250]`
→ `Ks = [200, 150, 250]`、`mean = 200` → `Rs = [1.0, 0.75, 1.25]`（全層 ≥0.6）。
- 層間変形角 `δ/h` の規定（既定 1/200 等）チェックも出す。

**DoD（T1）:** Rs が手計算一致（厳密寄り）。層間変形角が規定値と比較される。

## 5. T2/T3: 偏心率 Re（剛心＝D値法）と Fes

設計書 §11.1。**偏心率は剛心が要る → v1 は D値法（略算）で剛心を出す**（規準例題に整合）。

### 5.1 剛心（D値法）

- 各柱の **D値**（部材剛性から略算した水平剛性指標）を求め、**剛心 = D値で重み付けした位置の重心**。
- 重心は質量分布から。`e = |重心 − 剛心|`（X・Y 各方向）。

```rust
// sc-design-jp/src/eccentricity.rs
pub struct StoryCenters { pub center_of_mass: [f64;2], pub center_of_rigidity: [f64;2] }
/// 層の各柱 D値から剛心を、質量から重心を求める（D値法・略算）。
pub fn story_centers(model: &sc_core::model::Model, story: sc_core::ids::StoryId) -> StoryCenters { todo!() }
```

> **将来（精算）:** 剛床マスター節点（P0/P1）の層 3×3 剛性（Fx,Fy,Mz ↔ ux,uy,θz）から剛心を導く
> 精算を後で追加できる（単一モデル思想）。v1 は D値法。

### 5.2 偏心率

```
偏心距離     e（重心−剛心）
弾力半径     re = √( KR / ΣK )   （KR＝層のねじり剛性 = Σ Ki·di², di＝剛心からの距離）
偏心率       Re = e / re          （規定：Re ≤ 0.15）
```

> **⚠ 要検証:** `re`（弾力半径）と `Re` の正確な定義・方向別の扱いは告示1792を一次資料に。

**確定値の例（剛心・e は計算可能・テスト用）:** X方向に構面2つ、水平剛性 `K=[100, 300]`、位置
`x=[0, 6000]` → 剛心 `x = (100·0 + 300·6000)/400 = 4500`。重心 `x=3000`（中央）→ `e = 1500`。
（`re`・`Re` は上式に代入。`re` 定義は要検証。）

### 5.3 Fes（形状係数）

```
Fs（剛性率より）: Rs ≥ 0.6 → 1.0、未満は表で 1.0〜2.0
Fe（偏心率より）: Re ≤ 0.15 → 1.0、超過は表で 1.0〜1.5
Fes = Fs · Fe                       （層ごと。表は外部データ・告示1792）
```

**DoD（T2/T3）:** 剛心・偏心距離 e が手計算一致（厳密寄り）。Fs・Fe・Fes が規準表どおり（外部データ照合）。

## 9. 数式・理論補足（ジュニア向け）

### 9.1 二次設計の枠組み（→ §2,§3）
一次設計（許容応力度、P3）で中地震 OK を確認後、大地震に対し**建物が終局までに保有する水平耐力 Qu**が
**必要量 Qun を上回る**ことを確認するのが二次設計。`Qun = Ds·Fes·Qud`。Ds は「粘り（変形能力）が
あるほど小さくできる」低減係数、Fes は「剛性・重心のばらつきによる割増」係数。

### 9.2 剛性率 Rs（→ §4）
各層の「硬さ」を層間変形角の逆数で測り、全層平均との比をとる。ある層だけ柔らかい（Rs 小）と
そこに変形が集中して危険。規定は Rs ≥ 0.6（下回ると Fs で割増）。

### 9.3 偏心率 Re（→ §5）
重心（質量中心）と剛心（剛性中心）がずれると、地震時に建物がねじれる。ずれ e を弾力半径 re で
正規化したのが Re。規定は Re ≤ 0.15（超えると Fe で割増）。剛心は v1 では D値法（柱剛性の重み重心）で略算。

### 9.4 Ds（構造特性係数）（→ §7）
部材が粘り強く（曲げ降伏先行・幅厚比が小さい）全体崩壊する架構ほど Ds は小さく（必要耐力が小さくて
よい）、脆い破壊（せん断先行・幅厚比大）や層崩壊だと Ds は大きい。部材ランク（FA〜FD）と崩壊機構から
告示の表で決まる。自動分類は提案で、技術者が上書きできる。
