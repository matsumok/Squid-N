## 0. このフェーズについて

### 0.1 目的

線形（弾性）の骨組解析が「実モデルで商用ソフトと一致する」ところまで作る。具体的には:

- **ティモシェンコ梁要素**（せん断変形考慮、剛域・端部接合条件込みで 12×12 弾性剛性）。
- **剛域の自動算定**（§6.2.1）と**端部接合・端部ばね**（§6.2.2）、**評価断面での応力復元**（§6.2.3）。
- **パネルゾーン弾性**（§6.7：フェースモーメント変換・パネルせん断 pQ・検算）。
- **拘束の縮約**（剛床ダイアフラム／MPC／リジッドリンク、`K_red = TᵀKT`。設計書 §4.6＝**P0 から繰越**）。
- **要素 → 全体剛性のアセンブラ**（決定的組立。設計書 §5.2＝**P0 から繰越**）。
- 上記を結ぶ **最小の線形静的「検証ハーネス」**（1荷重ケースを解いて変位・部材応力を返すだけ）。

### 0.2 完了像（ゴール）

設計書 §18 の P1 完了基準を、次のテストで判定する（**すべて手計算で照合可能＝外部参照なしで自走可。**
商用ソフトは入手できれば補助）:

1. **片持ち梁の先端たわみ**が、曲げ＋せん断の理論値と一致（§11.2、相対 1e-6）。`φ→0` で
   ベルヌーイ・オイラー梁（曲げのみ）に収束。
2. **剛域ありラーメン**の材端モーメント・フェース位置モーメント・層間変形が**手計算（たわみ角法/
   slope-deflection）と一致**（§11.4。商用ソフトは入手できれば補助照合）。
3. **パネルゾーン**：添付資料『パネルゾーンの力学』(小野瀬) の数値例（ケース1: `pQc=851.135 kN`,
   `pQb=1702.273 kN`, `τc=τb=42.557/tp`）と一致（§11.5）。
4. **剛床ダイアフラム付き架構**を検証ハーネスで解いた変位・部材応力が**手計算と一致**（§11.6。
   剛床は「マスター節点へ剛体拘束」なので少数構面なら手計算可能。商用ソフトは補助）。
5. 縮約・組立は **CPU・単一スレッドでビット一致**（決定性。P0 §4.4 / R28 を `TᵀKT` まで拡張）。

### 0.4 P0 からの設計引き継ぎ・前提の差分

- 拘束の**縮約演算**は P0 でデータ構造のみ用意した（`Constraint`）。P1 で実演算 `TᵀKT` を実装する。
- アセンブラ（要素→全体）は P0 では作っていない。P1 で実装する。`squid-n-math::assemble_csc`
  （決定的トリプレット組立）を**そのまま再利用**する。
- 決定性の保証範囲を「`TᵀKT` の疎行列積も CPU 単一スレッドでビット一致」まで広げる
  （faer をシリアル固定。P0 §4.4 の注記を踏襲）。
- **層構造の制約（P0 の `xtask check-deps` が守る）:** `squid-n-core` は `squid-n-math` に依存できない
  （両者は基盤層）。したがって `TᵀKT` の疎行列積は **`squid-n-solver` に置く**（`squid-n-core`/`DofMap`
  の隣に疎行列積を置くと依存方向違反になる）。

## 2. T0: squid-n-core の Section / Material 幾何データ拡張

### 2.1 なぜ必要か

P1 の3機能が、P0 の `Section{area, iy, iz, j}` には無い**断面幾何**を要求する:

- 剛域 自動算定 `λ = D_orth/2 − D_self/4`（§6.2.1）→ 部材**せい D**。
- パネルゾーン `dc = Dc − tcs`, `db = Db − tbs`, `tp`（§6.7）→ **せい・幅・パネル板厚**。
- ティモシェンコのせん断項 `φ = 12EI/(G·As·L²)`（§6.2）→ **せん断有効断面積 As**。

これらが無いと T3/T5/T2 が着手できないため、**P1 の最初のタスク**とする。

### 2.2 方針（layering を壊さない）

設計書 §2.1 の層順は `squid-n-core, squid-n-math < squid-n-material < squid-n-section < squid-n-element`。
よって**断面の挙動計算クレート `squid-n-section` を作って `squid-n-core::Section` を置換する案は採らない**
（`squid-n-core` が `squid-n-section` に依存する循環になる）。代わりに:

- **データは `squid-n-core::Section` に幾何フィールドを足す**（core = 純データ、の原則を維持。R1）。
- せん断有効断面積 `As` などの**導出が要るものは、入力値をそのまま保持**するか、`squid-n-core` 内の
  軽量ヘルパで算出する（本格的な断面分割＝ファイバは P4 の `squid-n-section` で実装）。

### 2.3 データ拡張

```rust
// squid-n-core/src/model.rs（Section を拡張）
#[derive(Clone, Debug, serde::Serialize, serde::Deserialize)]
pub struct Section {
    pub id: SectionId,
    pub name: String,

    pub area: f64,      // 断面積 A [mm^2]

    pub iz: f64,        // 同（z 軸まわり）[mm^4]
    pub j: f64,         // ねじり定数 [mm^4]
    // --- P1 追加：幾何 ---
    pub depth: f64,     // 部材せい D（強軸方向の断面高さ）[mm]
    pub width: f64,     // 部材幅 B [mm]
    pub as_y: f64,      // せん断有効断面積（y 方向せん断）[mm^2]
    pub as_z: f64,      // せん断有効断面積（z 方向せん断）[mm^2]
    pub panel_thickness: Option<f64>, // パネル板厚 tp [mm]（柱梁接合部に関わる断面のみ）
}
```

```rust
// squid-n-core/src/model.rs（Material を拡張）
#[derive(Clone, Debug, serde::Serialize, serde::Deserialize)]
pub struct Material {
    pub id: MaterialId,
    pub name: String,
    pub young: f64,     // E [N/mm^2]
    pub poisson: f64,   // ν
    pub density: f64,   // [N·s^2/mm^4]
    pub shear: Option<f64>, // G [N/mm^2]。None なら G = E / (2(1+ν)) で算出（§2.4）
}
```

### 2.4 ヘルパ（squid-n-core）

```rust
impl Material {
    /// せん断弾性係数 G。明示値があればそれ、無ければ等方弾性式で算出。
    pub fn shear_modulus(&self) -> f64 {
        self.shear.unwrap_or_else(|| self.young / (2.0 * (1.0 + self.poisson)))
    }
}

/// 矩形断面のせん断有効断面積（せん断補正係数 k=6/5 → As = 5A/6）。
/// 入力で as_y/as_z を与えない場合の参考算出に使う。型鋼等は web 面積など別途。
pub fn rect_shear_area(area: f64) -> f64 { area * 5.0 / 6.0 }
```

> **検算用確定値（§11.1 のテストに使う矩形断面 B=200, D=400, 鋼材）:**
> `A = 200×400 = 80000 mm²`、`Iy(強軸) = B·D³/12 = 200·400³/12 = 1.0666667e9 mm⁴`、
> `As = 5A/6 = 66666.67 mm²`、`E = 205000`, `ν=0.3 → G = 205000/2.6 = 78846.15 N/mm²`。

```rust
    /// この要素が占める「全体自由度番号(GlobalDof)」の並び。組立で使う。
    fn global_dofs(&self, dof: &DofMap) -> SmallVec<[usize; 24]>;

    fn tangent_stiffness(&self, state: &ElemState, ctx: &Ctx) -> LocalMat;

    /// 現在変位における内力（非線形の釣合用）。P1 では線形スタブ（K·u）でよい。
    fn internal_force(&self, state: &ElemState, ctx: &Ctx) -> LocalVec;

    /// 状態更新。P1（線形）は no-op。非線形で実装。
    fn update_state(&mut self, _du: &LocalVec, _commit: bool, _ctx: &Ctx) {}

    fn mass_matrix(&self, opt: MassOption) -> LocalMat;
```

## 4. T2: ティモシェンコ梁 弾性剛性（12×12）

設計書 §6.2 に対応。2節点・各端6自由度＝12自由度。**せん断変形を考慮**する。

### 4.1 局所剛性（IIE / φ項）

**12×12 の全項を以下に明示する（閉形式。文献転記は不要）。** 出典は Przemieniecki / Cook（IIE・
せん断補正）。検算は §4.4 の極限（`φ→0` で Bernoulli、片持ち先端たわみ）で担保する。

定義（ローカル x = 部材軸、y/z = 断面主軸）:

```
A  = 断面積,  E = ヤング係数,  G = せん断弾性係数,  L = 可とう長
Iz = z軸まわり断面二次モーメント,  Iy = y軸まわり断面二次モーメント

【せん断有効断面積の対応（必ずこの組合せで使う）】
  z軸まわりの曲げ → たわみは y 方向 → その方向のせん断 → as_y を使う
  y軸まわりの曲げ → たわみは z 方向 → その方向のせん断 → as_z を使う

φz = 12 E Iz / (G · as_y · L^2)   ← xy面内（y方向たわみ・z軸曲げ）に使う
φy = 12 E Iy / (G · as_z · L^2)   ← xz面内（z方向たわみ・y軸曲げ）に使う
```

> **⚠ 取り違え注意:** `φz` に `as_z` を使うのは**誤り**。曲げ軸とせん断方向の対応
> （z軸曲げ ↔ y方向せん断 ↔ `as_y`）を必ず守る。§11.1 の片持ち梁は1方向のみで
> 取り違えても通ってしまうため、この対応はコードレビューで明示確認すること
> （二軸曲げが入る P2 で初めて顕在化するバグ源）。

**DOF 並び（0..11）:** `[ux_i, uy_i, uz_i, rx_i, ry_i, rz_i, ux_j, uy_j, uz_j, rx_j, ry_j, rz_j]`。
非ゼロ項（対称。`K[a][b]=K[b][a]`）:

```
■ 軸（ux_i, ux_j）:   K00 = K66 = EA/L,   K06 = −EA/L
■ ねじり（rx）:        K33 = K99 = GJ/L,   K39 = −GJ/L

■ xy面内 曲げ（uy=1,7・rz=5,11、z軸曲げ）:  az = E·Iz / ((1+φz)·L³)
   K11 = K77 = 12·az
   K17 = −12·az
   K15 = K1,11 = 6·az·L
   K57 = K7,11 = −6·az·L
   K55 = K11,11 = (4+φz)·az·L²
   K5,11 = (2−φz)·az·L²

■ xz面内 曲げ（uz=2,8・ry=4,10、y軸曲げ）:  ay = E·Iy / ((1+φy)·L³)
   K22 = K88 = 12·ay
   K28 = −12·ay
   K24 = K2,10 = −6·ay·L      ← xy面と符号が逆（ry の向きの規約）
   K48 = K8,10 = 6·ay·L
   K44 = K10,10 = (4+φy)·ay·L²
   K4,10 = (2−φy)·ay·L²
```
`φz=φy=0` を代入すると Bernoulli 梁の標準 12×12 に一致する（§4.4 で確認）。

```rust
    /// 剛域・端ばねを考慮しない「素のティモシェンコ 12×12」（ローカル系）。上の式をそのまま埋める。
    pub fn local_stiffness_raw(&self) -> crate::behavior::LocalMat {
        let (e,g,a,iy,iz,jj,l) = (self.e,self.g,self.a,self.iy,self.iz,self.j,self.length);
        let phiz = 12.0*e*iz/(g*self.as_y*l*l);
        let phiy = 12.0*e*iy/(g*self.as_z*l*l);
        let az = e*iz/((1.0+phiz)*l*l*l);
        let ay = e*iy/((1.0+phiy)*l*l*l);
        let mut k = crate::behavior::LocalMat::zeros(12);
        let mut s = |i,j,v| { k.set(i,j,v); if i!=j { k.set(j,i,v); } };
        // 軸・ねじり
        s(0,0, e*a/l); s(6,6, e*a/l); s(0,6, -e*a/l);
        s(3,3, g*jj/l); s(9,9, g*jj/l); s(3,9, -g*jj/l);
        // xy面 曲げ
        s(1,1, 12.0*az); s(7,7, 12.0*az); s(1,7, -12.0*az);
        s(1,5, 6.0*az*l); s(1,11, 6.0*az*l); s(5,7, -6.0*az*l); s(7,11, -6.0*az*l);
        s(5,5, (4.0+phiz)*az*l*l); s(11,11, (4.0+phiz)*az*l*l); s(5,11, (2.0-phiz)*az*l*l);
        // xz面 曲げ
        s(2,2, 12.0*ay); s(8,8, 12.0*ay); s(2,8, -12.0*ay);
        s(2,4, -6.0*ay*l); s(2,10, -6.0*ay*l); s(4,8, 6.0*ay*l); s(8,10, 6.0*ay*l);
        s(4,4, (4.0+phiy)*ay*l*l); s(10,10, (4.0+phiy)*ay*l*l); s(4,10, (2.0-phiy)*ay*l*l);

    }
}
```

### 5.1 剛域（RigidZone）

### 5.2 剛域長の自動算定（§6.2.1）

## 7. T5: パネルゾーン弾性（フェースモーメント・パネルせん断）

設計書 §6.7。**正準定式は添付資料『パネルゾーンの力学』(小野瀬, 2009)**。**v1 は弾性のみ**。

### 7.1 パネル寸法と符号（§6.7(1)）

```
dc = Dc − tcs   （柱せい方向 = 梁のパネル幅）
db = Db − tbs   （梁せい方向 = 柱のパネル高さ）
tp = パネル板厚
```
符号は「左から水平力を受けたときの標準 M 図」を正とする（資料の規約）。

### 7.2 フェースモーメント変換（§6.7(2)、§6.2.3 と直結）

節点中心モーメント（`*` 付き）から、パネル枠位置のフェースモーメントを求める:

```
梁フェース:  bML = ML*b − bQL·(dc/2),   bMR = MR*b − bQR·(dc/2)
柱フェース:  cML = ML*c − cQL·(db/2),   cMU = MU*c − cQU·(db/2)
```

### 7.3 パネルせん断 pQ（§6.7(3)）

```
pQc·db = (bML + bMR) − (cQL + cQU)·(db/2)      … 梁側 (5)
pQb·dc = (cMU + cML) − (bQL + bQR)·(dc/2)      … 柱側 (6)
整合条件:  pQc·db = pQb·dc                        … (4)
パネルせん断応力:  τ = pQc/(dc·tp) = pQb/(db·tp)
```

＋型・ト型・L型・T型・連続梁の各節点は、**欠落部材の項を 0** にすることで同一式でカバー（資料 図23）。

### 7.4 剛性モデル（§6.7(4)）

- **方式A：剛域近似（既定・軽量）** — 接合部を剛域（§5）として扱う。
- **方式B：弾性せん断パネル** — パネルを純せん断する弾性パネル（`Kp = G·tp·dc` 相当）として
  フレーム水平変形へ寄与。軸・曲げ変形は生じないものとする。
- いずれも梁端・柱端と剛リンクで接続し、§5 の剛域・端ばねと整合させる。

> **剛性への寄与（組立 §8 と直結・二重計上注意）:**
> - **方式A（既定）**：接合部の剛性は接続する梁・柱の**剛域（§5）が表現済み**。パネル自身は
>   別の剛性を持たない → `tangent_stiffness` は**ゼロ行列**を返し、アセンブラ（§8）は
>   この要素を**スキップ**する。パネルは**求解後の評価（フェースM・pQ・τ）専用**。
>   方式A でパネルにも剛性を足すと剛域と二重計上になる。
> - **方式B**：パネルが弾性せん断パネル剛性（`Kp = G·tp·dc` 相当）を持つ → `tangent_stiffness`
>   が非ゼロを返し、通常どおり組み立てる。この場合は剛域近似と**併用しない**（どちらか一方）。

```
[2] 全体組立（§8）  → 支点拘束済み active 次元の K_free（決定的トリプレット）
[3] 全体拘束（剛床/MPC/RigidLink §4.6）
        → T 行列を作り  K_red = Tᵀ · K_free · T  （独立自由度のみに縮約）
[4] 求解（P0 ソルバ）→ u_indep
[5] 展開  u_free = T · u_indep  → 全体変位へ散布（支点=0）
[6] 応力復元（§6）  → 部材内力
```

> **二重計上トラップ（必ず守る）:**
> 1. 要素内縮約 [1] は**組立前**に行う（§4.6）。組立後にもう一度やると二重に縮約される。
> 2. **剛床ダイアフラムを適用した階の床シェルは、面内剛性を無効化**（§6.5/R13）。
>    面内剛性を剛床拘束とシェルの両方が持つと二重計上になる。
>    **P1 にはシェルが無い（P1.5）**ため現時点で発生しないが、**P1.5 でシェルを足す際の前提条件**
>    として本書に明記しておく（P1.5 仕様書の冒頭にも引き継ぐ）。

### 9.2 T 行列の構築

`T` は「独立自由度（マスター＋拘束に関与しない自由度）」→「全 active 自由度」への写像
（`u_free = T · u_indep`）。

- 拘束に関与しない active 自由度：`T` の対応行は単位（その独立自由度をそのまま）。
- **剛床ダイアフラム**（面内剛体）：スレーブ節点の `(Ux, Uy, Rz)` をマスター節点の
  `(Ux, Uy, Rz)` で表す（§12.4 の剛体変位式）。`Uz, Rx, Ry` は独立のまま。
- **MPC**：スレーブ自由度 `= Σ coef · マスター自由度`（`Constraint::Mpc::terms`）。
- **RigidLink**：指定 `dofs` についてスレーブをマスターに剛結。

## 12. 数式・理論補足（ジュニア向け）

### 12.1 なぜせん断変形（ティモシェンコ）か（→ §4）

細長い梁では曲げが支配的だが、せいの大きい梁（D/L が大きい）ではせん断たわみが無視できない。
ティモシェンコ梁は曲げに加えてせん断たわみ `PL/(G·As)` を含む。パラメータ
`φ = 12EI/(G·As·L²)` がせん断の効き具合を表し、`φ→0`（せん断剛性無限大＝細長い梁）で
曲げのみの Bernoulli 梁に戻る。せん断ロッキングを避けるため IIE（相互依存補間）の閉形式を使う。

### 12.2 剛域の変換（→ §5.1）

接合部の有限寸法により部材端が剛体的に動く。可とう部の端部自由度 `u_flex` と節点（接合部中心）の
自由度 `u_node` は、剛体リンクの幾何関係 `u_flex = Tr · u_node` で結ばれる。剛性はこの `Tr` で
`K_node = Trᵀ K_flex Tr` と回す。曲げ・軸・せん断・ねじりすべてに効く。

### 12.3 端部ばねの静縮約（→ §5.3）

端部に回転ばね `kθ` を直列に入れると、要素端に「内部自由度」が1つ増える。これを
**静縮約（static condensation）**で消去して 12×12 に戻す。分割行列
`[[Kaa, Kab],[Kba, Kbb]]`（a=残す自由度, b=内部自由度）に対し
`K* = Kaa − Kab·Kbb⁻¹·Kba`。`kθ→∞`（剛接）と `kθ→0`（ピン）の極限で標準の境界条件に一致する。

### 12.4 剛床（剛体ダイアフラム）の変位式（→ §9.2）

床面内を剛体とみなすと、スレーブ節点 s の面内変位はマスター m の剛体運動で決まる:

```
ux_s = ux_m − (y_s − y_m)·rz_m
uy_s = uy_m + (x_s − x_m)·rz_m
rz_s = rz_m
```

この関係を `T` 行列の該当行に書き込む。鉛直・面外回転（`uz, rx, ry`）は拘束しない。

### 12.5 拘束の縮約 K_red = TᵀKT（→ §9）

独立自由度 `u_indep` と全（支点除外）自由度 `u_free` を `u_free = T·u_indep` で結ぶと、
ひずみエネルギー `½ u_freeᵀ K_free u_free` は `½ u_indepᵀ (Tᵀ K_free T) u_indep` になる。
よって縮約系の剛性は `K_red = Tᵀ K_free T`、荷重は `f_red = Tᵀ f_free`。解いた後
`u_free = T·u_indep` で戻す。要素内縮約（剛域・端ばね）と全体縮約（剛床・MPC）を分けることで
二重計上を避ける（§4.6）。

### 12.6 フェースモーメント（→ §7.2）

解析が返すのは節点芯（接合部中心）のモーメント。実際に断面算定すべきは柱・梁フェース位置。
フェースでは中心モーメントから「せん断 × 半パネル寸法」を差し引く：
`bML = ML*b − bQL·(dc/2)` 等。これにより危険断面（§6.2.3）とパネルせん断（§7.3）が整合する。
