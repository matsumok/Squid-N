## 2. T0: UniaxialMaterial トレイト（sc-material）

設計書 §7。**一軸履歴則を共通トレイトで**提供し、ファイバ／MS／壁ばねが共有する。

```rust
// sc-material/src/lib.rs
pub trait UniaxialMaterial {
    /// 試行ひずみ strain に対する (応力, 接線剛性)。状態は内部に試行値として保持。
    fn trial(&mut self, strain: f64) -> (f64 /*stress*/, f64 /*tangent*/);
    /// 試行を確定（収束後にコミット）。
    fn commit(&mut self);
    /// 試行を破棄して直前のコミット状態へ戻す（リジェクト時）。
    fn revert(&mut self);
}
```

> **commit/rollback パターン（設計書 §6.1）:** 非線形解析は「試行ステップ→収束判定→確定 or 巻き戻し」。
> 材料は **試行状態（trial）と確定状態（committed）を内部に2系統**持ち、`trial` は確定状態から計算、
> `commit` で試行→確定、`revert` で試行破棄。全履歴則でこの規律を統一する。

### 2.1 状態の持ち方（例）

```rust
// 履歴変数の確定／試行の2系統を持つのが基本形。
struct HistoryState { strain: f64, stress: f64, /* 折点・最大経験ひずみ・除荷点 等 */ }
pub struct SomeMaterial { committed: HistoryState, trial: HistoryState, params: Params }
```

### 2.2 係数のデータ駆動（設計書 §7 末尾）

- 包絡線（応力–ひずみ式）・除荷剛性低下指数などの係数は **テーブル／設定ファイル**から読む。
  **コードにハードコードしない。** 出典（規準・式番号）はコメントに残す。

**DoD（T0）:** トレイトがコンパイルでき、ダミー線形材料で `trial→commit→trial→revert` が
期待どおり状態遷移する（revert で1つ前のコミットへ戻る）。

## 3. T1: コンクリートモデル

## 4. T2: 鋼・鉄筋モデル

設計書 §7。

- **バイリニア**（弾完全塑性／ひずみ硬化）。
- **Menegotto–Pinto**（バウシンガー効果を滑らかに表現。鉄筋の繰り返しに重要）。

**Menegotto–Pinto の式（正規化形。閉形式。文献転記不要）:**
```
正規化:  ε* = (ε − εr)/(ε0 − εr),   σ* = (σ − σr)/(σ0 − σr)
         （(εr,σr)=直前の反転点, (ε0,σ0)=漸近線の交点）
本式:    σ* = b·ε* + (1 − b)·ε* / (1 + |ε*|^R)^(1/R)
曲率:    R  = R0 − a1·ξ / (a2 + ξ)      （ξ=反転後の塑性ひずみ振幅。R0,a1,a2 は係数）
   b = ひずみ硬化比（= 硬化勾配/E）。代表値 R0≈20, a1≈18.5, a2≈0.15（原論文値・外部設定で可）。
接線:    dσ/dε は σ* の ε* 微分に (σ0−σr)/(ε0−εr) を掛けて得る。
```

```rust
// sc-material/src/steel.rs
pub struct Bilinear { pub e: f64, pub fy: f64, pub hardening: f64 }
impl UniaxialMaterial for Bilinear { /* 弾性→降伏→ひずみ硬化、除荷は弾性勾配 */ }

pub struct MenegottoPinto { pub e: f64, pub fy: f64, pub b: f64,
                            pub r0: f64, pub a1: f64, pub a2: f64 } // 代表 20/18.5/0.15
impl UniaxialMaterial for MenegottoPinto { /* 上式。反転点・ξ を状態に保持し commit/revert */ }
```

**DoD（T2）:** バイリニアの単調・繰り返しが理論どおり（降伏点・硬化勾配・弾性除荷）。MP が
バウシンガー効果を示すループ（参照と一致、§8.1）。

## 5. T3: 履歴則（部材レベル・せん断）

設計書 §7。

- **RC 部材の曲げ履歴**：**武田モデル（Takeda）**と degrading 版（剛性低下型トリリニア）。
- **せん断履歴**：原点指向型、スリップ型。

**武田モデルの核（剛性低下型トリリニア。閉形式。文献転記不要）:**
```
スケルトン: トリリニア（ひび割れ Mc/φc, 降伏 My/φy, 終局 Mu/φu）。
除荷剛性（降伏後）: Ku = Ky · (φm / φy)^(−α)
   Ky=降伏点割線剛性, φm=最大経験曲率, α=除荷剛性低下指数（代表 α≈0.4。外部設定）。
再載荷: 反対側の最大経験点（無ければ降伏点）を指向する直線。
内側ループ: 上記の点を結ぶ（武田の規則）。スリップ型は再載荷を原点付近で寝かせる。
```
代表 α は 0.4〜0.5。係数（α・スリップ量）は外部設定で上書き可。

```rust
// sc-material/src/hysteresis.rs
pub enum HysteresisRule { Takeda { /* trilinear折点・低下指数 */ },
                          TakedaDegrading { /* … */ },
                          OriginOriented { /* せん断 */ }, Slip { /* … */ } }
```

> **メモ:** 武田は「部材の M-θ 履歴」に使う（§7 のスケルトンに付随）。ファイバ要素（P5）は
> 一軸材料（§3/§4）の積分で履歴を作るので、武田は集中ばね（one/two-component）系で使う。

**DoD（T3）:** 規定の繰り返し変位履歴で各則のループが参照と一致（`insta`、§8.1）。

## 7. T5: 部材スケルトン自動算定（sc-skeleton）★製品核心

設計書 §8。**断面・配筋・材料・スパン・想定軸力からスケルトン曲線を自動生成**する。

処理フロー（設計書 §8）:

```
1. 断面ファイバモデルから M–φ 関係を数値積分で算定（軸力依存：N を固定して κ を増やす）。
2. ひび割れ・降伏・終局点を抽出し トリリニア化（折れ点の決め方は設計規準準拠）。
3. 部材の境界条件・スパン・反曲点位置から M–φ → M–θ（部材端モーメント–回転角）へ積分変換（柔性法）。
4. せん断・付着・抜出しによる回転を加算。
5. 履歴則（武田型等）と組み合わせ、要素に渡すスケルトン＋履歴パラメータを確定。
```

## 9. 数式・理論補足（ジュニア向け）

### 9.1 一軸履歴と commit/rollback（→ §2）
材料は「現在ひずみ→応力」だけでなく**過去の経験（最大ひずみ・除荷点・ひび割れ）に依存**する
（履歴依存）。非線形解析は試行と確定を繰り返すので、材料は試行値（trial）と確定値（committed）を
分け、`commit`/`revert` で確定・巻き戻しする。これを全材料で統一すると、要素・解析の巻き戻しが安全。

### 9.2 ファイバ積分と平面保持（→ §6）
「断面は平面を保つ」と仮定すると、断面内の任意点のひずみは `ε(y,z) = ε0 − κz·y + κy·z`
（ε0＝図心軸ひずみ、κ＝曲率）。各ファイバのひずみ→材料則で応力→面積を掛けて積分すると断面力
`N=Σσ·A`, `M=Σσ·A·距離`。接線断面剛性 3×3 も同様に各ファイバの接線 Et を積分して作る。
これで軸力–二軸曲げの相互作用（N–M 相関）が自然に出る（MS 要素や柱に重要）。

### 9.3 M–φ とトリリニア化（→ §7）
断面に軸力 N を一定に保ち曲率 φ を増やしながらファイバ積分すると、モーメント M が得られる
（M–φ 曲線）。ひび割れ・鉄筋降伏・コンクリート圧壊で勾配が折れるので、3直線（トリリニア）に
近似する（折点＝ひび割れ Mc／降伏 My／終局 Mu）。

### 9.4 M–φ → M–θ（柔性法）（→ §7）
部材の端回転 θ は、部材長に沿った曲率 φ の積分。反曲点位置・境界条件から曲率分布を仮定して
`θ = ∫ φ dx` を求める（柔性法）。これに、せん断変形・鉄筋抜出し（pullout）・付着すべりによる
回転を加えて、部材端の M–θ スケルトンを完成させる。

## 10. 用語（P4 で増えるもの）
| 用語 | 意味 |
|---|---|
| 一軸材料（UniaxialMaterial） | ひずみ→応力の1次元履歴則。ファイバ/ばねが共有 |
| commit/revert | 試行の確定／巻き戻し。非線形の収束反復で使う |
| ファイバ断面 | 断面を小片（ファイバ）に分割し各々に材料を割当 |
| 平面保持 | 断面が平面を保つ仮定。ひずみ＝軸＋曲率の線形分布 |
| M–φ | モーメント–曲率関係。断面の非線形特性 |
| トリリニア | ひび割れ／降伏／終局で折れる3直線近似 |
| M–θ | 部材端モーメント–回転角。M-φ を部材長で積分 |
