# 数量積算の 参照実装マニュアル照合レポート

**照合日:** 2026-07-16
**原典:** 参照実装マニュアル 計算編「09. 数量積算」
**方法:** マニュアル全節（各部位共通事項／基礎梁・基礎フーチング／柱／大梁／小梁／
床／片持ち床・出隅・入隅／壁／ブレース）と実装（`squid-n-design-jp::quantity`、
`squid-n-app::{quantity_view, summary}`、`squid-n-mcp::quantity_takeoff`）を突き合わせ、
式レベルで一致を確認して実装した項目（B）、モデル制約・設計判断として残置した
項目（C）、原典照合上の注意（D）に分類して記録する。

**方針:** 数量積算は本照合まで実装が存在しなかったため、全項目が「本照合に伴い
新規実装」（B）に該当する。データモデルに情報が無く算定不能な項目のみ C 節に
残置し、結果の注記（`QuantityTakeoff::notes`）にも同内容を明示する。
照合の再現は `crates/squid-n-design-jp/src/quantity/{member.rs, tests.rs}` の
単体テスト（マニュアルの式の手計算値と突合）による。

## D. 原典照合上の注意（式の解釈）

| # | 内容 |
|---|------|
| D1 | ハンチ付きコンクリート体積の加算項の分母は、原典の組版では判別しづらいが、平均断面 `(B+Bi)/2 × (D+Di)/2` の展開により 4 と解釈した（`[(B+Bi)(D+Di)]/4 × Li`）。 |
| D2 | 基礎梁の型枠式は原典どおり「側面 1 面＋底面」（大梁と異なり側面の ×2 が付かない）で実装した。地中梁の掘削面・耐圧版側は型枠不要となる取り扱いと解釈。 |
| D3 | 基礎梁型枠のハンチ項は原典で i 端の項が 2 回現れる（`(Di−D)/2·Li` が重複）が、i/j の誤記と解釈し `(Di−D)/2·Li + (Dj−D)/2·Lj` とした。 |
| D4 | 大梁型枠の底面最終項は原典で `(Bi−B)/2·Lj` と i/j が混在するが、`(Bj−B)/2·Lj` の誤記と解釈した。 |
| D5 | 壁筋の `(L+2S)·W`・`(H+2S)·h` の W・h は原典に凡例が無いが、横筋・縦筋の本数（配筋列数を含む）と解釈した。実装はせん断補強筋比 ps による等価換算（C6）。 |
| D6 | 柱フープ本数の `H/cP` は「柱長さ H ÷ フープピッチ」と解釈した（原典の「柱内法部分フープ本数」の記述に対し、柱長さ＝床上〜床上をそのまま用いる保守側の取り扱い）。 |
| D7 | 円形柱は原典に規定が無いため、V=πD²/4·H・型枠=πDH・フープ一組=legs·πD の等価式で算定した。CFT は鋼管重量に加えて充填コンクリート体積（内空断面×H）を計上する（原典の対象外だが実量として妥当な拡張）。 |
