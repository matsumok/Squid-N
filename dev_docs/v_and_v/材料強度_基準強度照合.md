# 材料強度・基準強度 資料照合（2026-07）

- **対象**: 材料定数（E・G・ν・単位体積重量）、鋼材の基準強度 F（板厚区分・大臣認定品）、
  鋼材の許容応力度（ft/fs/fc/fb）、コンクリートの許容応力度、鉄筋の基準強度・許容応力度
- **参照解の出典**: 材料強度資料（構造材の定数 4-1／鋼材の許容応力度等 4-2／コンクリートの
  許容応力度 4-3／鉄筋の許容応力度等 4-4。H12 建告第2464号・第2466号、H13 国交告第1024号、
  令90・91・96〜98条、JFE スチール大臣認定品の基準強度表に基づく資料）
- **許容差**: 告示値一致（厳密）

## 照合範囲

資料 4-1（材料定数）・4-2（鋼材の基準強度・許容応力度）・4-3（コンクリート）・
4-4（鉄筋）を照合した。4-5（幅厚比種別）・溶接継目・高力ボルトは本照合の対象外
（幅厚比は `secondary/width_thickness` に実装済みだが別途の照合とする）。

## 発見した相違と対応（本照合での修正）

| # | 相違 | 対応 |
|---|------|------|
| 1 | TMCP 鋼材（325/355/385/440）・低降伏点鋼（LY100=80、LY225=205）が未実装 | `material_grade` に追加（TMCP は板厚 40mm 超でも F 低減なし） |
| 2 | ST-Bridge 取込（`material_std.rs`）で接尾辞なし `SN490` が F=355（520N 級扱い）に誤分類。検定側（F=325）と不整合 | 対応表を `squid_n_core::material_grade` に一本化して解消（F=325） |
| 3 | 名称→F 値の対応表が 3 箇所（design-jp／io／app）に独立実装され不整合の温床 | `squid_n_core::material_grade` に一本化。3 箇所とも同表を参照 |
| 4 | 鋼材検定で名称から F を解決できない材料が無警告で F=235 にフォールバックし、直接入力材料の Fy が反映されない | 名称で解決できない場合は `Material.fy` を基準強度に用いるフォールバックを追加（それも無ければ従来どおり 235） |

## 既知の相違・残課題（未対応）

- **G の慣用値**: 資料は G=79000 N/mm² だが、実装は G=E/(2(1+ν))=78846 N/mm² を
  都度算出（相対差 0.2%）。資料の値は丸め値であり、算定式そのものは一致。
- **材料強度の割増**（追記: 実装済み）: 材料強度は基準強度の 1.1 倍以下
  （590N 級の SA440・TMCP440 等は 1.05 倍以下）とできる規定（資料 4-2 注記 *1）を、
  プッシュオーバー（保有水平耐力計算）の部材耐力算定に実装した。
  鋼材は 1.1 倍（590N 級 1.05 倍）、RC 主筋は 1.1 倍（fy 未設定の既定値 345 にも適用）、
  せん断補強筋は割増対象外。材料ごとの割増係数の直接入力
  （`Material::strength_factor`。直接入力材料向け）が自動判定より優先される。
  実装は部材組み立ての強度基準区分
  （`squid_n_element::factory::StrengthBasis::MaterialStrength`）と
  プッシュオーバー専用のヒンジ・せん断降伏閾値への係数適用
  （`material_grade::{material_strength_factor_steel, material_strength_factor_rebar}`）による。
  当初のモデル複製方式（fy を割増した複製モデル）は、RC 主筋の fy 未設定既定値
  （345）に係数を掛けられない・fy の Some/None がせん断降伏の分岐に使われている、
  という理由で廃止し、組み立て時の係数配線方式へ変更した。
  時刻歴応答解析・許容応力度計算には適用しない。
  指示は「SA440 以外は 1.1 倍」だったが、資料の注記 *1 では HBL®440/G440
  （本実装の TMCP440）を含む 590N 級が 1.05 倍のため、TMCP440 も 1.05 倍とした。
- **丸鋼の付着許容応力度**（0.7 系の値）は未実装（`concrete_allowable_bond` は異形筋のみ）。
- **板厚 100mm 超**の F 値は規定が無いため最終区分値を流用（非保守的になり得る。
  実装コメントに明記済み）。
- 溶接継目（令98条）・高力ボルト（令96条・H12 建告第2466号）の許容応力度は本照合の対象外。
- JBCR385・BSH325・P-325 等の円形鋼管系大臣認定品、SM520 の 490N 級 P-SM520 板厚特例
  （*3）はグレード表に未収載（必要になったら `material_grade` に追加する）。
