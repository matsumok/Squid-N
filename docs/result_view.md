# 結果の表示（3D ビューア）

3D ビューアにおける解析結果の表示仕様のうち、計算を伴うもの（変形図の表示倍率と、
床・二次部材の変形追従）の根拠を示す。ここでの計算はいずれも**画面表示のための
処理**であり、解析結果（節点変位・部材応力）そのものには影響しない。

## 変形図の表示倍率（自動計算）

変形図・モード形・応力図への変形重ね表示では、変位を実寸で描くと視認できない
ため、拡大した変位を節点座標に加えて描画する。表示倍率 $s$ はモデルの
バウンディングボックスから自動で決定する:

$$
s = \frac{0.1 \, D}{\delta_{\max}}
$$

- $D$: モデル全節点のバウンディングボックス対角線長
- $\delta_{\max}$: 全節点の並進変位成分（$u_x, u_y, u_z$）の最大絶対値

すなわち、**最大変位がバウンディングボックス対角長の 10% で表示される**倍率で
ある。表示座標は `原座標 + s × 変位（並進成分）` となる。採用された倍率は
ビュー左上に「変形倍率 ×N（自動）」として表示する。

モード形（固有モード）も同じ正規化で表示するが、固有ベクトルの規模は任意の
ため倍率の数値に物理的な意味はなく、倍率の注記は表示しない。

## 応力図（N/Q/M 図）への変形の重ね表示

N/Q/M 図の表示中に「変形表示」を有効にすると、上記の倍率で変形させた節点
座標を基準に架構を描画する。応力図は変形後の材軸に沿って要素ローカル y 軸
方向へ張り出すため、図も変形形状に追従する。応力の値・最大値の正規化は
変形の重ね表示による影響を受けない。

## 床・二次部材の変形追従（表示用の変位補間）

スラブ境界・小梁支持点・二次部材（小梁・間柱）の節点のうち、主架構要素が
接続せず、拘束（剛床・剛リンク・MPC）のマスターでもない節点には解析自由度が
割り当てられず、解析結果の変位は 0 になる（零剛性自由度による特異行列を
避けるための扱い。荷重は床荷重分配で主架構へ変換される）。

これらの節点をそのまま描くと変形図で床・二次部材だけが原位置に残るため、
表示上は次の補間で主架構の変形に追従させる:

1. 対象節点の座標 $p$ を、主架構の各 2 節点要素（線材）の線分へ射影し、
   射影点までの距離が最小の線分を選ぶ。
2. 選ばれた線分（端点 $i$, $j$、変位 $u_i$, $u_j$）上の射影位置
   $t \in [0, 1]$（線分外への射影は端点へクランプ）で、変位を線形補間する:

$$
u = (1 - t)\, u_i + t\, u_j
$$

二次部材の支持点が大梁のスパン中間に節点共有なしで載る一般的なモデルでは、
支持点が取り付く大梁の変形に沿って床・二次部材が追従する。

この補間は**描画専用の近似**であり、次の限界がある:

- 小梁・二次部材自体のたわみ（支持点間の曲げ変形）は表現しない。両端
  支持点の変位による追従のみである。
- 補間ソースは線材のみのため、壁（面要素）にしか近接しない節点は最寄りの
  線材から補間される。

なお、床（スラブ・小梁）と二次部材の表示は「床・二次部材」トグルで
まとめて非表示にできる。

## 検定比の表示

表示モード「検定比」では、部材検定の結果に基づいて部材を着色する。

- 部材の色は、その部材の全検定位置（材端・材中間などの複数箇所）における
  **最大検定比**で決まる。色の対応は検定表と共通の3段階規約:
  - 検定比 ≤ 0.8: 緑（良好）
  - 検定比 ≤ 1.0: 黄（注意）
  - 検定比 > 1.0: 赤（NG。超過）
- 部材中点には検定比の数値（全検定位置中の最大値）をラベル表示する。
  NG 部材（検定比 > 1.0）は赤字・太字相当の大きめのフォントで、かつ線の
  太さも太くして強調する。
- 壁（面要素）は同じ色規約の半透明ポリゴンで塗り、輪郭も検定比の色で描く。
- 柱梁接合部・パネルゾーン・耐震壁などの節点単位の検定は、部材の線とは
  別に、対象節点上の色付き円（白い輪郭付き）で表示する。円の色も同じ
  3段階規約に従う。
- 検定を実施していない部材・節点はグレーのまま（着色されない）。
- ビュー左上に、検定比の最大値・NG 件数の集計、色の凡例、モデル編集後に
  解析結果が陳腐化している場合の注意（「モデルが編集されています。解析を
  再実行してください。」）を表示する。解析結果が無い場合は、検定比図の
  代わりに解析の実行を促す案内のみを表示する。
