# はじめに

**Squid-N** は、日本の建築構造計算一貫プログラムです。Rust で実装されています。

このサイトは、Squid-N の開発ドキュメントを集約したものです。

- **[アーキテクチャ](./architecture.md)** — 15 クレートから成る階層型構成の全体像
- **設計仕様（specs）** — フェーズ単位の実装仕様書と原典（法令・規準）照合リスト
- **検証（V&V）** — 各要素・各設計式の Verification & Validation レポート
- **開発運用** — ロードマップ、申し送り、UI 関連ドキュメント

## リポジトリ

ソースコードは [github.com/hrntsm/squid-n](https://github.com/hrntsm/squid-n) にあります。

ビルド・テスト・静的解析の手順はリポジトリの `README.md` を参照してください。
