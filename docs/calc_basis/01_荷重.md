# 1. 荷重・外力

固定荷重・積載荷重・積雪荷重・風圧力・地震力の算定根拠。地震力は建築基準法施行令88条および
昭和55年建設省告示第1793号、風圧力は令87条および平成12年建設省告示第1454号を根拠とする。

> 記載フォーマット: 📖 根拠 / ∑ 算定式 / 💻 実装 / ✓ 整合性（[凡例](./README.md#このセクションの読み方記載フォーマット)）

---

## 1.1 地震力

### 1.1.1 地震層せん断力係数 Ci と層せん断力 Qi

- 📖 **根拠**: 建築基準法施行令88条。`Ci = Z · Rt · Ai · C0`、`Qi = Ci · Wi`（Wi は当該層が支える
  部分の固定荷重＋積載荷重の合計＝当該層以上の累積重量）。
- ∑ **算定式**（実装）:
  - `αi = Wi / Wtotal`（当該層以上の累積重量比）
  - `Ai = 1 + (1/√αi − αi) · 2T/(1 + 3T)`
  - `Ci = Z · Rt · Ai · C0`
  - `Qi = Ci · αi · Wtotal = Ci · Wi`
  - 各層水平外力 `Pi = Qi − Qi+1`（最上層は `Pi = Qi`）
- 💻 **実装**: `squid_n_load::ai::ai_distribution`（`crates/squid-n-load/src/ai.rs`）。層せん断力は
  `Qi = Ci · Wi`（累積重量）で算定。`pi_from_qi` が Qi 差分から Pi を求め、数値誤差による微小負値
  （`< -1e-9·Q1`）を検出したうえで `max(0)` にクランプする。
- ✓ **整合性**: コメントに「Qi = Ci·Wi（Wi＝当該層以上の累積重量＝令88条）」と明記。過去に
  `Ci·単層重量`（下層 Pi が負になり基部せん断が過小）だったバグを修正済み（V&V #10、
  `test_story_shear_uses_cumulative_weight`）。C0 は引数（一次設計 0.2／二次設計 1.0 を上位で与える）。

### 1.1.2 振動特性係数 Rt

- 📖 **根拠**: 昭和55年建設省告示第1793号 第2。地盤種別ごとの卓越周期 Tc により3分岐。
- ∑ **算定式**:

  $$R_t = \begin{cases} 1.0 & (T < T_c) \\ 1 - 0.2\,(T/T_c - 1)^2 & (T_c \le T < 2T_c) \\ 1.6\,T_c/T & (2T_c \le T) \end{cases}$$

- 💻 **実装**: `squid_n_load::ai::rt(t, tc)`（`ai.rs:25`）。分岐しきい値 `tc`・`2·tc`、係数 `0.2`・`1.6`。
- ✓ **整合性**: 式は告示1793号第2の Rt と一致。`rt()` 関数自体には告示番号のコメントは付されていないが、
  呼び出し側 `seismic.rs` の doc に告示1793号を明記。[原典照合リスト](../specs/原典照合リスト.md)で Rt は
  Web2源照合済（☑）。

### 1.1.3 地盤種別と卓越周期 Tc

- 📖 **根拠**: 昭和55年建設省告示第1793号 第2。第1種／第2種／第3種地盤の Tc。
- ∑ **算定式（表）**: 第1種 `Tc = 0.4 s`、第2種 `0.6 s`、第3種 `0.8 s`。
- 💻 **実装**: `squid_n_load::ai::tc_of(soil)`（`ai.rs:35`）、`SoilClass { I, II, III }`。
- ✓ **整合性**: 値は告示規定と一致。コード上のコメントによる告示引用はなし（[原典照合リスト](../specs/原典照合リスト.md)
  で Tc は要原典確認 ☐）。

### 1.1.4 高さ方向分布係数 Ai

- 📖 **根拠**: 昭和55年建設省告示第1793号。`Ai = 1 + (1/√αi − αi)·2T/(1+3T)`。
- ∑ **算定式**: 上式そのまま。`t_factor = 2T/(1+3T)`。
- 💻 **実装**: `ai_distribution`（`ai.rs:80`）。
- ✓ **整合性**: 告示式と一致。[原典照合リスト](../specs/原典照合リスト.md)で Ai は Web 照合済（☑）。

### 1.1.5 地域係数 Z

- 📖 **根拠**: 昭和55年建設省告示第1793号 別表第2（地方区分ごとの Z 値）。取り得る値は 1.0／0.9／0.8／0.7。
- ∑ **算定式**: 市町村名 → Z 値のルックアップ（式ではなくテーブル参照）。
- 💻 **実装**: `squid_n_load::z_table`（`z_table.rs`）。`ZTable::from_csv`／`lookup`。定数
  `Z_VALUES = [1.0, 0.9, 0.8, 0.7]`。全市町村データは同梱せず、利用者提供の CSV（`市町村名,Z値`）に委ねる方針
  （CSV 値が `Z_VALUES` のいずれとも一致しなければパースエラー）。
- ✓ **整合性**: doc コメントに告示1793号別表第2を明記。市町村対応表は Category A の外部データ（版管理）。

### 1.1.6 標準せん断力係数 C0

- 📖 **根拠**: 令88条。一次設計 `C0 ≥ 0.2`、保有水平耐力計算（二次設計）`C0 = 1.0`。
- 💻 **実装**: C0 は `ai_distribution`/`build_seismic_load_case` の引数として上位から与えられる。
- ✓ **整合性**: 一次 0.2／二次 1.0 の値は解析設定側で指定。[原典照合リスト](../specs/原典照合リスト.md)参照。

### 1.1.7 一次固有周期の略算 T

- 📖 **根拠**: 令88条・昭和55年建設省告示第1793号。`T = h(0.02 + 0.01α)`（α は鉄骨造部分の
  高さ比）。RC 造 0.02、S 造 0.03 に相当。
- ∑ **算定式**:
  - `T = h · (0.02 + 0.01·α)`
  - 鉄骨造比 `α = Σ(S造階の階高) / Σ(全階高)`（0〜1 にクランプ）
- 💻 **実装**: `squid_n_load::ai::approx_t(height_m, steel_ratio)`（`ai.rs:108`）、鉄骨造比は
  `seismic::steel_height_ratio`（`seismic.rs:38`）。準精算モード（`AiMode::SemiPrecise`）では固有値解析の
  1次周期を用いる（フォールバック 0.3 s）。
- ✓ **整合性**: `steel_height_ratio` の doc に「令88条・平成12年建設省告示第1793号」を明記。以前 α を常に
  0.0 にハードコードしていたバグを修正済み（レビュー §1.5）。

### 1.1.8 塔屋（PH）階・地下階を含む地震層せん断力分布

- 📖 **根拠**: 令88条の考え方に基づく実務的取扱い（RESP-D マニュアル「地震荷重の計算」に準拠）。
- ∑ **算定式**:
  - **一般階**: αi 算定に PH 重量を含め地下重量は含めず、通常の Ai 分布式。`Qi = Z·Rt·Ai·C0·Wi`。
  - **PH（塔屋）階**: `Qi = k · ΣWj`（k は 0.5〜1.0 の指定震度、その階以上の PH 累積重量）。
  - **地下階**: `Qi = Q(i+1) + Ki·Wi`、`Ki = 0.1·(1 − min(Hi, 20)/40)·Z`（Hi は地盤面からの深さ[m]）。
- 💻 **実装**: `squid_n_load::ai::seismic_shear_distribution`（`ai.rs:140`）。全階が一般階のときは
  `ai_distribution` に委譲し厳密一致。地下階式の定数 `0.1`・`20`（深さ上限）・`40`。
- ✓ **整合性**: doc に RESP-D マニュアルを引用。PH の k・地下の低減式は実務慣用式であり、告示の直接規定
  ではない旨に留意（Category B 寄り）。

## 1.2 風圧力

### 1.2.1 速度圧 q

- 📖 **根拠**: 令87条・平成12年建設省告示第1454号。`q = 0.6·E·V0²`。
- ∑ **算定式**:
  - `E = Er² · Gf`
  - `q = 0.6 · E · V0²` [N/m²]（V0 は基準風速）
- 💻 **実装**: `squid_n_load::wind::wind_forces`（`wind.rs:132`）。速度圧係数 `0.6`。
- ✓ **整合性**: ファイル doc に「令87条・平成12年建設省告示第1454号に相当する略算式（RESP-D マニュアル
  記載のもの）」と明記。

### 1.2.2 平均風速の高さ方向分布係数 Er と地表面粗度区分

- 📖 **根拠**: 告示1454号（別表: 地表面粗度区分 I〜IV の Zb・ZG・α）。
- ∑ **算定式**:
  - 粗度パラメータ: I `(Zb=5, ZG=250, α=0.10)`、II `(5, 350, 0.15)`、III `(5, 450, 0.20)`、
    IV `(10, 550, 0.27)`
  - `Er = 1.7·(Zb/ZG)^α`（H ≤ Zb）、`Er = 1.7·(H/ZG)^α`（H > Zb）
- 💻 **実装**: `wind::terrain_params`（`wind.rs:17`）、`wind::er_factor`（`wind.rs:45`）。係数 `1.7`。
- ✓ **整合性**: doc に「地表面粗度区分（告示1454号別表）」を明記。値は告示別表と一致。

### 1.2.3 ガスト影響係数 Gf

- 📖 **根拠**: 告示1454号（構造骨組用ガスト影響係数の表）。
- ∑ **算定式**: 粗度区分ごとに (H≤10m の値, H≥40m の値) を取り、10〜40m は直線補間。
  I `(2.0, 1.8)`、II `(2.2, 2.0)`、III `(2.5, 2.1)`、IV `(3.1, 2.3)`。
- 💻 **実装**: `wind::gust_factor`（`wind.rs:28`）。高さ境界 10.0／40.0。
- ✓ **整合性**: 表値・補間規則は告示・RESP-D マニュアルに準拠。

### 1.2.4 層区間の高さ方向分布係数 Kz と層水平力

- 📖 **根拠**: 告示1454号（速度圧の高さ方向分布）。層の負担区間で Kz を平均化。
- ∑ **算定式**:
  - `Kz = (z/H)^{2α}`（区間平均。2式にまたがる場合は区間積分の平均）
  - 風力係数合成 `Cf_total = Cpe_windward·Kz − Cpe_leeward`（既定 `0.8·Kz + 0.4`、内圧 Cpi 相殺）
  - 層水平力 `F = q · Cf_total · 見付け幅 · 負担高さ`
- 💻 **実装**: `wind::kz_for_interval`（`wind.rs:60`）、`wind::wind_forces`。既定 `cpe_windward=0.8`,
  `cpe_leeward=-0.4`。solver 側 `wind::wind_static`（`statics/analysis/wind.rs`）が建物高さ
  `H = 最上階床高 + パラペット/2`、各層負担区間、階別見付け幅を算定し、剛床へ重量比按分する。
- ✓ **整合性**: doc に RESP-D マニュアル「風荷重の計算」を引用。内圧相殺 `0.8·Kz+0.4` の導出をコメントに
  明記。PH 階への風荷重接続は未対応（残課題として明記）。

## 1.3 積載荷重

### 1.3.1 柱の積載荷重低減（支持床数による低減）

- 📖 **根拠**: 建築基準法施行令85条2項（支える床の数に応じた柱・基礎の積載荷重低減）。
- ∑ **算定式（低減率表）**: 支持床数 1→1.0、2→0.95、3→0.90、4→0.85、5→0.80、6→0.75、
  7→0.70、8→0.65、9以上→0.60。
- 💻 **実装**: `squid_n_load::live_load::column_live_load_reduction`（`live_load.rs:30`）、
  支持床数は `floors_supported_by_column`。`Model.load_cfg.live_load_reduction`（既定 false、
  RESP-D と同じく既定では低減しない）が有効時のみ適用する想定。
- ✓ **整合性**: doc に「令85条2項」を明記。低減率表は令85条の表と一致。

## 1.4 床荷重の分配（スラブ → 梁・柱）

- 📖 **根拠**: 力学（負担面積法・45°振り分けの三角形／台形分布）。RESP-D マニュアル「床荷重」
  「片持ちスラブ」「出隅の片持ちスラブ」に準拠。総和保存（Σ梁・小梁・柱荷重 = w×面積）を全経路で満たす。
- ∑ **算定式（主な経路）**:
  - **三角形・台形（TriTrapezoid）**: 45°振り分け。正方形は各辺三角形分布 `w0 = w·L/2`。
    矩形は短辺三角形・長辺台形。
  - **一方向（OneWay）**: 負担辺へ `w_line = w·スパン/2` の等分布。
  - **負担面積等分布換算（TributaryArea）**: 短辺 `w·short/4`、長辺 `w·(short·long/2 − short²/4)/long`。
  - **多角形負担面積法**: 200×200 格子で各セルを最近接辺へ帰属し辺荷重を算定（凸多角形で誤差 1% 未満）。
  - **片持ち・出隅**: 片持ちは辺へ `w·出し幅` の等分布、出隅は構造芯〜先端の矩形重量を節点集中荷重で柱へ。
- 💻 **実装**: `squid_n_load::floor`（`floor/mod.rs`）。`distribute_slab` が形状・手法で経路分岐。
- ✓ **整合性**: 各経路とも総和保存を設計要件とし、台形／三角形は極限（a→L/2, a→0）で理論値に一致することを
  doc に明記。小梁二段階伝達など簡易化した仮定は doc に明示。

### 1.4.1 固定端モーメント・せん断力（CMQ）

- 📖 **根拠**: 力学（両端固定梁の固定端力の閉形式。[V&V](../v_and_v/README.md)で総和保存・固定端値検証）。
- ∑ **算定式**:
  - 等分布: `C = ±w·L²/12`、`Q = w·L/2`
  - 三角形: `C = ±5·w0·L²/96`、`Q = w0·L/4`
  - 台形: `FEM = (1/L²)∫₀ᴸ w(x)·x·(L−x)² dx` を閉形式評価
  - 一般（非対称・剛域考慮）: `FEM_i = (1/L²)∫w(ξ)ξ(L−ξ)²dξ`、`FEM_j = (1/L²)∫w(ξ)ξ²(L−ξ)dξ`、
    `Q = R0 ± (FEM_i − FEM_j)/L`（点・分布は閉形式、その他は合成シンプソン則）
- 💻 **実装**: `floor::{fem_uniform, fem_triangle, fem_trapezoid, cmq_flexible_span, cmq_with_rigid_zone}`
  （`floor/mod.rs`）。剛域考慮は3モード（剛域外力を CMQ 加算／柱へ伝達／無視）。
- ✓ **整合性**: 台形式は a→L/2 で三角形式、a→0 で等分布式に一致（検算コメントあり）。剛域考慮は RESP-D
  マニュアル「剛域の考慮」に準拠。

## 1.5 荷重組合せ

- 📖 **根拠**: 建築基準法施行令82条（標準荷重組合せ）、令86条（多雪区域）。
- ∑ **算定式（組合せ）**:
  - 長期: `G + P`（多雪区域: `G + P + δ1·S`）
  - 短期積雪: `G + P + S`
  - 短期地震: `G + P ± K`（多雪区域: `G + P + δ3·S ± K`）
  - 短期暴風: `G + P ± W`（多雪区域: `G + P + δ2·S ± W`）
  - 積雪低減係数 既定 `δ1 = 0.7`、`δ2 = δ3 = 0.35`
- 💻 **実装**: `squid_n_load::combo`（`combo.rs`）。`standard_combinations`／`SnowFactors`。
  長期・短期の判別は `is_short_term_combo`（K/E/W を含む、または S 項係数 ≥ 1.0 なら短期）。
- ✓ **整合性**: doc に「令82条」「令86条・82条」および RESP-D マニュアル 04 断面検定「荷重の組合せ」を
  明記。δ1/δ2/δ3 は[原典照合リスト](../specs/原典照合リスト.md)で令86条と整合として照合済（☑）。

## 1.6 地震用重量・層データの生成

- 📖 **根拠**: 令88条（地震用重量＝固定荷重＋積載荷重）。自重の梁せん断力（CMoQo による Q0）
  節点配分は RESP-D マニュアル「地震荷重の計算」「柱梁自重」に準拠。
- ∑ **算定式（概要）**:
  - 線材自重 `w = ρ·A·L_eff·g·factor`（RC/SRC 大梁は柱面間距離、S 梁は節点間距離）
  - 壁・シェル自重 `w = ρ·t·(A − 開口)·g`（Newell の公式で面積算定）
  - 仕上げ・付加線重量・雑壁重量を節点へ配分し、階ごとに集計
  - 剛床代表節点は慣性力重心（重量重み付き重心）に自動生成
- 💻 **実装**: `squid_n_load::story_gen`（`story_gen/mod.rs`）。鋼材単位体積重量 `γs = 77 kN/m³`
  （RESP-D マニュアル準拠、コメントに明記）。回転慣性重量 `Σ W·ir²`。
- ✓ **整合性**: 各処理の doc に RESP-D マニュアルの該当節を引用。単純梁静定反力・自重長さの取扱い
  （柱面間距離／節点間距離、下階柱なし時の付加）を実装 doc に明示。
