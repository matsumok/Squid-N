# 1. 荷重・外力

このソフトは、固定荷重、積載荷重、積雪荷重、風圧力、地震力を算定する。地震力は建築基準法施行令88条および昭和55年建設省告示第1793号、風圧力は令87条および平成12年建設省告示第1454号を根拠とする。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。（[凡例](./README.md#このセクションの読み方記載フォーマット)）

---

## 1.1 地震力

### 1.1.1 地震層せん断力係数 Ci と層せん断力 Qi

地震層せん断力係数 Ci と層せん断力 Qi は、建築基準法施行令88条に基づいて算定する。`Ci = Z · Rt · Ai · C0`、`Qi = Ci · Wi` であり、Wi は当該層が支える部分の固定荷重と積載荷重の合計、すなわち当該層以上の累積重量を表す。

**算定式**
- `αi = Wi / Wtotal`（当該層以上の累積重量比）
- `Ai = 1 + (1/√αi − αi) · 2T/(1 + 3T)`
- `Ci = Z · Rt · Ai · C0`
- `Qi = Ci · αi · Wtotal = Ci · Wi`
- 各層水平外力 `Pi = Qi − Qi+1`（最上層は `Pi = Qi`）

**実装**：`squid_n_load::ai::ai_distribution`（`crates/squid-n-load/src/ai.rs`）が算定する。層せん断力は `Qi = Ci · Wi`（累積重量）で求める。`pi_from_qi` が Qi の差分から Pi を求め、数値誤差による微小負値（`< -1e-9·Q1`）を検出したうえで `max(0)` にクランプする。

**整合性**：コメントに「Qi = Ci·Wi（Wi＝当該層以上の累積重量＝令88条）」と明記している。過去に `Ci·単層重量` としていたために下層 Pi が負になり基部せん断が過小となるバグがあったが、これは修正済みである（V&V #10、`test_story_shear_uses_cumulative_weight`）。C0 は引数であり、一次設計 0.2、二次設計 1.0 を上位で与える。

### 1.1.2 振動特性係数 Rt

振動特性係数 Rt は、昭和55年建設省告示第1793号 第2に基づき、地盤種別ごとの卓越周期 Tc によって3つに場合分けして算定する。

**算定式**

$$R_t = \begin{cases} 1.0 & (T < T_c) \\ 1 - 0.2\,(T/T_c - 1)^2 & (T_c \le T < 2T_c) \\ 1.6\,T_c/T & (2T_c \le T) \end{cases}$$

**実装**：`squid_n_load::ai::rt(t, tc)`（`ai.rs:25`）が算定する。分岐しきい値は `tc` と `2·tc`、係数は `0.2` と `1.6` である。

**整合性**：この式は告示1793号第2の Rt と一致する。`rt()` 関数自体には告示番号のコメントを付していないが、呼び出し側 `seismic.rs` の doc に告示1793号を明記している。[原典照合リスト](../specs/原典照合リスト.md)で Rt は Web2源照合済（☑）である。

### 1.1.3 地盤種別と卓越周期 Tc

卓越周期 Tc は、昭和55年建設省告示第1793号 第2に基づき、第1種、第2種、第3種の地盤種別ごとに定める。

**算定式**：第1種 `Tc = 0.4 s`、第2種 `0.6 s`、第3種 `0.8 s`。

**実装**：`squid_n_load::ai::tc_of(soil)`（`ai.rs:35`）が算定し、地盤種別は `SoilClass { I, II, III }` で表す。

**整合性**：値は告示規定と一致する。コード上のコメントによる告示引用はなく、[原典照合リスト](../specs/原典照合リスト.md)で Tc は要原典確認（☐）である。

### 1.1.4 高さ方向分布係数 Ai

高さ方向分布係数 Ai は、昭和55年建設省告示第1793号に基づいて算定する。

**算定式**：`Ai = 1 + (1/√αi − αi)·2T/(1+3T)`。実装では `t_factor = 2T/(1+3T)` を用いる。

**実装**：`ai_distribution`（`ai.rs:80`）が算定する。

**整合性**：告示式と一致する。[原典照合リスト](../specs/原典照合リスト.md)で Ai は Web 照合済（☑）である。

### 1.1.5 地域係数 Z

地域係数 Z は、昭和55年建設省告示第1793号 別表第2の地方区分ごとの Z 値に基づいて与える。取り得る値は 1.0、0.9、0.8、0.7 である。

**算定式**：市町村名から Z 値を引くルックアップであり、式ではなくテーブル参照による。

**実装**：`squid_n_load::z_table`（`z_table.rs`）の `ZTable::from_csv` と `lookup` が処理する。定数は `Z_VALUES = [1.0, 0.9, 0.8, 0.7]` である。全市町村データは同梱せず、利用者提供の CSV（`市町村名,Z値`）に委ねる方針であり、CSV 値が `Z_VALUES` のいずれとも一致しなければパースエラーとなる。

**整合性**：doc コメントに告示1793号別表第2を明記している。市町村対応表は Category A の外部データ（版管理）である。

### 1.1.6 標準せん断力係数 C0

標準せん断力係数 C0 は、令88条に基づき、一次設計では `C0 ≥ 0.2`、保有水平耐力計算（二次設計）では `C0 = 1.0` とする。

**実装**：C0 は `ai_distribution` や `build_seismic_load_case` の引数として上位から与えられる。

**整合性**：一次 0.2、二次 1.0 の値は解析設定側で指定する。[原典照合リスト](../specs/原典照合リスト.md)を参照。

### 1.1.7 一次固有周期の略算 T

一次固有周期 T は、令88条および昭和55年建設省告示第1793号に基づき `T = h(0.02 + 0.01α)` で略算する。α は鉄骨造部分の高さ比であり、RC 造 0.02、S 造 0.03 に相当する。

**算定式**
- `T = h · (0.02 + 0.01·α)`
- 鉄骨造比 `α = Σ(S造階の階高) / Σ(全階高)`（0〜1 にクランプ）

**実装**：`squid_n_load::ai::approx_t(height_m, steel_ratio)`（`ai.rs:108`）が算定し、鉄骨造比は `seismic::steel_height_ratio`（`seismic.rs:38`）が求める。準精算モード（`AiMode::SemiPrecise`）では固有値解析の1次周期を用い、フォールバックは 0.3 s である。

**整合性**：`steel_height_ratio` の doc に「令88条・平成12年建設省告示第1793号」を明記している。以前 α を常に 0.0 にハードコードしていたバグは修正済みである（レビュー §1.5）。

### 1.1.8 塔屋（PH）階・地下階を含む地震層せん断力分布

塔屋階と地下階を含む地震層せん断力分布は、令88条の考え方に基づく実務的取扱い（RESP-D マニュアル「地震荷重の計算」に準拠）として算定する。

**算定式**
- **一般階**: αi の算定に PH 重量を含め地下重量は含めず、通常の Ai 分布式による。`Qi = Z·Rt·Ai·C0·Wi`。
- **PH（塔屋）階**: `Qi = k · ΣWj`（k は 0.5〜1.0 の指定震度、その階以上の PH 累積重量）。
- **地下階**: `Qi = Q(i+1) + Ki·Wi`、`Ki = 0.1·(1 − min(Hi, 20)/40)·Z`（Hi は地盤面からの深さ[m]）。

**実装**：`squid_n_load::ai::seismic_shear_distribution`（`ai.rs:140`）が算定する。全階が一般階のときは `ai_distribution` に委譲し厳密に一致させる。地下階式の定数は `0.1`、深さ上限 `20`、`40` である。

**整合性**：doc に RESP-D マニュアルを引用している。PH の k と地下の低減式は実務慣用式であり、告示の直接規定ではない点に留意する（Category B 寄り）。

## 1.2 風圧力

### 1.2.1 速度圧 q

速度圧 q は、令87条および平成12年建設省告示第1454号に基づき `q = 0.6·E·V0²` で算定する。

**算定式**
- `E = Er² · Gf`
- `q = 0.6 · E · V0²` [N/m²]（V0 は基準風速）

**実装**：`squid_n_load::wind::wind_forces`（`wind.rs:132`）が算定する。速度圧係数は `0.6` である。

**整合性**：ファイル doc に「令87条・平成12年建設省告示第1454号に相当する略算式（RESP-D マニュアル記載のもの）」と明記している。

### 1.2.2 平均風速の高さ方向分布係数 Er と地表面粗度区分

平均風速の高さ方向分布係数 Er は、告示1454号の別表（地表面粗度区分 I〜IV の Zb、ZG、α）に基づいて算定する。

**算定式**
- 粗度パラメータ: I `(Zb=5, ZG=250, α=0.10)`、II `(5, 350, 0.15)`、III `(5, 450, 0.20)`、
  IV `(10, 550, 0.27)`
- `Er = 1.7·(Zb/ZG)^α`（H ≤ Zb）、`Er = 1.7·(H/ZG)^α`（H > Zb）

**実装**：`wind::terrain_params`（`wind.rs:17`）と `wind::er_factor`（`wind.rs:45`）が算定する。係数は `1.7` である。

**整合性**：doc に「地表面粗度区分（告示1454号別表）」を明記しており、値は告示別表と一致する。

### 1.2.3 ガスト影響係数 Gf

ガスト影響係数 Gf は、告示1454号の構造骨組用ガスト影響係数の表に基づいて算定する。

**算定式**：粗度区分ごとに (H≤10m の値, H≥40m の値) を取り、10〜40m は直線補間する。I `(2.0, 1.8)`、II `(2.2, 2.0)`、III `(2.5, 2.1)`、IV `(3.1, 2.3)`。

**実装**：`wind::gust_factor`（`wind.rs:28`）が算定する。高さ境界は 10.0 と 40.0 である。

**整合性**：表値と補間規則は告示および RESP-D マニュアルに準拠する。

### 1.2.4 層区間の高さ方向分布係数 Kz と層水平力

層区間の高さ方向分布係数 Kz と層水平力は、告示1454号の速度圧の高さ方向分布に基づき、層の負担区間で Kz を平均化して算定する。

**算定式**
- `Kz = (z/H)^{2α}`（区間平均。2式にまたがる場合は区間積分の平均）
- 風力係数合成 `Cf_total = Cpe_windward·Kz − Cpe_leeward`（既定 `0.8·Kz + 0.4`、内圧 Cpi 相殺）
- 層水平力 `F = q · Cf_total · 見付け幅 · 負担高さ`

**実装**：`wind::kz_for_interval`（`wind.rs:60`）と `wind::wind_forces` が算定する。既定は `cpe_windward=0.8`、`cpe_leeward=-0.4` である。solver 側の `wind::wind_static`（`statics/analysis/wind.rs`）が建物高さ `H = 最上階床高 + パラペット/2`、各層負担区間、階別見付け幅を算定し、剛床へ重量比按分する。

**整合性**：doc に RESP-D マニュアル「風荷重の計算」を引用し、内圧相殺 `0.8·Kz+0.4` の導出をコメントに明記している。PH 階への風荷重接続は未対応であり、残課題として明記している。

## 1.3 積載荷重

### 1.3.1 柱の積載荷重低減（支持床数による低減）

柱の積載荷重は、建築基準法施行令85条2項に基づき、支える床の数に応じて柱および基礎の積載荷重を低減する。

**算定式（低減率表）**：支持床数 1→1.0、2→0.95、3→0.90、4→0.85、5→0.80、6→0.75、7→0.70、8→0.65、9以上→0.60。

**実装**：`squid_n_load::live_load::column_live_load_reduction`（`live_load.rs:30`）が算定し、支持床数は `floors_supported_by_column` が求める。`Model.load_cfg.live_load_reduction`（既定 false、RESP-D と同じく既定では低減しない）が有効なときのみ適用する想定である。

**整合性**：doc に「令85条2項」を明記しており、低減率表は令85条の表と一致する。

## 1.4 床荷重の分配（スラブ → 梁・柱）

床荷重の分配は、負担面積法と45°振り分けによる三角形・台形分布の力学に基づいて算定する。RESP-D マニュアル「床荷重」「片持ちスラブ」「出隅の片持ちスラブ」に準拠し、総和保存（Σ梁・小梁・柱荷重 = w×面積）を全経路で満たす。

**算定式（主な経路）**
- **三角形・台形（TriTrapezoid）**: 45°振り分け。正方形は各辺三角形分布 `w0 = w·L/2`。
  矩形は短辺三角形・長辺台形。
- **一方向（OneWay）**: 負担辺へ `w_line = w·スパン/2` の等分布。
- **負担面積等分布換算（TributaryArea）**: 短辺 `w·short/4`、長辺 `w·(short·long/2 − short²/4)/long`。
- **多角形負担面積法**: 200×200 格子で各セルを最近接辺へ帰属し辺荷重を算定（凸多角形で誤差 1% 未満）。
- **片持ち・出隅**: 片持ちは辺へ `w·出し幅` の等分布、出隅は構造芯〜先端の矩形重量を節点集中荷重で柱へ。

**実装**：`squid_n_load::floor`（`floor/mod.rs`）が算定する。`distribute_slab` が形状と手法によって経路を分岐する。

**整合性**：各経路とも総和保存を設計要件とし、台形と三角形は極限（a→L/2, a→0）で理論値に一致することを doc に明記している。小梁二段階伝達など簡易化した仮定は doc に明示している。

### 1.4.1 固定端モーメント・せん断力（CMQ）

固定端モーメントとせん断力（CMQ）は、両端固定梁の固定端力の閉形式に基づいて算定する。[V&V](../v_and_v/README.md)で総和保存と固定端値を検証している。

**算定式**
- 等分布: `C = ±w·L²/12`、`Q = w·L/2`
- 三角形: `C = ±5·w0·L²/96`、`Q = w0·L/4`
- 台形: `FEM = (1/L²)∫₀ᴸ w(x)·x·(L−x)² dx` を閉形式評価
- 一般（非対称・剛域考慮）: `FEM_i = (1/L²)∫w(ξ)ξ(L−ξ)²dξ`、`FEM_j = (1/L²)∫w(ξ)ξ²(L−ξ)dξ`、
  `Q = R0 ± (FEM_i − FEM_j)/L`（点・分布は閉形式、その他は合成シンプソン則）

**実装**：`floor::{fem_uniform, fem_triangle, fem_trapezoid, cmq_flexible_span, cmq_with_rigid_zone}`（`floor/mod.rs`）が算定する。剛域考慮は3モード（剛域外力を CMQ 加算、柱へ伝達、無視）である。

**整合性**：台形式は a→L/2 で三角形式、a→0 で等分布式に一致する（検算コメントあり）。剛域考慮は RESP-D マニュアル「剛域の考慮」に準拠する。

## 1.5 荷重組合せ

荷重組合せは、建築基準法施行令82条（標準荷重組合せ）および令86条（多雪区域）に基づいて設定する。

**算定式（組合せ）**
- 長期: `G + P`（多雪区域: `G + P + δ1·S`）
- 短期積雪: `G + P + S`
- 短期地震: `G + P ± K`（多雪区域: `G + P + δ3·S ± K`）
- 短期暴風: `G + P ± W`（多雪区域: `G + P + δ2·S ± W`）
- 積雪低減係数 既定 `δ1 = 0.7`、`δ2 = δ3 = 0.35`

**実装**：`squid_n_load::combo`（`combo.rs`）の `standard_combinations` と `SnowFactors` が算定する。長期と短期の判別は `is_short_term_combo` が行い、K/E/W を含むか、または S 項係数が 1.0 以上なら短期とする。

**整合性**：doc に「令82条」「令86条・82条」および RESP-D マニュアル 04 断面検定「荷重の組合せ」を明記している。δ1/δ2/δ3 は[原典照合リスト](../specs/原典照合リスト.md)で令86条と整合として照合済（☑）である。

## 1.6 地震用重量・層データの生成

地震用重量と層データは、令88条に基づき固定荷重と積載荷重の合計として生成する。自重の梁せん断力（CMoQo による Q0）の節点配分は RESP-D マニュアル「地震荷重の計算」「柱梁自重」に準拠する。

**算定式（概要）**
- 線材自重 `w = ρ·A·L_eff·g·factor`（RC/SRC 大梁は柱面間距離、S 梁は節点間距離）
- 壁・シェル自重 `w = ρ·t·(A − 開口)·g`（Newell の公式で面積算定）
- 仕上げ・付加線重量・雑壁重量を節点へ配分し、階ごとに集計
- 剛床代表節点は慣性力重心（重量重み付き重心）に自動生成

**実装**：`squid_n_load::story_gen`（`story_gen/mod.rs`）が算定する。鋼材単位体積重量は `γs = 77 kN/m³`（RESP-D マニュアル準拠、コメントに明記）、回転慣性重量は `Σ W·ir²` である。

**整合性**：各処理の doc に RESP-D マニュアルの該当節を引用している。単純梁静定反力と自重長さの取扱い（柱面間距離／節点間距離、下階柱なし時の付加）は実装 doc に明示している。
