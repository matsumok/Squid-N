# 8.7 終局検定の統括

部材別のせん断、付着、軸の余裕度を統括して算定する（保有水平耐力計算における部材種別判定・余裕度検定の実務的取扱い）。
両端ヒンジは \\( Q_{mu} = \text{上限強度倍率} \cdot (M_{u\text{上}} + M_{u\text{下}})/\text{内法} \\) とし、余裕度 \\( Q_{su}/Q_{mu}, Q_{bu}/Q_{mu} \ge 1.0 \\) で OK と判定する。

**算定式**：部材別にせん断、付着、軸の余裕度を算定する。
梁の余裕率は長期せん断力 QL を分子から控除した \\( \frac{Q_{su} - Q_L}{Q_{mu}} \\)・\\( \frac{Q_{bu} - Q_L}{Q_{mu}} \ge 1.0 \\) とする。
せん断補強筋に MK785/SPR785/SPR685 を使用した部材では、控除する QL を長期荷重による単純梁せん断力 Q0 に読み替える（各製品の技術評定の規定。Q0 は重力ケースの部材荷重から算定）。
柱の 2 軸せん断余裕度は \\( 1/((Q_{mx}/Q_{ux})^2 + (Q_{my}/Q_{uy})^2)^{1/2} \\) とする。
曲げ Mu は at 式（`rc_mu_simple`）または ACI を切り替え、せん断は塑性理論式（既定）または靭性指針式(6.4)を切り替える。
接合部終局の有効幅は \\( b_{ai} = \min(b_i/2, D/4) \\)（許容応力度検定の「大きい方」とは異なる）。

**実装**：`ultimate::mod`（`collect_rc_ultimate_checks`）。

**整合性**：詳細と未実装項目（靭性指針式 Vu の一部、二軸曲げ余裕度、プッシュオーバー応答の直接反映）は[終局検定_参照実装照合](https://github.com/hrntsm/squid-n/blob/main/v_and_v/終局検定_参照実装照合.md)を参照。
