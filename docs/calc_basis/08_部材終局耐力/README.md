# 8. 部材終局耐力

本章は、保有水平耐力計算、終局検定、非線形解析で用いる部材終局耐力の算定式と出典をまとめる。
対象は、曲げ Mu、曲げ終局時せん断 Qmu、終局せん断 Qsu、付着割裂 Qbu、軸終局 Nu、接合部終局、耐震壁非線形、CFT 終局である。

法令はこれらの終局耐力式を直接与えていない。
そのため本章の各式は、日本建築学会の各指針（靭性保証型、終局強度型）および『建築物の構造関係技術基準解説書』の略算式と塑性理論式に主に依拠する（Category B）。
多くの係数には、コード上で「要・原典照合（[原典照合リスト](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md)）」の申し送りが付されている。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

> **単位系の注意:** 荒川式や耐震壁 Qc/βu などは、工学単位系（kgf/cm²）で導出された実験式である。
> 実装では \\( 1\ \mathrm{kgf/cm^2} = 0.0980665\ \mathrm{N/mm^2} \\)、\\( 1\ \mathrm{kgf} = 9.80665\ \mathrm{N} \\) で換算して評価する。
