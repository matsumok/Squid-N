# 3. 断面性能

本章では、断面諸量（断面積 A、断面二次モーメント I、断面係数 Z、塑性断面係数 Zp、ねじり定数 J、せん断有効断面積 As）、せん断形状係数、コンクリートのヤング係数、SRC/CFT の等価断面性能、およびファイバー断面の算定方法を示す。断面公式は力学の閉形式、ヤング係数は RC規準の Ec 式、SRC/CFT の等価剛性はヤング係数比による累加（SRC規準の考え方）に基づく。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

実装本体は `squid_n_core::section_shape`（`crates/squid-n-core/src/section_shape/mod.rs`）にある。`squid_n_section::shape` はこれへ再エクスポートしている。

## 3.1 主要定数

| 定数 | 値 | 意味 | 根拠 |
|------|----|----|----|
| `N_S_EQ` | 15.0 | SRC 鉄骨の等価ヤング係数比（材料から算定できない場合のフォールバック。値は暫定既定・要照合） | ヤング係数比 n = Es/Ec（式形は力学） |
| `KAPPA_RC` | 1.2 | RC のせん断形状係数 κ（As = A/κ） | 力学（矩形断面の慣用値） |
| `E_STEEL` | 205000 N/mm² | 鋼材ヤング係数 | 力学（材料定数） |
| `NU_STEEL` | 0.3 | 鋼材ポアソン比 | 力学（材料定数） |
| `NU_CONCRETE` | 0.2 | コンクリートポアソン比 | 力学（材料定数） |
| `GAMMA_CONCRETE` | 23.0 kN/m³ | Ec 算定用の単位体積重量（普通コンクリートの気乾単位体積重量。値は暫定既定・要照合） | RC規準（Ec 式の γ） |

## 3.2 断面諸量（弾性）

各断面形状の A、I、Z、J は力学の閉形式（標準的な断面公式）で算定する。

**算定式（代表例）**:

- H 形 断面積 `A = 2·B·tf + (H − 2tf)·tw`
- H 形 強軸 `Iy = (B·H³ − (B − tw)·hw³)/12`（hw = H − 2tf）
- H 形 塑性断面係数 `Zp = B·tf·(H − tf) + tw·(H − 2tf)²/4`
- 箱形 閉断面ねじり（Bredt）`J = 4·A0²·t/周長`
- 円形鋼管 `A = π(r² − ri²)`、`J = π/2·(r⁴ − ri⁴)`
- 矩形 RC `A = b·d`、`Iy = b·d³/12`、ねじり `J = rect_torsion_j(b, d)`

**実装**：`section_shape::{calc_area, calc_iy, calc_iz, calc_j, plastic_modulus_strong}` が算定する。非対称断面（山形、溝形、T 形）は図心を求めて平行軸定理で合成する。

**整合性**：各式は標準的な断面公式に一致する。塑性断面係数はテストで H 形 `Zp = B·tf·(H − tf) + tw·(H − 2tf)²/4` を照合している。矩形ねじり定数は次項で扱う。

### 3.2.1 矩形ねじり定数

矩形断面のねじり定数 J は、Saint-Venant のねじり理論による矩形断面の級数近似式で算定する。

**算定式**：`J = (b³·h/16)·[16/3 − 3.36·(b/h)·(1 − (1/12)(b/h)⁴)]`（b = 短辺、h = 長辺）。

**実装**：`section_shape::rect_torsion_j` が算定する。

**整合性**：b/h → 0 で β → 1/3 に漸近することを確認している（薄板の理論値と一致）。

## 3.3 せん断有効断面積 As

せん断有効断面積 As は、せん断形状係数 κ による標準的な取扱いで算定する。RC/SRC は `As = A/κ`（κ = 1.2）とし、S 造はウェブ・フランジ断面（強軸はウェブ、弱軸はフランジ）から `As = Aw/κ`（κ = 1.0）とする。

**算定式（代表例）**:

- H 形 `as_y（弱軸）= 2·B·tf`、`as_z（強軸）= H·tw`
- 矩形 RC `as = b·d/1.2`
- 円形鋼管 `as = A/2`

**実装**：`section_shape::to_section` が算定する。強軸曲げ Iy と z 方向せん断 as_z、弱軸曲げ Iz と y 方向せん断 as_y を対応させる規約に従う（P1 §4.1）。

**整合性**：RC の κ = 1.2、および矩形せん断の k = 6/5（As = 5A/6）は、矩形断面のせん断形状係数の力学値に整合する。

---

## 3.4 コンクリートのヤング係数 Ec

コンクリートのヤング係数 Ec は、RC規準の Ec 式で算定する。基本形は `Ec = 3.35×10⁴·(γ/24)²·(Fc/60)^{1/3}` である。

**算定式**：`Ec = 3.35×10⁴·(23/24)²·(Fc/60)^{1/3}`（γ = 23 kN/m³ 固定、Fc ≤ 0 は 0 とする）。

**実装**：`section_shape::concrete_young_modulus` が算定する。

**整合性**：RC規準式に一致する。ただし γ = 23 固定については[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）である。

---

## 3.5 耐震壁のせん断形状係数 κ

耐震壁のせん断形状係数 κ は、側柱付き壁を I 形断面とみなしたせん断形状係数（力学）で算定する。

**算定式**：`κ = 3(1+ξ)/(5·(1 − ξ³(1−η))²)·[η + ξ(1−η)·((15/8)(1 − ξ²)² − ξ⁴·η)]`。

**実装**：`section_shape::wall_shear_shape_factor` が算定する。ここで ξ は壁板内法長さ/側柱外面間全長、η は壁厚/側柱幅を表す。退化時（denom ≤ 0 等）は KAPPA_RC = 1.2 にフォールバックする。

**整合性**：要注意。ξ、η の定義は原典ページに明示がなく、実装は仮定を置いている。ξ = 1 で矩形の κ = 1.2 に一致することのみ確認済みである。[原典照合リスト](../specs/原典照合リスト.md)に要照合（☐）として登録している。

## 3.6 SRC / CFT の等価断面性能

SRC/CFT の等価断面性能は、ヤング係数比による等価換算断面の累加（SRC規準の考え方）で算定する。SRC は `An = rcAn + sAn·(ns−1)`、`Ie = rcIe + sIe·(ns−1)`、`As = rcAs + sAs·(ngs−1)`、`J = cJ + (sG/cG)·sJ` によって求める。CFT は SRC 柱に準じ、鋼基準の 1/n 換算で累加する。

**算定式**:

- `ns = E_steel/Ec`、`ngs = ns·(1 + νc)/(1 + νs)`（νs = 0.3、νc = 0.2）
- SRC 軸剛性用 `A = b·d + (ns−1)·s_a`、曲げ `Iy = b·d³/12 + (ns−1)·s_iy`
- CFT（充填コンクリートを鋼基準へ換算）`A = A_steel + c_a/n`、`Iy = Iy_steel + c_iy/n`

**実装**：`section_shape::{src_equivalent_props, cft_equivalent_props}` が算定する。ns の暫定既定は N_S_EQ = 15 とする。

**整合性**：SRC の複合ねじり換算 `J = cJ + (sG/cG)·sJ` は上式のとおり実装済みである。ns はまず材料から `Es/Ec` を算定し、算定できない場合のみ暫定既定 `N_S_EQ = 15` にフォールバックする（値は[原典照合リスト](../specs/原典照合リスト.md)で要照合）。

## 3.7 ファイバー断面（M–φ 積分）

ファイバー断面の断面力と接線剛性は、平面保持仮定に基づくファイバー断面積分（設計書 §9.2）で算定する。

**算定式**:

- ファイバーひずみ `ε = ε0 − κz·y + κy·z`（平面保持）
- 断面力 `N = Σσ·A`、`My = Σσ·A·z`、`Mz = Σ(−σ·A·y)`
- 接線剛性 3×3 は各ファイバーの `Et·A`、`Et·A·y`、`Et·A·z` の総和として求める

**実装**：`squid_n_section::fiber::section_response`（`fiber.rs`）が算定する。各ファイバーは独立履歴状態を保持する（設計書 §6.3）。矩形分割は `rect_fiber_section` による。M–φ / M–θ 曲線は `mn_surface` で求める（弾完全塑性、二分法で軸力整合、`θ(M) = M·L/(6·EI0) + Lp·(φ(M) − M/EI0)`）。

**整合性**：弾性域を機械精度（1e-9）で検証し、手計算 `My = at·σy·j`、`Mu = 0.9·at·σy·j` を照合している（V&V #21、#22）。M–N 相関曲面（`mn_surface`）は全塑性支持点法で剛塑性降伏曲面を構築する。
