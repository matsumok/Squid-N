# 3. 断面性能

断面諸量（断面積 A・断面二次モーメント I・断面係数 Z・塑性断面係数 Zp・ねじり定数 J・せん断有効
断面積 As）、せん断形状係数、コンクリートのヤング係数、SRC/CFT の等価断面性能、およびファイバー断面。
剛性計算に用いる断面性能は RESP-D マニュアル「計算編 02 剛性計算」に準拠する。

> 記載フォーマット: 📖 根拠 / ∑ 算定式 / 💻 実装 / ✓ 整合性

実装本体は `squid_n_core::section_shape`（`crates/squid-n-core/src/section_shape/mod.rs`）。
`squid_n_section::shape` はこれへの再エクスポート。

## 3.1 主要定数

| 定数 | 値 | 意味 | 根拠 |
|------|----|----|----|
| `N_S_EQ` | 15.0 | SRC 鉄骨の等価ヤング係数比（暫定既定） | RESP-D 計算編 02 |
| `KAPPA_RC` | 1.2 | RC のせん断形状係数 κ（As = A/κ） | RESP-D 計算編 02「剛性計算」 |
| `E_STEEL` | 205000 N/mm² | 鋼材ヤング係数 | — |
| `NU_STEEL` | 0.3 | 鋼材ポアソン比 | — |
| `NU_CONCRETE` | 0.2 | コンクリートポアソン比 | — |
| `GAMMA_CONCRETE` | 23.0 kN/m³ | Ec 算定用の単位体積重量 | RESP-D 計算編 02 |

## 3.2 断面諸量（弾性）

- 📖 **根拠**: 力学（各断面形状の A・I・Z・J の閉形式）。RESP-D 計算編 02「剛性計算」の規約に整合。
- ∑ **算定式（代表例）**:
  - H 形 断面積 `A = 2·B·tf + (H − 2tf)·tw`
  - H 形 強軸 `Iy = (B·H³ − (B − tw)·hw³)/12`（hw = H − 2tf）
  - H 形 塑性断面係数 `Zp = B·tf·(H − tf) + tw·(H − 2tf)²/4`
  - 箱形 閉断面ねじり（Bredt）`J = 4·A0²·t/周長`
  - 円形鋼管 `A = π(r² − ri²)`、`J = π/2·(r⁴ − ri⁴)`
  - 矩形 RC `A = b·d`、`Iy = b·d³/12`、ねじり `J = rect_torsion_j(b, d)`
- 💻 **実装**: `section_shape::{calc_area, calc_iy, calc_iz, calc_j, plastic_modulus_strong}`。非対称断面
  （山形・溝形・T 形）は図心を求めて平行軸定理で合成。
- ✓ **整合性**: 各式は標準的な断面公式に一致。塑性断面係数はテストで H 形
  `Zp = B·tf·(H − tf) + tw·(H − 2tf)²/4` を照合。矩形ねじり定数は次項。

### 3.2.1 矩形ねじり定数

- 📖 **根拠**: RESP-D 計算編 02「剛性計算」のねじり定数式（Saint-Venant 矩形の級数近似）。
- ∑ **算定式**: `J = (b³·h/16)·[16/3 − 3.36·(b/h)·(1 − (1/12)(b/h)⁴)]`（b = 短辺、h = 長辺）。
- 💻 **実装**: `section_shape::rect_torsion_j`。
- ✓ **整合性**: b/h → 0 で β → 1/3 に漸近することを確認。doc に RESP-D 計算編 02 を明記。

## 3.3 せん断有効断面積 As

- 📖 **根拠**: RESP-D 計算編 02「剛性計算」。RC/SRC は `As = A/κ`（κ = 1.2）、S 造はウェブ・フランジ
  断面（強軸はウェブ、弱軸はフランジ）で `As = Aw/κ`（κ = 1.0）。
- ∑ **算定式（代表例）**:
  - H 形 `as_y（弱軸）= 2·B·tf`、`as_z（強軸）= H·tw`
  - 矩形 RC `as = b·d/1.2`
  - 円形鋼管 `as = A/2`
- 💻 **実装**: `section_shape::to_section`。強軸曲げ Iy ↔ z 方向せん断 as_z、弱軸曲げ Iz ↔ y 方向
  せん断 as_y の対応規約（P1 §4.1）。
- ✓ **整合性**: RC の κ = 1.2、矩形せん断の k = 6/5（As = 5A/6）は力学・RESP-D 規約と整合。

## 3.4 コンクリートのヤング係数 Ec

- 📖 **根拠**: RC規準の Ec 式（RESP-D 計算編 02 記載形）。`Ec = 3.35×10⁴·(γ/24)²·(Fc/60)^{1/3}`。
- ∑ **算定式**: `Ec = 3.35×10⁴·(23/24)²·(Fc/60)^{1/3}`（γ = 23 kN/m³ 固定、Fc ≤ 0 は 0）。
- 💻 **実装**: `section_shape::concrete_young_modulus`。
- ✓ **整合性**: RC規準式に一致。γ = 23 固定は[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。

---

## 3.5 耐震壁のせん断形状係数 κ

- 📖 **根拠**: RESP-D 計算編 02「剛性計算」耐震壁（I 形断面のせん断形状係数）。
- ∑ **算定式**: `κ = 3(1+ξ)/(5·(1 − ξ³(1−η))²)·[η + ξ(1−η)·((15/8)(1 − ξ²)² − ξ⁴·η)]`。
- 💻 **実装**: `section_shape::wall_shear_shape_factor`。ξ = 壁板内法長さ/側柱外面間全長、η = 壁厚/側柱幅。
  退化時（denom ≤ 0 等）は KAPPA_RC = 1.2 にフォールバック。
- ✓ **整合性**: **要注意** — ξ・η の定義は原典ページに明示がなく実装は仮定を置いている。ξ = 1 で矩形の
  κ = 1.2 に一致することのみ確認済み。[原典照合リスト](../specs/原典照合リスト.md)に要照合（☐）として登録。

## 3.6 SRC / CFT の等価断面性能

- 📖 **根拠**: RESP-D 計算編 02「剛性計算」。SRC は `An = rcAn + sAn·(ns−1)`、
  `Ie = rcIe + sIe·(ns−1)`、`As = rcAs + sAs·(ngs−1)`、`J = cJ + (sG/cG)·sJ`。
  CFT は SRC 柱に準じ、鋼基準の 1/n 換算で累加する。
- ∑ **算定式**:
  - `ns = E_steel/Ec`、`ngs = ns·(1 + νc)/(1 + νs)`（νs = 0.3、νc = 0.2）
  - SRC 軸剛性用 `A = b·d + (ns−1)·s_a`、曲げ `Iy = b·d³/12 + (ns−1)·s_iy`
  - CFT（充填コンクリートを鋼基準へ換算）`A = A_steel + c_a/n`、`Iy = Iy_steel + c_iy/n`
- 💻 **実装**: `section_shape::{src_equivalent_props, cft_equivalent_props}`。ns の暫定既定は N_S_EQ = 15。
- ✓ **整合性**: doc に RESP-D 計算編 02 の累加式を明記。SRC のねじりは内蔵鉄骨寄与を近似（複合換算
  `J = (sG/cG)·sJ + cJ` は今後の課題として明記）。ns = 15 固定は[原典照合リスト](../specs/原典照合リスト.md)で要照合。

## 3.7 ファイバー断面（M–φ 積分）

- 📖 **根拠**: 平面保持仮定に基づくファイバー断面積分（設計書 §9.2）。RESP-D「05 非線形モデル」の
  ファイバーモデルに整合。
- ∑ **算定式**:
  - ファイバーひずみ `ε = ε0 − κz·y + κy·z`（平面保持）
  - 断面力 `N = Σσ·A`、`My = Σσ·A·z`、`Mz = Σ(−σ·A·y)`
  - 接線剛性 3×3 は各ファイバーの `Et·A`・`Et·A·y`・`Et·A·z` の総和
- 💻 **実装**: `squid_n_section::fiber::section_response`（`fiber.rs`）。各ファイバーは独立履歴状態
  （設計書 §6.3）。矩形分割 `rect_fiber_section`。M–φ / M–θ 曲線は `mn_surface`（弾完全塑性、
  二分法で軸力整合、`θ(M) = M·L/(6·EI0) + Lp·(φ(M) − M/EI0)`）。
- ✓ **整合性**: 弾性域を機械精度（1e-9）で検証、手計算 `My = at·σy·j`、`Mu = 0.9·at·σy·j` を照合
  （V&V #21・#22）。M–N 相関曲面（`mn_surface`）は全塑性支持点法で剛塑性降伏曲面を構築。
