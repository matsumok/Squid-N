# 9. 限界耐力計算

限界耐力計算（性能評価ルート）の根拠。建築基準法施行令 第82条の5、平成12年建設省告示第1457号を
根拠とし、等価1自由度化・加速度応答スペクトル・容量スペクトル法（性能点）・損傷限界／安全限界の判定で
構成される（仕様: [P12](../specs/P12_限界耐力計算.md)）。

> 記載フォーマット: 📖 根拠 / ∑ 算定式 / 💻 実装 / ✓ 整合性

> ⚠️ **実装状況の重要な注意:** 本章の理論・出典は仕様（P12）として確定しているが、
> **`capacity_spectrum.rs` は現時点でスタブ実装**であり、告示1457号の加速度応答スペクトル So・
> 表層地盤増幅 Gs・応答低減 Fh・容量スペクトル法の性能点算定は**コード上に未実装**である。
> 実装済みは (a) 簡易な等価1自由度変換（有効質量比 0.75 の暫定近似）、(b) `acc = Rt/T` 系の
> 簡易需要スペクトルのみ。詳細は下記「整合性」に明記する。

---

## 9.1 等価1自由度化（容量曲線）

- 📖 **根拠**: 建築基準法施行令 第82条の5、設計書 §11.1。プッシュオーバーの多自由度応答を代表変位
  Δ1・代表加速度 A1 の 1 自由度系へ縮約。
- ∑ **算定式（仕様 P12 §2 の閉形式）**:
  - 有効質量 `M1* = (Σmi·δi)²/Σmi·δi²`
  - 代表変位 `Δ1 = (Σmi·δi²)/(Σmi·δi)`
  - 代表加速度 `A1 = Qb/M1*`
- 💻 **実装**: `squid_n_design_jp::capacity_spectrum::to_equivalent_sdof`（`capacity_spectrum.rs`）。
- ✓ **整合性**: ⚠️ **実装は仕様と不一致（簡略化）。** 現在の実装は有効質量比を `mef = 0.75` の
  固定値とし `equiv_mass = total_mass·0.75`、`sd = d/mef` としており、仕様 P12 §2 の
  `M1* = (Σmi·δi)²/Σmi·δi²` による厳密なモード縮約は未実装。今後、仕様どおりの閉形式へ置換する必要がある。

---

## 9.2 加速度応答スペクトル So・表層地盤増幅 Gs

- 📖 **根拠**: 平成12年建設省告示第1457号。工学的基盤の加速度応答スペクトル So（3分岐）と、
  表層地盤による増幅 Gs。
- ∑ **算定式（告示1457号、仕様 P12 §3）**:
  - 安全限界 `So = 3.2 + 30T`（T<0.16）／`8.0`（0.16≤T≤0.64）／`5.12/T`（0.64<T）[m/s²]
  - 損傷限界 `So = 安全限界の 1/5`（`0.64 + 6T`／`1.6`／`1.024/T`）
  - 表層地盤増幅 `Gs`（地盤種別・略算/精算）
  - 応答加速度 `Sa = Z·Gs·So·Fh`
- 💻 **実装**: ⚠️ **未実装**。`capacity_spectrum.rs` に So・Gs の算定は存在しない。`demand_spectrum` は
  `acc = t < 0.1 ? 0.1/max(t, 0.02) : Rt/t` という簡易式で、告示スペクトルの形状・地盤増幅を反映しない。
- ✓ **整合性**: ⚠️ **未実装のため要実装**。So の 3 分岐値・損傷限界=安全限界の 1/5・Gs は
  [原典照合リスト 優先度5](../specs/原典照合リスト.md#優先度5限界耐力計算の加速度応答スペクトルv20p12)に
  照合対象として登録済み（☐）。実装時に告示1457号の原典と照合すること。

## 9.3 等価粘性減衰・応答低減 Fh

- 📖 **根拠**: 告示1457号。等価粘性減衰 h による応答低減。
- ∑ **算定式（仕様 P12 §4）**: `Fh = 1.5/(1 + 10h)`。
- 💻 **実装**: ⚠️ **未実装**。
- ✓ **整合性**: ⚠️ **未実装のため要実装**。Fh = 1.5/(1+10h) は[原典照合リスト](../specs/原典照合リスト.md)に

## 9.4 性能点（容量スペクトル法）と限界状態の判定

- 📖 **根拠**: 令82条の5、告示1457号。容量曲線（A1–Δ1）と需要スペクトル（減衰低減つき ADRS）の
  交点＝性能点（応答）。損傷限界（中地震）・安全限界（大地震）を判定。
- ∑ **算定式（仕様 P12 §5・§6）**: 容量スペクトルと需要スペクトルの交点を求め、性能点の応答変位・
  加速度が各限界状態の耐力・変形以内かを判定。
- 💻 **実装**: ⚠️ **未実装**（性能点の交点算定・限界状態判定ともに `capacity_spectrum.rs` に存在しない）。
- ✓ **整合性**: ⚠️ **未実装のため要実装**。V&V 索引でも限界耐力（#18）は ❌（未実装）。
