# 10. 免震・制振

本章では、免震支承材（積層ゴム、鉛プラグ入り、高減衰、すべり、転がり、球面すべり）、制振ダンパー（オイル／マクスウェル型、履歴型）、座屈拘束ブレース（BRB）の力学モデルと係数の根拠を示す。モデルは免震・制振部材の標準的な力学モデル（バイリニア・摩擦・マクスウェル要素等）、製品別の係数は各製品の技術資料（メーカー公表値・大臣認定値）に依拠する（Category B）。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

要素剛性としての組込みは[4.7 ばね要素](./04_要素剛性.md#47-ばね要素免震制振独立せん断節点ばね)に示し、設計計算は `squid_n_design_jp::{isolator, brb}` が担当する。

---

## 10.1 免震アイソレータ

### 10.1.1 マルチシア（多層積層ゴム）低減率

多層積層ゴム（マルチシア配置）の剛性および耐力の低減率を、配置の幾何（せん断ばねの方向余弦の和）に基づいて算定する。

**算定式**

- 剛性低減率 `= 2/n`
- 耐力低減率 `= 1/Σ_{i=1}^{n/2}(cos(π/n·(i−1)) + sin(π/n·(i−1)))`

**実装**：`isolator::{multi_shear_stiffness_reduction, multi_shear_strength_reduction}` が算定する。

**整合性**：本数別表（2/4/6/8/16）と一致することを照合済みである（[原典照合リスト](../specs/原典照合リスト.md)☑）。

### 10.1.2 摩擦（すべり）支承

すべり支承の最大水平力を、クーロン摩擦則により `Qmax = μ·N`（N は長期軸力、引張は 0）として算定する。

**実装**：`isolator::friction_max_force` が最大力を算定し、要素側では 2 次元摩擦として合力を Qmax に射影する（`springs::isolator`）。

**整合性**：2 次元摩擦（合力の Qmax への射影）を要素側で実装している（[原典照合リスト](../specs/原典照合リスト.md)☑）。

### 10.1.3 鉛プラグ入り積層ゴム（LRB）統一型・歪依存

鉛プラグ入り積層ゴム（LRB）の剛性と切片荷重について、ひずみ γ への依存性と 20℃基準の温度換算を、製品技術資料の歪依存特性式（Category B）に基づいて算定する。

**算定式**

- 剛性ひずみ依存係数 `CKd(γ)`（`γ<0.25: 0.779·γ^−0.43`、`0.25≤γ<1.0: γ^−0.25`、
  `1.0≤γ: γ^−0.12`。第1分岐の指数は分岐点 γ=0.25 での連続性から −0.43）
- 切片荷重ひずみ依存係数 `CQd(γ)`（3 分岐）
- 温度換算 `Kd(t) = Kd20·exp(−0.00271(t−20))`、`Qd(t) = Qd20·exp(−0.00879(t−20))`
- 等価水平剛性 `keq = Qd/δ + Kd`、等価粘性減衰 `Heq = (2/π)·Qd(δ − Qd/((β−1)Kd))/(keq·δ²)`

**実装**：`isolator::{lrb_stiffness_strain_factor, lrb_intercept_strain_factor, lrb_stiffness_at_temperature, equivalent_stiffness, equivalent_damping}` が算定する。

**整合性**：製品係数は Category B（外部データ確定対象）である。

### 10.1.4 転がり支承・球面すべり支承・高減衰ゴム

転がり支承、球面すべり支承、高減衰ゴムの各特性を、各製品の技術資料（ブリヂストン高減衰ゴム多項式ほか、Category B）に基づいて算定する。

**算定式（代表）**

- 転がり支承摩擦 `μ = (1.2 + 7.8·Pv/Po)/1000`
- 球面すべり支承 標準摩擦係数（MN型 `μ0 = 0.043(2.03σ^{−0.19} + 0.068)` ほか）、速度依存 `μ = μ0(1 − 0.55·e^{−0.019|V|})`
- 高減衰ゴム（ブリヂストン E6/E4）: Geq・Heq・U のひずみ多項式

**実装**：`isolator::{rolling_bearing_friction, spherical_bearing_mu0_mn, spherical_bearing_mu_mn, hdr_bridgestone_e6, hdr_bridgestone_e4}` が算定する。

**整合性**：各製品の係数は Category B である。

## 10.2 制振ダンパー

制振ダンパーの復元力特性を、標準的な力学モデル（マクスウェル要素・弾塑性バイリニア）で算定する。

**算定式**

- マクスウェル型（バネ Kd ＋粘性ダッシュポット `Fc = C0·sign(V)·|V|^α` の直列、後退 Euler）
- 履歴型（弾塑性バイリニア、鋼材系／鉛／U形ダンパー）

**実装**：`springs::damper`（`MaxwellDamperElement`、`HystereticDamperElement`）が算定する。

**整合性**：マクスウェル要素の等価接線剛性 `K_eff = Kd·C0/(C0 + Δt·Kd)` をテストで照合している。

## 10.3 座屈拘束ブレース（BRB）

座屈拘束ブレースの断面検定を、各製品の技術資料・大臣認定値（Category B）に基づいて算定する。製品は JFE シビル二重鋼管座屈補剛ブレースおよび日鉄エンジニアリングアンボンドブレースを対象とし、メーカー許容値を直接使用する（座屈考慮 fc は用いない。座屈拘束により全断面降伏まで座屈しない製品特性のため）。

**算定式**

- 軸力検定比 `|N| / Na`（`Na` = 短期許容軸力、長期は `/1.5`）
- 座屈長さ検定比 `Lk / 限界長さ`（`Lk = 部材長 − 2·端部低減`）
- 総合 `ratio = max(軸力比, 座屈長さ比)`

**実装**：`brb`（`brb.rs::brb_check`）が算定する。

**整合性**：doc に製品名を明記している。メーカー許容値は Category B である。
