# 8. 部材終局耐力

本章は、保有水平耐力計算、終局検定、非線形解析で用いる部材終局耐力の算定式と出典をまとめる。対象は、曲げ Mu、曲げ終局時せん断 Qmu、終局せん断 Qsu、付着割裂 Qbu、軸終局 Nu、接合部終局、耐震壁非線形、CFT 終局である。

法令はこれらの終局耐力式を直接与えていない。そのため本章の各式は、日本建築学会の各指針（靭性保証型、終局強度型）および『建築物の構造関係技術基準解説書』の略算式と塑性理論式に主に依拠する（Category B）。多くの係数には、コード上で「要・原典照合（[原典照合リスト](../specs/原典照合リスト.md)）」の申し送りが付されている。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

> **単位系の注意:** 荒川式や耐震壁 Qc/βu などは、工学単位系（kgf/cm²）で導出された実験式である。実装では `1 kgf/cm² = 0.0980665 N/mm²`、`1 kgf = 9.80665 N` で換算して評価する。

## 8.1 RC 部材 略算終局耐力（曲げ・せん断）

実装本体は `squid_n_core::rc_capacity` にあり、design-jp はこれを再エクスポートする。全係数に「要・原典照合」のコメントが付されている。

### 8.1.1 曲げ終局モーメント（梁）

引張鉄筋降伏型の梁について、曲げ終局モーメントを略算する。AIJ 非線形解析指針等の簡易式および技術基準解説書 P.623 に基づく。

**算定式**：`j = 7·d_e/8`、`Mu = 0.9·at·σy·j`。

**実装**：`rc_capacity::rc_mu_simple`。

**整合性**：ファイバー照合により `Mu = 0.9·at·σy·j` を確認した（V&V #22）。

### 8.1.2 曲げ終局時せん断力（梁）

両端が曲げ降伏し反曲点が中央にあると仮定して、曲げ終局時のせん断力を略算する。

**算定式**：`Qmu = 2·Mu/h0`。

**実装**：`rc_capacity::rc_qmu_simple`。

**整合性**：略算仮定を doc に明記している。

### 8.1.3 柱の軸力考慮終局曲げ

柱の終局曲げモーメントを軸力の大小に応じて算定する。2007年版『建築物の構造関係技術基準解説書』付録1-3 の閉形式に基づく（要・原典照合）。

**算定式**（3分岐、`N_bal = 0.4·b·D·Fc`）:

- `N > N_bal`: `Mu = (0.8·at·σy·D + 0.12·b·D²·Fc)·(Nmax − N)/(Nmax − N_bal)`
- `0 ≤ N ≤ N_bal`: `Mu = 0.8·at·σy·D + 0.5·N·D·(1 − N/(b·D·Fc))`
- `N < 0`: `Mu = 0.8·at·σy·D + 0.4·N·D`

**実装**：`rc_capacity::rc_column_mu_simple`。RESP-D マニュアル 04 の柱設計用 `QD1 = ΣcMy/h'` に使用する。

**整合性**：技術基準解説書付録の式であり、[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。

### 8.1.4 終局せん断耐力（荒川 mean 式系）

梁のせん断終局耐力を荒川 mean 式系の略算式で算定する。靭性保証型指針および技術基準解説書に基づく（要・原典照合）。

**算定式**:

$$Q_{su} = \left\{ \frac{0.068\,p_t^{0.23}(F_c+18)}{M/(Q\,d_e)+0.12} + 0.85\sqrt{p_w \sigma_{wy}} + 0.1\,\sigma_0 \right\} b\,j$$

- `pt = 100·at/(b·d_e)` [%]、`j = 7·d_e/8`
- せん断スパン比 `M/(Q·d_e) = h0/(2d_e)` を 1.0〜3.0 にクランプ
- `pw` を 0〜0.012（1.2%）にクランプ、`σ0` を 0〜0.4Fc にクランプ（引張は 0）

**実装**：`rc_capacity::rc_qsu_simple`。係数は 0.068/0.23/+18/+0.12/0.85/0.1。

**整合性**：係数とクランプ範囲は[原典照合リスト 優先度1](../specs/原典照合リスト.md#優先度1ds構造特性係数-最重要形が複雑)で要照合（☐）。

### 8.1.5 曲げ降伏時剛性低下率 αy（菅野式）

トリリニア骨格の降伏点変形を求めるため、曲げ降伏時の剛性低下率 αy を菅野式で算定する。梅村魁『鉄筋コンクリート建物の動的耐震設計法』P.106-108 および RESP-D「05」に基づく。

**算定式**（a/D 2分岐、[1,5] クランプ）:

- `2 ≤ a/D ≤ 5`: `αy = (0.043 + 1.635·n·pt + 0.043·(a/D))·(d/D)²`
- `1 ≤ a/D < 2`: `αy = (−0.0836 + 0.159·(a/D))·(d/D)²`

**実装**：`rc_capacity::rc_alpha_y_sugano`。降伏点変形 θy = θe/αy に使用する。

**整合性**：梅村魁 P.106-108 として[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。

## 8.2 RC 終局せん断（塑性理論式・靭性指針式）

「06 終局検定」の照合のため、荒川式に加えて塑性理論式と靭性指針式を実装している。

### 8.2.1 塑性理論式 Qsu（トラス＋アーチ機構）

トラス機構とアーチ機構の累加により、RC 部材の終局せん断耐力を算定する。日本建築学会『靭性保証型耐震設計指針』（終局強度型設計指針、藤井・森田式系）に基づく。

**算定式**：`Qsu = truss + arch`

- `truss = b·jt·pw·σwy·cotφ`（`pw·σwy ≤ ν·Fc/2` に制約）
- `arch = k1·(1 − k2)·b·D·ν·Fc`
- 有効係数 `ν0 = max(0.7 − Fc/200, 0)`、`ν = factor·ν0`（factor は塑性率 Rp 依存）
- `cotφ`（Rp 依存、1.0〜2.0）、`k1 = (√((L/D)²+1) − L/D)/2`、`k2 = min(2pw·σwy/(ν·Fc), 1)`
- 軽量コンクリートは 0.9 倍

**実装**：`ultimate::rc_shear`（`rc_shear.rs::rc_shear_qsu_plastic`）。

**整合性**：靭性保証型指針を doc に明記している。RESP-D「06 終局検定」共通事項の軽量 0.9 倍を実装した。

### 8.2.2 付着割裂耐力 Qbu

付着割裂で耐力が決まる場合の Qbu を算定する。靭性保証型指針 P.175-181 に基づく。

**算定式**：`Qbu = jt·τbu·Σφ + k1·(1 − k3)·b·D·ν·Fc`。付着信頼強度は `τbu = αt·((0.085·b1 + 0.10)·√Fc + kst)`。

**実装**：`ultimate::rc_shear`（`rc_shear_qbu_bond`、`bond_reliable_strength_deformed`）。

**整合性**：靭性保証型指針のページを doc に明記している。

### 8.2.3 靭性指針式せん断信頼強度 Vu / Vbu

トラス項とアーチ項からせん断信頼強度 Vu を算定する。日本建築学会『鉄筋コンクリート造建物の靭性保証型耐震設計指針・同解説』(1997) 式(6.4.1)〜(6.4.11)、付着(6.8.14)〜(6.8.16) に基づく。

**算定式**：`Vu = min(Vu1, Vu2, Vu3)`（トラス項＋アーチ項）。`tanθ` は L/D に応じて `0.9D/(2L)`（式6.4.10）または `(√(L²+D²) − L)/D`（式6.4.11）とする。

**実装**：`ultimate::rc_shear_ductility`（`rc_shear_ductility.rs`）。式番号をコメントに明記している。

**整合性**：tanθ の符号は原典照合により √(L²+D²) と確定した（境界 L/D=1.5 で連続することを根拠とし、doc に明記）。

### 8.2.4 柱の軸終局耐力・ACI 曲げ終局

柱の軸終局耐力と、等価応力度ブロック法による曲げ終局モーメントを算定する。軸終局 Nuc/Nut は技術基準解説書、曲げ Mu は ACI318-95 に基づく。

**算定式**：`Nuc = b·D·Fc`（圧縮正）、`Nut = −ag·σy`（引張負）。ACI 曲げは、終局ひずみ `εcu = 0.003` と応力ブロック β1 のもとで中立軸 c を二分法により軸力整合させ、`Mu = cc(center − a/2) + Σσ·A·(center − di)` で求める。

**実装**：`ultimate::{rc_axial, rc_column_aci}`。

**整合性**：技術基準解説書 P.626-627/629-630 および ACI318-95 を doc に明記している。

## 8.3 RC 柱梁接合部 終局耐力

柱梁接合部のせん断終局耐力とパネル余裕度を算定する。RESP-D「06 終局検定」接合部および靭性保証型指針（`Fj = 0.8·Fc^0.7`）に基づく。

**算定式**：`Vju = κ·φ·Fj·bj·Dj`。形状係数は κ = 十字1.0/T字・ト字0.7/L字0.4 とし、`Qdu = α·(T + T' − Qcu)`。

**実装**：`ultimate::joint`（`joint.rs::rc_joint_ultimate`）。

**整合性**：`Vju = κ·φ·Fj·bj·Dj` の各項を実装した。Fj はマニュアルで定義式が省略されているため、靭性指針の標準式を採用した（doc 明記）。

## 8.4 RC 部材 非線形復元力特性

### 8.4.1 梁（曲げ・せん断・軸）

梁の曲げ、せん断、軸のトリリニア骨格を組み立てる。技術基準解説書 P.621-623（`Mc = κ·Fc·Ze`、κ=0.56）、菅野式（剛性低下率）、RESP-D「05 非線形モデル」に基づく。

**算定式**：曲げひび割れ `Mc = 0.56·Fc·Ze`、せん断ひび割れ `Qc = (0.061(Fc+49)/(M/Qd+1.7))·b·j`、軸 `Nct = 0.56·Fc·Ac`、`Nut = at·σy`、`Nuc = at·σy + Fc(Ac − at)`。これらからトリリニア骨格を組み立てる。

**実装**：`rc::beam_nonlinear`（`beam_nonlinear.rs`）。

**整合性**：各式の出典（技術基準解説書ページ、RESP-D「05」）を doc に明記している。κ=0.56 とせん断ひび割れ係数は[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。

### 8.4.2 耐震壁（せん断非線形トリリニア）

耐震壁のせん断非線形トリリニア骨格を組み立てる。技術基準解説書 P.635-639（Qc・βu）、荒川式および技術基準解説書 P.281-282（終局せん断 Qu）、RC終局強度設計資料 P.132（開口低減）に基づく。

**算定式**:

- せん断ひび割れ `Qc = (0.043·pg + 0.051)·√Fc·Aw`（工学単位）
- せん断降伏時剛性低下率 `βu = 0.46·pw·σy/Fc + 0.14`
- 終局せん断 `Qu = {k·pte^0.23(Fc+18)/(M/QD+0.12) + 0.85√(σwh·pwh) + 0.1σ0}·te·j·r`（k = 0.053／高強度0.068、pwh ≤ 1.2%、M/QD ∈ [1,3]、σ0 ∈ [0, 0.4Fc]）
- 開口低減 `r = 1 − max(r0, l0/lw, h0/h)`、`r0 = √(h0·l0/(h·lw))`

**実装**：`rc::wall_nonlinear`（`wall_nonlinear.rs`）。

**整合性**：各式の出典ページを doc に明記している。係数は[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。本実装は RESP-D「05」照合で新規実装し、設計と結果表示へ配線した（V&V #15/#16）。

### 8.4.3 鋼梁（横座屈 Mcr/Mp）

横座屈で耐力が決まる H形鋼梁について、横座屈耐力比から Mcr を算定する。鋼構造塑性設計指針に基づき、`Mp = Zp·σy`。

**算定式**：`x = lb·h/Af`。材料別のパラメータ (e1, e2, slope, tail) により 3 分岐する（SN400: 300/835/0.00075/500、SN490: 220/605/0.0010/363、一般: 70500/σy 等）。`Mcr = ratio·Mp`。

**実装**：`steel::beam_nonlinear`（`beam_nonlinear.rs::steel_beam_lateral_buckling_ratio`）。

**整合性**：鋼構造塑性設計指針を doc に明記している。表と一般式は[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。

## 8.5 SRC / SRRC 梁 せん断終局（充腹・非充腹）

SRC/SRRC 梁のせん断終局耐力を、充腹と非充腹に分けて算定する。技術基準解説書（SRC 梁せん断終局）と SRC規準（SRC 梁せん断終局）に基づき、非充腹は荒川 mean 式系による。

**算定式**:

- 充腹（累加）`Qu = rQu + sQu`。`rQu`（コンクリート・RC）に `sQu`（鉄骨ウェブ `sAw·sσy/√3`）を累加する。
- 非充腹 格子材 `Qsu = {κ·pt^0.23·kcs·(18+Fc)/(M/Qd+0.12) + 0.85√(rpw·rσwy) + 0.5√(spw·sσwy)}·be·j`（pt = rpt + spt、j = 0.8D、M/Qd ∈ [1,3]、κ = 0.053／高強度0.068）
- 非充腹 ラチス材 `Qsu = {…}·be·rj + sQu`

**実装**：`srrc::beam_nonlinear`（`beam_nonlinear.rs`）。

**整合性**：技術基準解説書、SRC規準、荒川 mean 式系を doc に明記している。係数は[原典照合リスト](../specs/原典照合リスト.md)で要照合（☐）。

## 8.6 CFT 柱 終局耐力

CFT 柱の終局耐力を柱分類に応じて算定する。日本建築学会『コンクリート充填鋼管（CFT）構造設計・施工指針』に基づき、柱を短柱(lk≤4D)、中柱(≤12D)、長柱(>12D) に分類する。

**算定式**:

- 短柱軸終局 `Ncu1 = cA·Fc + (1+ξ)·sA·Fy`（ξ = 円形0.27/角形0）
- コンクリート座屈応力 cσcr（規準化細長比 cλ1 の 2 分岐、cλ1=1 で連続）
- N-M 相互作用（角形・円形の (Nu, Mu) 曲面）を用いる。中柱と長柱は座屈低減 R を乗じる。

**実装**：`ultimate::{cft, cft_nm}`、統括は `ultimate::mod::collect_cft_ultimate_checks`。

**整合性**：CFT 指針を doc に明記している。sNcr 分岐条件、β2、角形 sMu の Fy 採用など一部は要原典照合として doc に明記した（OCR 誤りと判断した箇所は、その根拠も記載している）。

## 8.7 終局検定の統括

部材別のせん断、付着、軸の余裕度を統括して算定する。RESP-D「計算編 06 終局検定」に基づく。両端ヒンジは `Qmu = 上限強度倍率·(Mu上+Mu下)/内法` とし、余裕度 `Qsu/Qmu, Qbu/Qmu ≥ 1.0` で OK と判定する。

**算定式**：部材別にせん断、付着、軸の余裕度を算定する。柱の 2 軸せん断余裕度は `1/((Qmx/Qux)² + (Qmy/Quy)²)^{1/2}` とする。曲げ Mu は at 式（`rc_mu_simple`）または ACI を切り替え、せん断は塑性理論式（既定）または靭性指針式(6.4)を切り替える。

**実装**：`ultimate::mod`（`collect_rc_ultimate_checks`）。

**整合性**：詳細と未実装項目（靭性指針式 Vu の一部、二軸曲げ余裕度、プッシュオーバー応答の直接反映）は[終局検定_RESP-D照合](../v_and_v/終局検定_RESP-D照合.md)を参照。
