# 9.2 加速度応答スペクトル So・表層地盤増幅 Gs

工学的基盤の加速度応答スペクトル So（3分岐）と、表層地盤による増幅 Gs を算定する。
根拠は平成12年建設省告示第1457号である。

**算定式**（告示1457号、仕様 P12 §3）:
- 安全限界 \\( S_0 = 3.2 + 30T \\)（\\( T < 0.16 \\)）／\\( 8.0 \\)（\\( 0.16 \le T \le 0.64 \\)）／\\( 5.12/T \\)（\\( 0.64 < T \\)）[m/s²]
- 損傷限界 \\( S_0 = \text{安全限界の } 1/5 \\)（\\( 0.64 + 6T \\)／\\( 1.6 \\)／\\( 1.024/T \\)）
- 表層地盤増幅 `Gs`（地盤種別・略算/精算）

応答加速度:

\\[ S_a = Z \cdot G_s \cdot S_0 \cdot F_h \\]

**実装**: ⚠️ 未実装である。
`capacity_spectrum.rs` に So・Gs の算定は存在しない。
`demand_spectrum` は `acc = t < 0.1 ? 0.1/max(t, 0.02) : Rt/t` という簡易式であり、告示スペクトルの形状や地盤増幅を反映しない。

**整合性**: ⚠️ 未実装のため要実装である。
So の3分岐値、損傷限界=安全限界の 1/5、Gs は、[原典照合リスト 優先度5](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md#優先度5限界耐力計算の加速度応答スペクトルv20p12)に照合対象として登録済み（☐）である。
実装時には告示1457号の原典と照合すること。
