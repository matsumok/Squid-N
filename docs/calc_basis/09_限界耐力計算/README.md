# 9. 限界耐力計算

限界耐力計算（性能評価ルート）の根拠を示す。
本計算は建築基準法施行令 第82条の5および平成12年建設省告示第1457号を根拠とし、多自由度応答の等価1自由度化、加速度応答スペクトルの算定、容量スペクトル法による性能点の算定、損傷限界および安全限界の判定で構成される（仕様: [P12](https://github.com/hrntsm/squid-n/blob/main/specs/P12_限界耐力計算.md)）。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

> ⚠️ **実装状況の重要な注意:** 本章の理論および出典は仕様（P12）として確定している。
> しかし **`capacity_spectrum.rs` は現時点でスタブ実装**であり、告示1457号の加速度応答スペクトル So、表層地盤増幅 Gs、応答低減 Fh、容量スペクトル法による性能点の算定は、いずれも**コード上に未実装**である。
> 実装済みなのは、(a) 有効質量比を 0.75 とする暫定近似による簡易な等価1自由度変換と、(b) \\( \text{acc} = R_t/T \\) 系の簡易需要スペクトルのみである。
> 詳細は各項目の「整合性」に明記する。
