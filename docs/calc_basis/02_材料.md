# 2. 材料構成則・材料強度

この章は、材料の応力ひずみ関係（構成則・履歴則）と、許容応力度設計や終局耐力計算の基礎となる材料強度（F値、降伏強度、コンクリート強度）の根拠を示す。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

---

## 2.1 一軸材料構成則（ファイバー断面用）

このソフトは一軸材料を `UniaxialMaterial` トレイト（`trial`/`commit`/`revert`）として実装し、ファイバー断面積分と集中ばねに用いる。塑性率算定の基点には `reference_stress`（鋼・鉄筋は降伏強度、コンクリートはシリンダー強度）と `reference_strain` を用いる。

### 2.1.1 バイリニア鋼材（線形移動硬化）

このモデルは、弾塑性力学の線形移動硬化（kinematic hardening）に基づく。コード上に規準名を明示するコメントはない。

**算定式**:
- 塑性係数 `Hp = hardening·E/(1 − hardening)`（降伏後接線 `dσ/dε = hardening·E` となるよう定義）
- 弾性予測 `σ_tr = E·(ε − εp)`、背応力 `α = Hp·εp`、降伏関数 `f = |σ_tr − α| − fy`
- `f ≤ 0`: 弾性（接線 E）。`f > 0`: 戻し写像 `Δεp = f/(E + Hp)`、`σ = E·(ε − εp_new)`、接線 `E·Hp/(E+Hp)`

**実装**: `squid_n_material::uniaxial::Bilinear`（`uniaxial.rs:64`）が算定する。既定硬化率は用途側で指定し、ファイバー要素では 0.01 とする。

**整合性**: 標準的な移動硬化の戻し写像に一致する。規準式ではなく力学モデルである。

### 2.1.2 Menegotto–Pinto 鉄筋モデル

このモデルは、バウシンガー効果を平滑に表現する Menegotto–Pinto モデルに基づく。ξ（塑性ひずみ蓄積）の更新は Chang & Mander (2004) 系の簡易形であり、コードにコメントとして記載している。

**算定式**（正規化形）:
- `ε* = (ε − εr)/(ε0 − εr)`
- `R = max(R0 − a1·ξ/(a2 + ξ), 1)`
- `σ* = b·ε* + (1 − b)·ε*/(1 + |ε*|^R)^{1/R}`
- `σ = σr + (σ0 − σr)·σ*`

**実装**: `uniaxial::MenegottoPinto`（`uniaxial.rs:185`）が算定する。既定パラメータは `b = 0.01, R0 = 20.0, a1 = 18.5, a2 = 0.15` であり、反転時に漸近線を鏡映更新する。

**整合性**: Menegotto–Pinto の標準正規化式に一致する。係数は AIJ 系代表値（Category B）であり、製品運用では外部パラメータの設定を推奨する旨をコメントに明記している。insta スナップショットでループを固定している（V&V #19）。

### 2.1.3 コンクリート一軸履歴モデル

このモデルは、圧縮を放物線上昇からピーク、線形軟化、残留へと進め、引張を弾性からひび割れ、指数減衰（テンションスティフニング）へと進める。係数は AIJ 系代表値であり、コードにコメントとして記載している。

**算定式**:
- 初期接線 `E0 = 2·fc/|εc0|`
- 圧縮上昇（`ε ≥ εc0`）: `σ = −fc·(2r − r²)`、`r = ε/εc0`
- 圧縮軟化（`εc0 > ε ≥ εcu`）: `σ = −fc + slope·(ε − εc0)`、`slope = fc·(1 − 残留比)/(εcu − εc0)`
- 引張（`ε > εcr = ft/E0`）: `σ = ft·exp(−β·(ε/εcr − 1))`（β = テンションスティフニング係数）
- 除荷・再載荷は最大経験ひずみへの原点指向割線

**実装**: `uniaxial::Concrete`（`uniaxial.rs:344`）が算定する。既定値は `εc0 = −0.002, εcu = −0.0035, テンションスティフニング 0.5, 残留比 0.0` であり、せん断は `G0 = E0/(2(1+ν))`（ν = 0.2）とする。

**整合性**: 放物線モデル（Hognestad 系）の標準形である。係数は Category B 代表値である。

### 2.1.4 NewRC コンクリート構成則（ファイバー柱の既定）

この構成則は、RESP-D マニュアル「計算編 05 非線形モデル」柱ファイバーモデルのコンクリート応力ひずみ関係（NewRC 有理式）に基づく。適用範囲は `Fc ≤ 60` であり、超過時はバイリニア骨格へ切り替える。

**算定式**（工学単位系 kgf/cm² で導出。実装は `σB = Fc[N/mm²]×(1/0.0980665)` で換算）:
- `σc/σcB = (A·X + (D−1)·X²)/(1 + (A−2)·X + D·X²)`、`X = εc/εc0`
- `εc0 = 0.5243·σB^{1/4}×10⁻³`
- `Ec = 4k·(σB/1000)^{1/3}×10⁵·(γ/2.4)²`（k = 1.0）
- `A = Ec·εc0/σcB`、`D = 1.50 + 1.68×10⁻³·σB`、`σcB = σB`（コンファインド効果なし）
- 引張は Ec 弾性、ひび割れ（`εcr = ft/Ec`）後は応力を保持しない脆性型

**実装**: `squid_n_material::newrc::ConcreteNewRc`（`newrc.rs`）が算定する。除荷則は、静的解析では逆行型（`Retrace`、圧縮包絡線を辿る）、動的解析では原点指向型（`OriginOriented`）に切り替える。

**整合性**: doc に「RESP-D 05 非線形モデル」を明記している。係数 `0.5243 / 1.50 / 1.68×10⁻³` は[原典照合リスト](../specs/原典照合リスト.md)で NewRC 指針として要照合（☐、適用範囲 Fc≤60）である。

## 2.2 部材（集中ばね）履歴則

この節は、集中ばね（one/two-component）用の M–θ 履歴則を扱う。スケルトンは crack/yield/ultimate の3点で与えるトリリニア（またはバイリニア）とし、除荷則はモデルごとに異なる。

### 2.2.1 履歴モデル一覧

各履歴モデルは、RESP-D マニュアル「07 非線形解析（動的解析）」履歴特性（`Retrograde`/`Standard`/`MaxPointOriented`/`TsujiYamada`/`SteelBuckling`）、および設計書 §7・仕様書 §5（`Takeda` ほか）に基づく。

**算定式**（各モデルの除荷則）:
- **武田モデル（Takeda / TakedaDegrading）**: 剛性低下型トリリニア。除荷剛性は `Ku = K0·|δmax/δy|^{−α}`（K0 = ひび割れ勾配 Mc/θc、α = 除荷剛性低下指数）。劣化版はピーク耐力を `degradation^n`（n = ピーク経験回数）で低下させる。
- **原点指向型（OriginOriented）**: バイリニアで、除荷は原点へ向かう割線とする。
- **スリップ型（Slip）**: 原点付近でピンチする再載荷とする（`slip_factor`）。
- **逆行型（Retrograde）**: 常にスケルトン上を辿り、履歴ループを持たない（静的解析既定）。
- **標準型（Standard）**: Masing 則（相似則）による除荷とし、除荷開始剛性を初期剛性 K1 とする。
- **最大点指向型（MaxPointOriented）**: Clough 系のピーク指向とする。

**実装**: `squid_n_material::hysteresis::HysteresisRule`（`hysteresis.rs:4`）が算定する。武田の除荷剛性は `unloading_stiffness`（`hysteresis.rs:185`）で「原典式 `Kd+ = K0·|δmax/δy2|^{−ν}`」と明記している。

**整合性**: 各モデルの doc に RESP-D「07」を引用している。武田は insta スナップショットでループを固定している（V&V #20）。従来の降伏割線基準から初期勾配 K0 基準へ是正済みである。

### 2.2.2 辻・山田モデル（混合硬化）

このモデルは、RESP-D「07」履歴特性に基づき、バイリニア骨格に β による等方硬化と移動硬化の混合硬化則を組み合わせる（二重鋼管座屈補剛ブレース等）。

**算定式**: 硬化係数 `H = K1·K2/(K1 − K2)`。降伏時 `Δᾱ = (1−β)·H·Δεp`（移動硬化）、`Δσ̄ = β·H·Δεp`（等方硬化）。`β = 1` で等方硬化、`β = 0` で移動硬化となる。

**実装**: `hysteresis::TsujiYamada`（`hysteresis.rs:717`）が算定する。

**整合性**: doc に RESP-D「07」を引用している。

### 2.2.3 鋼梁 横座屈考慮 Mu/Mp 比

この算定は、RESP-D「07」鉄骨大梁の座屈を考慮した履歴、および井戸田ほか (2015) の式に基づく。

**算定式**: 細長比 λb・κ（両端モーメント比）から `Mu/Mp` を算定する（`qκ`・`r`・`αΛ`・`Λc'` などの多段の閉形式。結果は `[0.05, 5.0]` にクランプ）。原典既定は `残留応力 cres = 0、形状係数 f = 1、kres = 0.3、kdef = 1.0` である。

**実装**: `hysteresis::lateral_buckling_mu_ratio`（`hysteresis.rs:846`）が算定する。除荷は孟・大井・高梨の Ramberg–Osgood モデル（`γ = 5, Φ = 0.5`）を `SteelBuckling`（`hysteresis.rs:892`）で実装している。

**整合性**: doc に井戸田式・RO モデルの出典を明記している。係数は Category B（外部データ確定対象）である。

## 2.3 材料強度（F値・許容応力度の基礎）

> 許容応力度の検定式そのものは[6. 一次設計](./06_一次設計.md)、終局耐力は[8. 部材終局耐力](./08_部材終局耐力.md)に
> 記載する。ここでは材料強度定数の根拠を示す。

材料強度定数は、建築基準法施行令89〜91条・96条・98条、および告示（鋼材の F 値等）に基づく。詳細な埋込値と条番号の対応は[原典照合リスト 優先度2](../specs/原典照合リスト.md#優先度2許容応力度特に鉄筋の径区分)を参照する。

**算定式**（主要な材料強度）:
- 鋼材 F 値（令98条／告示）: SN400/SS400 = 235（t ≤ 40）、SN490/SM490 = 325 ほか
- 鉄筋降伏強度（令90条・JIS G3112）: SD295 = 295、SD345 = 345、USD685 = 685 ほか
- コンクリート設計基準強度 Fc（令91条）: 許容応力度は長期 Fc/3、短期 2Fc/3
- ヤング係数比 n（Fc 区分）: Fc≤27→15 / ≤36→13 / ≤48→11 / ≤60→9 / それ超→7

**実装**: `squid_n_design_jp::material_strength`（`material_strength.rs`）が算定し、ヤング係数比 n は `rc::young_ratio_n` が算定する。既定鉄筋強度は `squid-n-app` で SD345（主筋）・SD295（帯筋）とする。

**整合性**: ヤング係数比 n・USD685 許容応力度は RESP-D マニュアル 04 と照合済みである（[原典照合リスト](../specs/原典照合リスト.md)☑）。鋼材・コンクリートの許容応力度の条番号対応は[6. 一次設計](./06_一次設計.md)を参照する。
