# 5. 構造解析

線形静解析・施工段階解析・固有値解析・減衰・時刻歴応答（Newmark-β / HHT-α）・非線形解法
（Newton-Raphson / 弧長法）・プッシュオーバー解析・段階的耐力喪失・累積損傷度のアルゴリズム根拠。
応力解析の手法は RESP-D マニュアル「計算編 03 応力解析」「07 非線形解析（動的解析）」に準拠する。

> 記載フォーマット: 📖 根拠 / ∑ 算定式 / 💻 実装 / ✓ 整合性

> **決定性:** 全ソルバで `faer::set_global_parallelism(Par::Seq)` を設定し、CPU・単一スレッドで
> ビット一致を保証する（[V&V 決定性テスト](../v_and_v/README.md#決定性テスト)）。

## 5.1 線形代数ソルバ

- 📖 **根拠**: 数値線形代数（Cholesky LLᵀ 分解・LU 分解・前処理付き共役勾配法 PCG）。
- ∑ **算定式**:
  - 疎 Cholesky（対称正定値 `K·u = F`）、非対称・ラグランジュ乗数付き拘束は疎 LU にフォールバック
  - Jacobi 前処理付き PCG（内部 f32 演算）: `d[i] = 1/a_ii`、収束判定 `‖r‖/‖b‖ < tol`
- 💻 **実装**: `squid_n_math`（`solver.rs`/`cholesky.rs`/`lu.rs`/`pcg.rs`、faer ライブラリ利用）。
- ✓ **整合性**: 標準アルゴリズム。Cholesky 分解・疎行列組立の決定性を検証（V&V）。

---

## 5.2 線形静解析

- 📖 **根拠**: 変位法（有限要素法）。RESP-D 計算編 03「応力解析」。
- ∑ **算定式**:
  - 全体剛性 `K_free` を組立、拘束を縮約 `K_red = reducerᵀ·K_free·reducer`
  - `u = K_red⁻¹·F_red`、部材内力回復 `behavior.recover_forces(u_elem)`
  - 実内力 = 剛性応答 + 両端固定梁のスパン内力（`fixed_internal_local`）の重ね合わせ
- 💻 **実装**: `statics::linear`（`linear/mod.rs::linear_static_once`）。長期の柱軸力無効化は
  断面積に `AXIAL_DISABLE_FACTOR = 1e-6` を乗じる（RESP-D 計算編 03）。柱判定 `|ez/L| > 0.707`。
- ✓ **整合性**: doc に RESP-D 計算編 03 を明記。部材荷重の重ね合わせを実装。

### 5.2.1 施工段階解析

- 📖 **根拠**: RESP-D マニュアル計算編 03「応力解析 §施工時解析」。下層から 1 層ずつ生成し増分を累積。
- ∑ **算定式**: 各ステージで新規生成部材のみの部分構造に固定荷重増分を載荷し `linear_static_once` を解く。
  全ステージ後に積載荷重を完成形へ載荷し累積。
- 💻 **実装**: `statics::construction`（`construction.rs::construction_stage_analysis`）。
- ✓ **整合性**: doc に RESP-D 計算編 03「施工時解析」を明記。

## 5.3 固有値解析

- 📖 **根拠**: 部分空間反復法（Bathe の定石）＋一般化固有値問題の Jacobi 解法。
- ∑ **算定式**:
  - 部分空間サイズ `q = min(2p, p+8)`（p = モード数）
  - 反復 `y = K⁻¹·x` → 射影 `K̄ = yᵀKy, M̄ = yᵀMy` → 一般化固有値 `gevd_jacobi(K̄, M̄)` → `x_new = y·V`
  - 収束判定 `|θ − θ_prev| < 1e-10·max(θ,1)`、`period = 2π/√ω²`、固有ベクトルは M 正規化 `φᵀMφ = 1`
  - 参加係数 `β = φᵀMr/φᵀMφ`、有効質量 `Meff = (φᵀMr)²/φᵀMφ`
- 💻 **実装**: `dynamic::eigen`（`eigen/mod.rs::solve_eigen`）。`EIGEN_TOL = 1e-10`、
  `EIGEN_MAX_ITER = 200`。整合質量（Consistent）を使用。
- ✓ **整合性**: 1 自由度周期をテストで照合（`test_1dof_period`、V&V #9）。doc に「Bathe の部分空間反復」を明記。

## 5.4 減衰

- 📖 **根拠**: RESP-D「07」減衰マトリクス（設計書 §10.5）。質量比例・剛性比例・Rayleigh・モード別・
  接線剛性比例。
- ∑ **算定式**:
  - 質量比例 `C = a0·M`（`a0 = 2hω`）、剛性比例 `C = a1·K`（`a1 = 2h/ω`）
  - Rayleigh `C = a0·M + a1·K`、`βk = 2(h2ω2 − h1ω1)/(ω2² − ω1²)`、
    `αm = 2ω1ω2(h1ω2 − h2ω1)/(ω2² − ω1²)`
  - モード別 `C = Σ 2hiωi(Mφi)(Mφi)ᵀ`、接線剛性比例 `C = (2h/ω)·Kt`
- 💻 **実装**: `dynamic::damping`（`damping.rs`）。h1 一定・累積型/非累積型の減衰力評価にも対応。
- ✓ **整合性**: doc に RESP-D「07」減衰マトリクス・モード別減衰・h1 一定を明記。

## 5.5 時刻歴応答解析

支配方程式 `M·ü + C·u̇ + K·u = −M·r·ẍg(t)`（基盤一様加振・相対変位形式）。整合質量を使用。

### 5.5.1 線形時刻歴（Newmark-β）

- 📖 **根拠**: Newmark-β 法。平均加速度法（β = 1/4, γ = 1/2、無条件安定）、線形加速度法
  （β = 1/6, γ = 1/2、条件付安定）。
- ∑ **算定式**:
  - 有効剛性 `K̂ = K + c2·C + c1·M`（`c1 = 1/(βΔt²), c2 = γ/(βΔt)`）
  - `u_{n+1} = K̂⁻¹·p_eff`、`a_{n+1} = c1(u_{n+1} − u) − c3·v − c4·a`、
    `v_{n+1} = v + Δt((1−γ)a + γ·a_{n+1})`
  - 初期加速度 `M·a0 = −C·v0 − K·u0 − p(0)`
- 💻 **実装**: `dynamic::timehistory`（`timehistory/mod.rs::linear_time_history_with_state`）。
- ✓ **整合性**: SDOF 解析解 `T = 0.198692 s` で検証（[原典照合リスト](../specs/原典照合リスト.md)）。

### 5.5.2 線形時刻歴（HHT-α）

- 📖 **根拠**: HHT-α 法（Hilber-Hughes-Taylor）。β = 0.25, γ = 0.5 固定、α ∈ [−1/3, 0]（既定 −0.1、
  α = 0 で Newmark 平均加速度に一致）。
- ∑ **算定式**: `K̂_HHT = (1+α)K + (1+α)c2·C + c1·M`、有効荷重に前ステップ項を含む。
- 💻 **実装**: `timehistory/mod.rs::linear_hht_alpha_analysis`。
- ✓ **整合性**: α = 0 で Newmark と一致することが構造的に保証される。数値減衰の導入用。

### 5.5.3 非線形時刻歴（Newmark-β ＋ Newton-Raphson）

- 📖 **根拠**: Newmark-β ＋ Newton-Raphson 反復（commit/rollback）。RESP-D「07」「05 非線形モデル」。
- ∑ **算定式**:
  - 各反復で接線剛性 `Kt`・接線比例減衰を再構成、`K_eff = Kt + c2·C + c1·M`
  - 残差 `r = p − f_int − C·v − M·a`、収束判定 `‖r‖ < tol·max(‖p‖, 1)`
  - `δu = K_eff⁻¹·r`、収束時 commit・不収束時 rollback
- 💻 **実装**: `timehistory/mod.rs::nonlinear_time_history_analysis`。コンクリート履歴は動的解析で
  原点指向型に切替、制振要素へ `set_time_step(dt)`。位相差入力（ねじれ加振）にも対応。
- ✓ **整合性**: doc に RESP-D「07」「05」を明記。時刻歴の RESP-D 照合は
  [非線形動的解析_RESP-D照合](../v_and_v/非線形動的解析_RESP-D照合.md)を参照。

### 5.5.4 質点系（串団子）非線形時刻歴

- 📖 **根拠**: RESP-D「07」質点系解析モデル。プッシュオーバー Q–δ を等包絡面積則でトリリニア縮約し、
  せん断型串団子で Newmark 平均加速度＋Newton、三重対角 Thomas 法で解く。
- ∑ **算定式**: 復元力は最大点指向型（Clough 系）トリリニア。等包絡面積則で `A_tri(δ2) = A_actual`
  を解き折点を決定。初期剛性比例減衰 `a1 = 2h/ω1`（ω1 は逆反復）。
- 💻 **実装**: `dynamic::lumped_mass`（`lumped_mass.rs`）。
- ✓ **整合性**: doc に RESP-D「07」質点系解析モデル・等包絡面積則を明記。

## 5.6 非線形解法（弧長法）

- 📖 **根拠**: 円筒型弧長法（Crisfield 1981）＋修正 Newton。荷重-変位の限界点・軟化域追跡。
- ∑ **算定式**: 予測子 `du_t = K⁻¹·q`、円筒拘束 `a·δλ² + b·δλ + c = 0`（`c = (du+dū)ᵀ(du+dū) − Δl²`）。
  根選択は前ステップとのなす角が正・小の根。
- 💻 **実装**: `nonlinear::arc_length`（`arc_length.rs::ArcLengthSolver`）。既定 `max_iter = 20, tol = 1e-6`。
- ✓ **整合性**: doc に「Crisfield 1981」を明記（著者・年併記）。

---

## 5.7 プッシュオーバー解析（漸増静的）

- 📖 **根拠**: 保有水平耐力計算のための漸増静的解析。外力分布は Ai 分布（[1. 荷重](./01_荷重.md)）。
  荷重制御 → 変位制御 → 弧長法の段階制御。RESP-D「05 非線形モデル」「06 終局検定」。
- ∑ **算定式**:
  - 外力 `q[a] += dir·share`（Ai 分布、多剛床は重量比按分）、周期は略算式 `T = h(0.02 + 0.01α)`、
    `C0 = 0.2`, `Z = 1.0`, 第2種地盤
  - Newton 反復 `K_t·δu = f_ext − f_int`（幾何剛性オプション `use_kg`）、収束 `‖r‖ < 1e-6·max(‖f_ext‖,1)`
  - 保有水平耐力 `Qu = 性能曲線上の最大ベースシア`
- 💻 **実装**: `nonlinear::pushover`（`pushover/mod.rs::pushover_analysis_recording`）。変位制御は
  ペナルティ法（1e16）、崩壊機構判定は静的不静定次数 `r = 3m − 3n + r_support` に対し
  降伏ヒンジ数 ≥ r+1 のゲート。
- ✓ **整合性**: doc に RESP-D「05」「06」を明記。RESP-D 照合は
  [非線形モデル_RESP-D照合](../v_and_v/非線形モデル_RESP-D照合.md)参照。

### 5.7.1 曲げヒンジ判定（Mc・My・終局）

- 📖 **根拠**: RESP-D「05 非線形モデル」。RC は `Mc = κ·Fc·Ze`（κ = 0.56）、`My = 0.9·at·σy·j`、
  鋼は `Mp = Zp·σy`。
- ∑ **算定式**: ヒンジレベルは `m_max ≥ My` かつ `μ ≥ ULTIMATE_DUCTILITY(=4.0)` → 終局、
  `Mc ≤ m_max < My` → ひび割れ。
- 💻 **実装**: `pushover/mod.rs::member_moment_thresholds`。鋼材 σy 既定 235、RC 鉄筋 345。
- ✓ **整合性**: `ULTIMATE_DUCTILITY = 4.0` は「要・原典照合／ユーザー調整余地」とコメント明記
  （[原典照合リスト](../specs/原典照合リスト.md)☐）。κ = 0.56 は RESP-D「05」。

### 5.7.2 部材塑性率（ductility）の3方式

- 📖 **根拠**: RESP-D「05 非線形モデル」ファイバーモデルの塑性率。
- ∑ **算定式**:
  - 方式(1) 基点ひずみ: RC 引張 0.01・圧縮 0.005、鉄骨 0.01 を超えた曲率を基点、`μ = κ_max/κ_ref`
  - 方式(2) 重み付け平均 `Jm = Σσy·A·|ε|·μi / Σσy·A·|ε| ≥ 1.0` の時点を基点
  - 方式(3) 初降伏
- 💻 **実装**: `pushover/mod.rs::compute_ductility_refs`。
- ✓ **整合性**: 基点歪み（方式1）は[原典照合リスト](../specs/原典照合リスト.md)で RC 引張0.01/圧縮0.005・
  鉄骨0.01 を照合済（☑）。

### 5.7.3 せん断降伏判定（Qy）

- 📖 **根拠**: RESP-D 計算編 03「応力解析 §段階的耐力喪失解析」。鋼系 `Qy = As·fy/√3`（von Mises）、
  RC は荒川 mean 式系（`rc_qsu_simple`、軸力を各ステップ反映）。
- 💻 **実装**: `pushover/mod.rs::build_dir_threshold`。RC の σy = 345（SD345）、σwy = 295（SD295）は
  「要・原典照合」明記。局所せん断を ey/ez へ射影し方向別に判定。
- ✓ **整合性**: せん断終局式は[8. 部材終局耐力](./08_部材終局耐力.md)参照。

## 5.8 段階的耐力喪失解析

- 📖 **根拠**: RESP-D 計算編 03「応力解析 §段階的耐力喪失解析」。せん断降伏後の耐力劣化を再載荷で
  疑似考慮。FEMA 356 Table 6-7（塑性回転角 a）。
- ∑ **算定式**: 耐力喪失変形角超過部材を両端ピン・せん断非負担へ置換し荷重 0 から再載荷。
  FEMA C 判定 `s ≤ D/3 かつ Vs ≥ 0.75V`、塑性回転角 a は双線形補間（テーブル 4 隅値）。
- 💻 **実装**: `nonlinear::strength_loss`（`strength_loss/mod.rs`）。
- ✓ **整合性**: doc に RESP-D 計算編 03・FEMA 356 Table 6-7 を明記。柱の FEMA 非線形特性は未対応
  （RESP-D 自身も未対応）。

## 5.9 累積損傷度（鉄骨梁端部）

- 📖 **根拠**: RESP-D「07」その他の解析機能「鉄骨梁端部の累積損傷度計算」。レインフロー法
  （ASTM E1049-85 3 点法）＋ Miner 則。
- ∑ **算定式**: 破断寿命 `Nf = (μ/C)^{−1/β}`、`D = Σ count·(μ/C)^{1/β}`。
- 💻 **実装**: `damage.rs`。既定 `C = 20.0, β = 0.5`（「暫定既定・要原典照合」明記）。
- ✓ **整合性**: doc に ASTM E1049-85・Miner 則を明記。疲労パラメータは[原典照合リスト](../specs/原典照合リスト.md)対象。
