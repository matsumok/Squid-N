# 5. 構造解析

本章では、線形静解析、施工段階解析、固有値解析、減衰、時刻歴応答（Newmark-β / HHT-α）、
非線形解法（Newton-Raphson / 弧長法）、プッシュオーバー解析、段階的耐力喪失、累積損傷度の
アルゴリズム根拠を記す。応力解析の手法は RESP-D マニュアル「計算編 03 応力解析」
「07 非線形解析（動的解析）」に準拠する。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

> **決定性:** 全ソルバで `faer::set_global_parallelism(Par::Seq)` を設定し、CPU の単一スレッドで
> ビット一致を保証する（[V&V 決定性テスト](../v_and_v/README.md#決定性テスト)）。

## 5.1 線形代数ソルバ

構造解析の基礎となる連立一次方程式を、数値線形代数の標準手法（Cholesky LLᵀ 分解、LU 分解、
前処理付き共役勾配法 PCG）で解く。

**算定式**

- 対称正定値の `K·u = F` には疎 Cholesky を用い、非対称やラグランジュ乗数付き拘束では疎 LU に
  フォールバックする。
- Jacobi 前処理付き PCG（内部 f32 演算）では `d[i] = 1/a_ii` とし、収束は `‖r‖/‖b‖ < tol` で判定する。

**実装**：`squid_n_math`（`solver.rs`/`cholesky.rs`/`lu.rs`/`pcg.rs`、faer ライブラリ利用）が算定する。

**整合性**：いずれも標準アルゴリズムであり、Cholesky 分解と疎行列組立の決定性を V&V で検証している。

## 5.2 線形静解析

変位法（有限要素法）により部材内力と変位を求める。RESP-D 計算編 03「応力解析」に準拠する。

**算定式**

- 全体剛性 `K_free` を組立て、拘束を縮約して `K_red = reducerᵀ·K_free·reducer` を得る。
- `u = K_red⁻¹·F_red` を解き、部材内力を `behavior.recover_forces(u_elem)` で回復する。
- 実内力は、剛性応答と両端固定梁のスパン内力（`fixed_internal_local`）の重ね合わせとする。

**実装**：`statics::linear`（`linear/mod.rs::linear_static_once`）が算定する。長期の柱軸力無効化は
断面積に `AXIAL_DISABLE_FACTOR = 1e-6` を乗じて行い（RESP-D 計算編 03）、柱判定は `|ez/L| > 0.707`
による。

**整合性**：部材荷重の重ね合わせを実装し、doc に RESP-D 計算編 03 を明記している。

### 5.2.1 施工段階解析

施工過程を再現するため、下層から 1 層ずつ部材を生成して増分を累積する。RESP-D マニュアル
計算編 03「応力解析 §施工時解析」に準拠する。

**算定式**：各ステージでは、新規生成部材のみの部分構造に固定荷重増分を載荷して
`linear_static_once` を解く。全ステージ完了後に、積載荷重を完成形へ載荷して累積する。

**実装**：`statics::construction`（`construction.rs::construction_stage_analysis`）が算定する。

**整合性**：doc に RESP-D 計算編 03「施工時解析」を明記している。

## 5.3 固有値解析

固有周期とモードを、部分空間反復法（Bathe の定石）と一般化固有値問題の Jacobi 解法で求める。

**算定式**

- 部分空間サイズは `q = min(2p, p+8)`（p = モード数）とする。
- 反復は `y = K⁻¹·x` から射影 `K̄ = yᵀKy, M̄ = yᵀMy` を作り、一般化固有値 `gevd_jacobi(K̄, M̄)` を解いて
  `x_new = y·V` を更新する。
- 収束は `|θ − θ_prev| < 1e-10·max(θ,1)` で判定し、`period = 2π/√ω²`、固有ベクトルは M 正規化
  `φᵀMφ = 1` とする。
- 参加係数は `β = φᵀMr/φᵀMφ`、有効質量は `Meff = (φᵀMr)²/φᵀMφ` で算定する。

**実装**：`dynamic::eigen`（`eigen/mod.rs::solve_eigen`）が算定する。`EIGEN_TOL = 1e-10`、
`EIGEN_MAX_ITER = 200` とし、整合質量（Consistent）を使用する。

**整合性**：1 自由度周期をテストで照合し（`test_1dof_period`、V&V #9）、doc に「Bathe の部分空間反復」を
明記している。

## 5.4 減衰

減衰マトリクスを、質量比例、剛性比例、Rayleigh、モード別、接線剛性比例の各方式で構成する。
RESP-D「07」減衰マトリクス（設計書 §10.5）に準拠する。

**算定式**

- 質量比例は `C = a0·M`（`a0 = 2hω`）、剛性比例は `C = a1·K`（`a1 = 2h/ω`）とする。
- Rayleigh は `C = a0·M + a1·K` とし、`βk = 2(h2ω2 − h1ω1)/(ω2² − ω1²)`、
  `αm = 2ω1ω2(h1ω2 − h2ω1)/(ω2² − ω1²)` で係数を定める。
- モード別は `C = Σ 2hiωi(Mφi)(Mφi)ᵀ`、接線剛性比例は `C = (2h/ω)·Kt` とする。

**実装**：`dynamic::damping`（`damping.rs`）が算定し、h1 一定や累積型/非累積型の減衰力評価にも対応する。

**整合性**：doc に RESP-D「07」減衰マトリクス、モード別減衰、h1 一定を明記している。

## 5.5 時刻歴応答解析

支配方程式 `M·ü + C·u̇ + K·u = −M·r·ẍg(t)`（基盤一様加振、相対変位形式）を解く。整合質量を使用する。

### 5.5.1 線形時刻歴（Newmark-β）

線形の時刻歴応答を Newmark-β 法で解く。平均加速度法（β = 1/4, γ = 1/2、無条件安定）と
線形加速度法（β = 1/6, γ = 1/2、条件付安定）を用いる。

**算定式**

- 有効剛性を `K̂ = K + c2·C + c1·M`（`c1 = 1/(βΔt²), c2 = γ/(βΔt)`）とする。
- `u_{n+1} = K̂⁻¹·p_eff` を解き、`a_{n+1} = c1(u_{n+1} − u) − c3·v − c4·a`、
  `v_{n+1} = v + Δt((1−γ)a + γ·a_{n+1})` で更新する。
- 初期加速度は `M·a0 = −C·v0 − K·u0 − p(0)` から求める。

**実装**：`dynamic::timehistory`（`timehistory/mod.rs::linear_time_history_with_state`）が算定する。

**整合性**：SDOF 解析解 `T = 0.198692 s` で検証している（[原典照合リスト](../specs/原典照合リスト.md)）。

### 5.5.2 線形時刻歴（HHT-α）

数値減衰を導入するため、線形時刻歴を HHT-α 法（Hilber-Hughes-Taylor）で解く。β = 0.25, γ = 0.5 を
固定し、α ∈ [−1/3, 0]（既定 −0.1、α = 0 で Newmark 平均加速度に一致）とする。

**算定式**：`K̂_HHT = (1+α)K + (1+α)c2·C + c1·M` とし、有効荷重に前ステップ項を含める。

**実装**：`timehistory/mod.rs::linear_hht_alpha_analysis` が算定する。

**整合性**：α = 0 で Newmark と一致することが構造的に保証される。

### 5.5.3 非線形時刻歴（Newmark-β ＋ Newton-Raphson）

非線形の時刻歴応答を、Newmark-β と Newton-Raphson 反復（commit/rollback）で解く。
RESP-D「07」「05 非線形モデル」に準拠する。

**算定式**

- 各反復で接線剛性 `Kt` と接線比例減衰を再構成し、`K_eff = Kt + c2·C + c1·M` とする。
- 残差を `r = p − f_int − C·v − M·a` とし、収束を `‖r‖ < tol·max(‖p‖, 1)` で判定する。
- `δu = K_eff⁻¹·r` を解き、収束時に commit、不収束時に rollback する。

**実装**：`timehistory/mod.rs::nonlinear_time_history_analysis` が算定する。コンクリート履歴は動的解析で
原点指向型に切替え、制振要素へ `set_time_step(dt)` を設定する。位相差入力（ねじれ加振）にも対応する。

**整合性**：doc に RESP-D「07」「05」を明記している。時刻歴の RESP-D 照合は
[非線形動的解析_RESP-D照合](../v_and_v/非線形動的解析_RESP-D照合.md)を参照する。

### 5.5.4 質点系（串団子）非線形時刻歴

質点系（串団子）モデルの非線形時刻歴を解く。RESP-D「07」質点系解析モデルに準拠し、プッシュオーバー
Q–δ を等包絡面積則でトリリニアに縮約し、せん断型串団子で Newmark 平均加速度と Newton を用い、
三重対角 Thomas 法で解く。

**算定式**：復元力は最大点指向型（Clough 系）トリリニアとする。等包絡面積則で `A_tri(δ2) = A_actual`
を解いて折点を決定する。減衰は初期剛性比例 `a1 = 2h/ω1`（ω1 は逆反復で求める）とする。

**実装**：`dynamic::lumped_mass`（`lumped_mass.rs`）が算定する。

**整合性**：doc に RESP-D「07」質点系解析モデルと等包絡面積則を明記している。

## 5.6 非線形解法（弧長法）

荷重-変位の限界点と軟化域を追跡するため、円筒型弧長法（Crisfield 1981）と修正 Newton を用いる。

**算定式**：予測子を `du_t = K⁻¹·q` とし、円筒拘束 `a·δλ² + b·δλ + c = 0`
（`c = (du+dū)ᵀ(du+dū) − Δl²`）を解く。根選択は、前ステップとのなす角が正で小さい根を採る。

**実装**：`nonlinear::arc_length`（`arc_length.rs::ArcLengthSolver`）が算定する。既定は
`max_iter = 20, tol = 1e-6`。

**整合性**：doc に「Crisfield 1981」を著者・年併記で明記している。

## 5.7 プッシュオーバー解析（漸増静的）

保有水平耐力を求めるため、漸増静的解析を行う。外力分布は Ai 分布（[1. 荷重](./01_荷重.md)）とし、
荷重制御から変位制御、弧長法へと段階的に制御を切替える。RESP-D「05 非線形モデル」「06 終局検定」に
準拠する。

**算定式**

- 外力は `q[a] += dir·share`（Ai 分布、多剛床は重量比で按分）とし、周期は略算式
  `T = h(0.02 + 0.01α)`、`C0 = 0.2`, `Z = 1.0`, 第2種地盤を用いる。
- Newton 反復は `K_t·δu = f_ext − f_int`（幾何剛性オプション `use_kg`）とし、収束を
  `‖r‖ < 1e-6·max(‖f_ext‖,1)` で判定する。
- 保有水平耐力は `Qu = 性能曲線上の最大ベースシア` とする。

**実装**：`nonlinear::pushover`（`pushover/mod.rs::pushover_analysis_recording`）が算定する。
変位制御はペナルティ法（1e16）で行い、崩壊機構判定は静的不静定次数 `r = 3m − 3n + r_support` に対して
降伏ヒンジ数 ≥ r+1 をゲートとする。

**整合性**：doc に RESP-D「05」「06」を明記している。RESP-D 照合は
[非線形モデル_RESP-D照合](../v_and_v/非線形モデル_RESP-D照合.md)を参照する。

### 5.7.1 曲げヒンジ判定（Mc・My・終局）

曲げヒンジの発生レベルを判定する。RESP-D「05 非線形モデル」に準拠し、RC は `Mc = κ·Fc·Ze`
（κ = 0.56）、`My = 0.9·at·σy·j`、鋼は `Mp = Zp·σy` による。

**算定式**：`m_max ≥ My` かつ `μ ≥ ULTIMATE_DUCTILITY(=4.0)` のとき終局、`Mc ≤ m_max < My` のとき
ひび割れとする。

**実装**：`pushover/mod.rs::member_moment_thresholds` が算定する。鋼材 σy は既定 235、RC 鉄筋は
345 とする。

**整合性**：`ULTIMATE_DUCTILITY = 4.0` は「要・原典照合／ユーザー調整余地」とコメントに明記している
（[原典照合リスト](../specs/原典照合リスト.md)☐）。κ = 0.56 は RESP-D「05」による。

### 5.7.2 部材塑性率（ductility）の3方式

部材塑性率を3方式で算定する。RESP-D「05 非線形モデル」ファイバーモデルの塑性率に準拠する。

**算定式**

- 方式(1) は基点ひずみによる。RC 引張 0.01、圧縮 0.005、鉄骨 0.01 を超えた曲率を基点とし、
  `μ = κ_max/κ_ref` とする。
- 方式(2) は重み付け平均 `Jm = Σσy·A·|ε|·μi / Σσy·A·|ε| ≥ 1.0` の時点を基点とする。
- 方式(3) は初降伏による。

**実装**：`pushover/mod.rs::compute_ductility_refs` が算定する。

**整合性**：基点歪み（方式1）は[原典照合リスト](../specs/原典照合リスト.md)で RC 引張 0.01、
圧縮 0.005、鉄骨 0.01 を照合済である（☑）。

### 5.7.3 せん断降伏判定（Qy）

せん断降伏を判定する。RESP-D 計算編 03「応力解析 §段階的耐力喪失解析」に準拠し、鋼系は
`Qy = As·fy/√3`（von Mises）、RC は荒川 mean 式系（`rc_qsu_simple`、軸力を各ステップで反映）による。

**実装**：`pushover/mod.rs::build_dir_threshold` が算定する。RC の σy = 345（SD345）、σwy = 295
（SD295）は「要・原典照合」と明記する。局所せん断を ey/ez へ射影して方向別に判定する。

**整合性**：せん断終局式は[8. 部材終局耐力](./08_部材終局耐力.md)を参照する。

## 5.8 段階的耐力喪失解析

せん断降伏後の耐力劣化を再載荷で疑似的に考慮する。RESP-D 計算編 03「応力解析 §段階的耐力喪失解析」と
FEMA 356 Table 6-7（塑性回転角 a）に準拠する。

**算定式**：耐力喪失変形角を超過した部材を、両端ピンかつせん断非負担へ置換して荷重 0 から再載荷する。
FEMA C 判定は `s ≤ D/3 かつ Vs ≥ 0.75V` とし、塑性回転角 a はテーブル 4 隅値の双線形補間で求める。

**実装**：`nonlinear::strength_loss`（`strength_loss/mod.rs`）が算定する。

**整合性**：doc に RESP-D 計算編 03 と FEMA 356 Table 6-7 を明記している。柱の FEMA 非線形特性は未対応で
ある（RESP-D 自身も未対応）。

## 5.9 累積損傷度（鉄骨梁端部）

鉄骨梁端部の累積損傷度を算定する。RESP-D「07」その他の解析機能「鉄骨梁端部の累積損傷度計算」に
準拠し、レインフロー法（ASTM E1049-85 3 点法）と Miner 則を用いる。

**算定式**：破断寿命を `Nf = (μ/C)^{−1/β}`、損傷度を `D = Σ count·(μ/C)^{1/β}` とする。

**実装**：`damage.rs` が算定する。既定は `C = 20.0, β = 0.5`（「暫定既定・要原典照合」と明記）。

**整合性**：doc に ASTM E1049-85 と Miner 則を明記している。疲労パラメータは
[原典照合リスト](../specs/原典照合リスト.md)の対象である。
