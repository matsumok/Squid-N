# 3.4 コンクリートのヤング係数 Ec

コンクリートのヤング係数 Ec は、RC規準の Ec 式で算定する。
基本形は次式である。

\\[ E_c = 3.35 \times 10^4 \cdot (\gamma/24)^2 \cdot (F_c/60)^{1/3} \\]

**算定式**（\\( \gamma = 23 \\) kN/m³ 固定、\\( F_c \le 0 \\) は 0 とする）:

\\[ E_c = 3.35 \times 10^4 \cdot (23/24)^2 \cdot (F_c/60)^{1/3} \\]

**実装**：`section_shape::concrete_young_modulus` が算定する。
