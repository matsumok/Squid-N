# 3.4 コンクリートのヤング係数 Ec

コンクリートのヤング係数 Ec は、RC規準の Ec 式で算定する。
基本形は \\( E_c = 3.35 \times 10^4 \cdot (\gamma/24)^2 \cdot (F_c/60)^{1/3} \\) である。

**算定式**：\\( E_c = 3.35 \times 10^4 \cdot (23/24)^2 \cdot (F_c/60)^{1/3} \\)（\\( \gamma = 23 \\) kN/m³ 固定、\\( F_c \le 0 \\) は 0 とする）。

**実装**：`section_shape::concrete_young_modulus` が算定する。

**整合性**：RC規準式に一致する。
ただし \\( \gamma = 23 \\) 固定については[原典照合リスト](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md)で要照合（☐）である。
