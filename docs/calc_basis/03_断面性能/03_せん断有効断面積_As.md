# 3.3 せん断有効断面積 As

せん断有効断面積 As は、せん断形状係数 κ による標準的な取扱いで算定する。
RC/SRC は \\( A_s = A/\kappa \\)（\\( \kappa = 1.2 \\)）とする。
S 造はウェブ・フランジ断面（強軸はウェブ、弱軸はフランジ）から \\( A_s = A_w/\kappa \\)（\\( \kappa = 1.0 \\)）とする。

**算定式（代表例）**

H 形:

\\[ a_{s,y}\text{（弱軸）} = 2 \cdot B \cdot t_f \\]

\\[ a_{s,z}\text{（強軸）} = H \cdot t_w \\]

矩形 RC:

\\[ a_s = b \cdot d/1.2 \\]

円形鋼管:

\\[ a_s = A/2 \\]

**実装**：`section_shape::to_section` が算定する。
強軸曲げ Iy と z 方向せん断 as_z、弱軸曲げ Iz と y 方向せん断 as_y を対応させる規約に従う（P1 §4.1）。

**整合性**：RC の \\( \kappa = 1.2 \\)、および矩形せん断の \\( k = 6/5 \\)（\\( A_s = 5A/6 \\)）は、矩形断面のせん断形状係数の力学値に整合する。
