# 3.5 耐震壁のせん断形状係数 κ

耐震壁のせん断形状係数 κ は、側柱付き壁を I 形断面とみなしたせん断形状係数（力学）で算定する。

**算定式**：
\\[ \kappa = \frac{3(1+\xi)}{5 \cdot \left( 1 - \xi^3 (1-\eta) \right)^2} \cdot \left[ \eta + \xi(1-\eta) \cdot \left( \frac{15}{8} (1 - \xi^2)^2 - \xi^4 \cdot \eta \right) \right] \\]

**実装**：`section_shape::wall_shear_shape_factor` が算定する。
ここで ξ は壁板内法長さ/側柱外面間全長、η は壁厚/側柱幅を表す。
退化時（\\( \text{denom} \le 0 \\) 等）は KAPPA_RC = 1.2 にフォールバックする。
