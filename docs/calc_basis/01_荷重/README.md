# 1. 荷重・外力

このソフトは、固定荷重、積載荷重、積雪荷重、風圧力、地震力を算定する。
地震力は建築基準法施行令88条および昭和55年建設省告示第1793号、風圧力は令87条および平成12年建設省告示第1454号を根拠とする。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。（[凡例](../README.md#このセクションの読み方)）
