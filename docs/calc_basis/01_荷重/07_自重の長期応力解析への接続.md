# 1.7 自重の長期応力解析への接続（自重(自動)荷重ケース）

部材自重・壁自重・ダンパー自重を、固定荷重（`LoadCaseKind::Dead`）の荷重ケース「自重(自動)」として自動生成し、長期応力解析・荷重組合せへ接続する。
梁は自重による等分布荷重（梁自重による CMQ に相当）、柱は軸力、壁・シェルは頂点への節点荷重（三方スリット壁は上部大梁側の節点へ全量伝達）となる。

**算定式**：総重量は §1.6 の自重算定規則（`enumerate_self_weight`）と同一とし、線材は次式の等分布荷重、その他は節点荷重に変換する（総量保存）。

\\[ w = \text{総重量}/\text{節点間長} \\]

**実装**：`squid_n_load::self_weight::self_weight_case_content`（`self_weight.rs`）が生成し、アプリの解析実行系（`sync_self_weight_action`）が冪等に同期する。
地震用重量の重力ケース選択（`gravity_cases_for_seismic_weight`）からは名前で除外する（`story_gen` が自重を密度から直接集計するため、含めると二重計上になる）。

**整合性**：自重と地震用重量集計の単一ソース化をテスト（`test_self_weight_matches_story_gen_totals`）で検証している。
