# 5.6 非線形解法（弧長法）

荷重-変位の限界点と軟化域を追跡するため、円筒型弧長法（Crisfield 1981）と修正 Newton を用いる。

**算定式**：予測子を \\( du_t = K^{-1} \cdot q \\) とし、円筒拘束 \\( a \cdot \delta\lambda^2 + b \cdot \delta\lambda + c = 0 \\)（\\( c = (du + d\bar{u})^\mathsf{T}(du + d\bar{u}) - \Delta l^2 \\)）を解く。
根選択は、前ステップとのなす角が正で小さい根を採る。

**実装**：`nonlinear::arc_length`（`arc_length.rs::ArcLengthSolver`）が算定する。
既定は `max_iter = 20, tol = 1e-6`。

**整合性**：doc に「Crisfield 1981」を著者・年併記で明記している。
