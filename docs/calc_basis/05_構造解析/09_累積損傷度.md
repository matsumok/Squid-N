# 5.9 累積損傷度（鉄骨梁端部）

鉄骨梁端部の累積損傷度を算定する。
レインフロー法（ASTM E1049-85 3 点法）と Miner 則を用いる。

**算定式**：破断寿命を \\( N_f = (\mu/C)^{-1/\beta} \\)、損傷度を \\( D = \sum \text{count} \cdot (\mu/C)^{1/\beta} \\) とする。

**実装**：`damage.rs` が算定する。既定は \\( C = 20.0, \beta = 0.5 \\)（「暫定既定・要原典照合」と明記）。

**整合性**：doc に ASTM E1049-85 と Miner 則を明記している。
疲労パラメータは[原典照合リスト](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md)の対象である。
