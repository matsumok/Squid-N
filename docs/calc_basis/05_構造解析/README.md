# 5. 構造解析

本章では、線形静解析、施工段階解析、固有値解析、減衰、時刻歴応答（Newmark-β / HHT-α）、非線形解法（Newton-Raphson / 弧長法）、プッシュオーバー解析、段階的耐力喪失、累積損傷度のアルゴリズム根拠を記す。
応力解析・動的解析の手法は、変位法（マトリクス法・有限要素法）と構造動力学の標準的な数値解析法に基づく。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

> **決定性:** 全ソルバで `faer::set_global_parallelism(Par::Seq)` を設定し、CPU の単一スレッドでビット一致を保証する（[V&V 決定性テスト](https://github.com/hrntsm/squid-n/blob/main/v_and_v/README.md#決定性テスト)）。
