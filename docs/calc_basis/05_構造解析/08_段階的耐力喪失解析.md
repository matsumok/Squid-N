# 5.8 段階的耐力喪失解析

せん断降伏後の耐力劣化を再載荷で疑似的に考慮する（段階的耐力喪失解析）。
塑性回転角 a は FEMA 356 Table 6-7 に基づく。

**算定式**：耐力喪失変形角を超過した部材を、両端ピンかつせん断非負担へ置換して荷重 0 から再載荷する。
FEMA C 判定（せん断補強良好）は FEMA 356 Table 6-7 脚注のとおり \\( s \le d/3 \\) かつ \\( V_s \ge 0.75V \\)（d は有効せい）とする。
塑性回転角 a はテーブル 4 隅値の双線形補間で求める。

**実装**：`nonlinear::strength_loss`（`strength_loss/mod.rs`）が算定する。

**整合性**：doc に FEMA 356 Table 6-7 を明記している。
柱の FEMA 非線形特性は未対応である。
部材変形角は弦回転角（弾性成分込み）で評価しており、FEMA の塑性回転角に対して喪失判定が早まる保守側の近似である（実装 doc に明記）。
