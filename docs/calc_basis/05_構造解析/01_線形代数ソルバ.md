# 5.1 線形代数ソルバ

構造解析の基礎となる連立一次方程式を、数値線形代数の標準手法（Cholesky LLᵀ 分解、LU 分解、前処理付き共役勾配法 PCG）で解く。

**算定式**

- 対称正定値の \\( K \cdot u = F \\) には疎 Cholesky を用い、非対称やラグランジュ乗数付き拘束では疎 LU に
  フォールバックする。
- Jacobi 前処理付き PCG（内部 f32 演算）では \\( d[i] = 1/a_{ii} \\) とし、収束は \\( \\|r\\|/\\|b\\| < \text{tol} \\) で判定する。

**実装**：`squid_n_math`（`solver.rs`/`cholesky.rs`/`lu.rs`/`pcg.rs`、faer ライブラリ利用）が算定する。

**整合性**：いずれも標準アルゴリズムであり、Cholesky 分解と疎行列組立の決定性を V&V で検証している。
