# 4. 要素の定式化・剛性

各有限要素の剛性マトリクス・質量マトリクス・幾何剛性・内力回復の定式化。フレーム要素は
ティモシェンコ梁理論、板は MITC4 シェル、剛性計算に用いる断面性能・剛域・雑壁の扱いは
RESP-D マニュアル「計算編 02 剛性計算」に準拠する。

> 記載フォーマット: 📖 根拠 / ∑ 算定式 / 💻 実装 / ✓ 整合性

---

## 4.1 ティモシェンコ梁要素（12×12）

- 📖 **根拠**: ティモシェンコ梁理論（せん断変形を考慮した梁要素）。力学の閉形式（[V&V](../v_and_v/README.md)で
  片持ち先端たわみ δ = 0.417292 mm 等により検証）。
- ∑ **算定式**:
  - せん断パラメータ `φz = 12·E·Iz/(G·As_y·L²)`、`φy = 12·E·Iy/(G·As_z·L²)`
  - 曲げ係数 `az = E·Iz/((1+φz)·L³)`、`ay = E·Iy/((1+φy)·L³)`
  - 軸 `k = E·A/L`、ねじり `k = G·J/L`
  - 曲げ剛性 4×4 ブロック（例, xy 面）: `12·az`、`6·az·L`、`(4+φz)·az·L²`、`(2−φz)·az·L²`
- 💻 **実装**: `squid_n_element::frame::beam`（`beam/mod.rs::local_stiffness_raw`）。せん断有効断面積が
  0 のとき `rect_shear_area(A) = 5A/6` にフォールバック。組立順は「可撓長で raw 剛性 → 端部ばね静縮約
  → 剛域変換」。
- ✓ **整合性**: `φ = 0`（せん断変形無視）で Bernoulli 梁に収束することをテストで確認
  （`test_phi_zero_converges_to_bernoulli`）。せん断補正係数は 6/5（矩形）。

### 4.1.1 端部条件（ピン・半剛）の静縮約

- 📖 **根拠**: 力学（自由度縮約）。`K* = Kaa − Kab·Kbb⁻¹·Kba`。
- ∑ **算定式**: 端部回転ばね剛性を Fixed = `1e8·rot_scale`、Pinned = 0、SemiRigid = `k_theta` とし、
  18 自由度（外部 12 + 内部端 6）を 12 自由度へ静縮約。`rot_scale = E·max(Iz,Iy)/max(L,1)`。
- 💻 **実装**: `beam/mod.rs::condense_end_springs`。剛接ペナルティ係数 1e8（1e12 だと Iz 大で数値破綻）。
- ✓ **整合性**: ピン端でモーメントが解放されることをテストで確認（`test_pinned_end_releases_moment`）。

### 4.1.2 剛域

- 📖 **根拠**: RESP-D マニュアル「剛域の計算」。可撓長 `L' = L − λi − λj` で剛性を求め、剛域を剛体
  変換する。
- ∑ **算定式**（剛域長 λ）:
  - RC/SRC 造: `λ = reduction·(D_直交/2 − D_自身/4)`（負は 0）
  - S・CFT 造: `λ = D_直交/2`（D_自身/4 の控除・reduction 乗算なし）
  - 危険断面フェイス距離 `face = D_直交/2`
- 💻 **実装**: `beam/rigid_zone.rs::auto_rigid_zones`、変換は `beam/mod.rs::apply_rigid_zone_transform`。
  直交判定は軸内積 < 0.707（概ね 45°以上）。
- ✓ **整合性**: 標準式の照合は 🔶（V&V #2）。仕口に接続する部材がすべて S 造なら剛域長 0 とする規定を
  実装（doc に RESP-D マニュアルを明記）。

### 4.1.3 幾何剛性・整合質量

- 📖 **根拠**: 力学（P-Δ 効果の線形化幾何剛性、Hermite 梁の一貫質量）。[V&V](../v_and_v/README.md)で
  固有値・座屈傾向により検証。
- ∑ **算定式**:
  - 幾何剛性 `Kg`: `c = N/L`、係数 `6/5`、`L/10`、`2L²/15`、`−L²/30`（標準ビーム幾何剛性）
  - 整合質量: `c2 = m/420`、係数 `156, 54, 22, 13, 4, 3`（標準整合質量）
- 💻 **実装**: `beam/mod.rs::{geometric_stiffness, mass_matrix}`。集中質量は並進自由度に m/2。
- ✓ **整合性**: 標準的な梁の幾何剛性・整合質量係数に一致。

### 4.1.4 剛性計算上の付加（RESP-D 計算編 02）

RESP-D マニュアル計算編 02「剛性計算」に基づき、次の剛性補正を実装している（doc に各節を明記）。

- **スラブ協力幅による強軸曲げ剛性増大**（RC大梁）: 片側協力幅 `ba`（`a < L/2` で `(0.5 − 0.6a/L)·a`、
  上限 `0.1L`）から T 形断面の Ie を求め、`(Ie/I0)` 倍する。根拠は RC規準 8 条。
  実装 `beam/mod.rs::slab_stiffness_factor`。
- **壁エレメントモデル上下大梁の剛性倍率**: 既定 100 倍（`WALL_GIRDER_STIFF_FACTOR`）。
- **フレーム内雑壁（袖壁・腰壁・垂壁）**: 平行軸の定理で周辺部材の断面性能に合成
  （`compose`）。せん断は `As += Σaw/1.2`。実装 `beam/mod.rs`（袖壁・腰壁処理）。

## 4.2 トラス（ブレース）要素

- 📖 **根拠**: RESP-D マニュアル計算編 02「一般ブレースの剛性」。`KB = E·A/L`（軸のみ）。
- ∑ **算定式**: `ka = factor·E·A/L`。引張専用ブレースは弾性解析で剛性 1/2、弾塑性解析で初期剛性 1 倍。
  圧縮スラック側は残留剛性比 1e-6。
- 💻 **実装**: `frame::truss`（`truss.rs`）。
- ✓ **整合性**: doc に RESP-D 計算編 02「一般ブレースの剛性」を明記。

## 4.3 部材荷重（等価節点力・固定端内力）

- 📖 **根拠**: 力学（Hermite 形状関数による等価節点力＝consistent load、両端固定梁の固定端内力）。
- ∑ **算定式**:
  - 等価節点力 `Q = ∫ Nᵀ·w dx`（軸は線形形状関数、曲げは Hermite 形状関数、分布荷重は 3 点 Gauss 積分）
  - 固定端内力 `fFEF = −Q`（等価節点力の符号反転）に span 内力を重畳
- 💻 **実装**: `frame::member_load`（`member_load.rs::{consistent_load_local, fixed_internal_local}`）。
- ✓ **整合性**: テストで UDL 固定端モーメント `wL²/12`、中央集中 `PL/8`、片側三角形
  `wL²/30`(i)・`wL²/20`(j) を照合（CMQ の力学的閉形式）。

## 4.4 MITC4 シェル要素

- 📖 **根拠**: MITC4（Mixed Interpolation of Tensorial Components）。せん断ロッキングを回避する
  混合補間法。膜＋曲げ＋せん断＋ドリリング安定化。
- ∑ **算定式**:
  - 膜 `Dm = E·t/(1−ν²)·[...]`、曲げ `Db = E·t³/(12(1−ν²))·[...]`、せん断 `Ds = (5/6)·G·t·I`
    （せん断補正係数 5/6）
  - MITC4 せん断補間: タイング点 A(0,+1)/B(−1,0)/C(0,−1)/D(+1,0) の共変ひずみを補間し、逆ヤコビアンで
    直交座標へ射影
  - 剛性は 2×2 Gauss 積分 `Bᵀ·D·B`、ドリリング安定化 `scale = γ·G·t·A`（既定 `γ = 1e-3`）
- 💻 **実装**: `shell::{local_stiffness, shear_b_mitc4, add_drilling}`（`shell/mod.rs`）。剛床時は
  面内成分（Ux/Uy/Rz）を無効化。
- ✓ **整合性**: 膜・曲げのパッチテストを機械精度で通過（V&V #4・#5）。単純支持／固定板のたわみ収束
  ±2%（ロッキングなし、V&V #6）。文献名はコード上に明示なし（"MITC4 spec" コメントのみ）。

## 4.5 壁エレメントモデル（耐震壁）

- 📖 **根拠**: RESP-D マニュアル計算編 02「耐震壁（壁エレメントモデル）の剛性」。壁柱（BeamElement）＋
  剛梁変換の 4 節点 24 自由度モデル。
- ∑ **算定式**:
  - 壁柱の軸剛性用断面積 `a = t·lw·rebar_factor`、`rebar_factor = 1 + (E_steel/E − 1)·ps`（壁筋考慮）
  - 面内曲げ `iz = t·lw³/12·rebar_factor`、せん断形状係数は `wall_shear_shape_factor`
  - 剛梁変換 `K_24 = Aᵀ·K_col·A`（並進は平均、回転はレビ・チビタ記号で ω）
- 💻 **実装**: `wall::wall_panel`（`wall_panel.rs`）。側柱は面内両端ピン（`wall::side_column` で回転自由度を
  静縮約）。耐震壁成立判定は壁厚 ≥ 120 mm・開口周比 r0 ≤ 0.4（`wall::misc_wall::wall_is_seismic`）。
- ✓ **整合性**: doc に RESP-D 計算編 02 を明記。せん断形状係数の ξ・η 定義、壁筋比の縦横共通近似は
  [原典照合リスト](../specs/原典照合リスト.md)で要照合。壁（壁谷澤 TVLEM）の妥当性確認は実験照合が必須
  （Category B、V&V #16）。

## 4.6 パネルゾーン

- 📖 **根拠**: 添付資料『パネルゾーンの力学』（小野瀬, 2009）。フェイスモーメントからパネルせん断を算定。
- ∑ **算定式**:
  - フェイスモーメント `b_ml = ml_b − bql·dc/2` 等（梁・柱端モーメントを柱面・梁面へ換算）
  - パネルせん断 `pQc = ((b_ml + b_mr) − (cql + cqu)·db/2)/db`
  - パネルせん断応力 `τ = pQc/(dc·tp)`
  - 弾性せん断パネル剛性 `kp = G·tp·dc`
- 💻 **実装**: `springs::panel`（`panel.rs::evaluate`）。剛域近似（方式 A、零剛性で二重計上防止）と
  弾性せん断パネル（方式 B）を切替。
- ✓ **整合性**: 添付資料の数値例と照合済（ケース1 `pQc = 851.135 kN, pQb = 1702.273 kN`、V&V #7）。

## 4.7 ばね要素（免震・制振・独立せん断・節点ばね）

- 📖 **根拠**: RESP-D マニュアル「05 非線形モデル」（免震支承材）、「07 非線形解析（動的解析）」
  （制振要素・歪依存特性）、計算編 03「応力解析」（節点ばね）。
- ∑ **算定式**:
  - 免震支承（積層ゴム）: 各方向独立バイリニア。摩擦支承 `Qmax = μ·N`（長期軸力）を 2 次元摩擦で射影。
    歪依存 `Qd(γ) = Qd0·CQd(γ)`、`K2(γ) = K2_0·CKd(γ)`
  - マクスウェルダンパー: バネ Kd と粘性ダッシュポット `Fc = C0·sign(V)·|V|^α` の直列（後退 Euler、
    線形 α=1 は閉形式、非線形はスカラー Newton）
  - 独立せん断ばね `k_shear = G·As/L`、節点ばね（6 成分独立）
- 💻 **実装**: `springs::{isolator, damper, shear_spring, spring}`。詳細な設計式（等価剛性・等価減衰・
  製品別係数）は[10. 免震・制振](./10_免震制振.md)。
- ✓ **整合性**: 各要素の doc に RESP-D の該当節を明記。マクスウェル要素は `K_eff = Kd·C0/(C0 + Δt·Kd)`
  をテストで照合。

## 4.8 座標変換

- 📖 **根拠**: 力学（局所-全体座標変換）。
- ∑ **算定式**: フレーム要素は `ex = (pj − pi)/L`、参照ベクトルを直交化して ey・ez を構成、
  `K_global = Rᵀ·K_local·R`。
- 💻 **実装**: `common::transform`（`transform.rs`）。シェルは規約が逆で `K_global = R·K_local·Rᵀ`。
- ✓ **整合性**: 座標変換回帰テスト（`test_beam_to_global_transverse_uses_correct_inertia`）で検証。
