# 4. 要素の定式化・剛性

この章では、各有限要素の剛性マトリクス、質量マトリクス、幾何剛性、内力回復の定式化をまとめる。フレーム要素はティモシェンコ梁理論、板は MITC4 シェルによる。剛域は RC規準の剛域規定（技術基準解説書の運用）、スラブ協力幅は RC規準8条、雑壁の断面性能算入は等価断面の力学（平行軸の定理）に基づく。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

---

## 4.1 ティモシェンコ梁要素（12×12）

せん断変形を考慮したティモシェンコ梁理論に基づき、フレーム要素の 12×12 剛性を算定する。力学の閉形式であり、[V&V](../v_and_v/README.md)で片持ち先端たわみ δ = 0.417292 mm 等により検証している。

**算定式**:
- せん断パラメータ `φz = 12·E·Iz/(G·As_y·L²)`、`φy = 12·E·Iy/(G·As_z·L²)`
- 曲げ係数 `az = E·Iz/((1+φz)·L³)`、`ay = E·Iy/((1+φy)·L³)`
- 軸 `k = E·A/L`、ねじり `k = G·J/L`
- 曲げ剛性 4×4 ブロック（例, xy 面）: `12·az`、`6·az·L`、`(4+φz)·az·L²`、`(2−φz)·az·L²`

**実装**：`squid_n_element::frame::beam`（`beam/mod.rs::local_stiffness_raw`）が算定する。せん断有効断面積が 0 のときは `rect_shear_area(A) = 5A/6` にフォールバックする。組立順は「可撓長で raw 剛性 → 端部ばね静縮約 → 剛域変換」とする。

**整合性**：`φ = 0`（せん断変形無視）で Bernoulli 梁に収束することをテスト（`test_phi_zero_converges_to_bernoulli`）で確認している。せん断補正係数は矩形で 6/5 である。

### 4.1.1 端部条件（ピン・半剛）の静縮約

端部のピンや半剛接を表すため、自由度縮約 `K* = Kaa − Kab·Kbb⁻¹·Kba` により内部端の自由度を消去する。根拠は力学である。

**算定式**：端部回転ばね剛性を Fixed = `1e8·rot_scale`、Pinned = 0、SemiRigid = `k_theta` とし、18 自由度（外部 12 + 内部端 6）を 12 自由度へ静縮約する。ここで `rot_scale = E·max(Iz,Iy)/max(L,1)` である。

**実装**：`beam/mod.rs::condense_end_springs` が算定する。剛接ペナルティ係数は 1e8 とする（1e12 だと Iz が大きいとき数値破綻するため）。

**整合性**：ピン端でモーメントが解放されることをテスト（`test_pinned_end_releases_moment`）で確認している。

### 4.1.2 剛域

部材端の剛域を扱うため、可撓長 `L' = L − λi − λj` で剛性を求めたうえで剛域を剛体変換する。剛域長は RC規準の剛域規定（部材フェイスから部材せいの 1/4 内側までを剛域とする取り方）に基づく。

**算定式**（剛域長 λ）:
- RC/SRC 造: `λ = reduction·(D_直交/2 − D_自身/4)`（負は 0）
- S・CFT 造: `λ = D_直交/2`（D_自身/4 の控除・reduction 乗算なし）
- 危険断面フェイス距離 `face = D_直交/2`
- `D_直交` は仕口部で直交する **RC/SRC 系** 部材の最大せい（λ 用）。仕口に
  接続する部材がすべて S 系なら剛域長は 0 とする。face 用の `D_直交` は
  幾何量として構造種別を問わない

**実装**：剛域長を `beam/rigid_zone.rs::auto_rigid_zones` が求め、変換を `beam/mod.rs::apply_rigid_zone_transform` が行う。直交判定は軸内積 < 0.707（概ね 45°以上）とする。

**整合性**：標準式との照合は 🔶（V&V #2）である。剛域を持つ部材の軸剛性は l0/l 補正により EA/l（節点間長基準）を保ち、剛域は曲げのみ剛とする（軸断面積の l0/l 補正）。

### 4.1.3 幾何剛性・整合質量

P-Δ 効果の線形化幾何剛性と Hermite 梁の一貫質量を算定する。根拠は力学であり、[V&V](../v_and_v/README.md)で固有値・座屈傾向により検証している。

**算定式**:
- 幾何剛性 `Kg`: `c = N/L`、係数 `6/5`、`L/10`、`2L²/15`、`−L²/30`（標準ビーム幾何剛性）
- 整合質量: `c2 = m/420`、係数 `156, 54, 22, 13, 4, 3`（標準整合質量）

**実装**：`beam/mod.rs::{geometric_stiffness, mass_matrix}` が算定する。集中質量は並進自由度に m/2 を与える。

**整合性**：標準的な梁の幾何剛性・整合質量係数に一致する。

### 4.1.4 剛性計算上の付加（協力幅・壁エレメント上下梁・雑壁）

次の剛性補正を実装している。

- **スラブ協力幅による強軸曲げ剛性増大**（RC大梁）: 隣接する平行梁との内法距離
  a（軸間距離から自梁・相手梁の幅の半分ずつを控除）から片側協力幅 `ba` を
  `a < L/2` で `ba = (0.5 − 0.6a/L)·a`、`a ≥ L/2` で `ba = 0.1L` として求め、
  T 形断面の Ie による `(Ie/I0)` 倍とする。根拠は RC規準 8 条。
  実装 `beam/mod.rs::slab_stiffness_factor`。
- **壁エレメントモデル上下大梁の剛性倍率**: 既定 100 倍（`WALL_GIRDER_STIFF_FACTOR`。
  壁の剛性を四隅節点へ伝えるための実務的モデル化）。
- **フレーム内雑壁（袖壁・腰壁・垂壁）**: 平行軸の定理で周辺部材の断面性能に合成
  （`compose`）。せん断は `As += Σaw/1.2`。「腰壁・垂壁のヤング係数は母材と同じ」
  の仮定によるため、対象はコンクリート系（RC/SRC）部材のみとする（S 造部材への
  無換算合成はしない）。実装 `beam/mod.rs`（袖壁・腰壁処理）。

## 4.2 トラス（ブレース）要素

軸剛性のみをもつブレースを `KB = E·A/L` で算定する（トラス要素の力学）。

**算定式**：`ka = factor·E·A/L` とする。引張専用ブレースは弾性解析で剛性を 1/2、弾塑性解析で初期剛性を 1 倍とし、圧縮スラック側は残留剛性比 1e-6 とする。

**実装**：`frame::truss`（`truss.rs`）が算定する。

**整合性**：引張専用ブレースの弾性 1/2 化・弾塑性スラック側の残留剛性比は実装 doc に明記している。

## 4.3 部材荷重（等価節点力・固定端内力）

部材に作用する分布荷重を節点力へ変換し、固定端内力を算定する。根拠は力学で、Hermite 形状関数による等価節点力（consistent load）と両端固定梁の固定端内力に基づく。

**算定式**:
- 等価節点力 `Q = ∫ Nᵀ·w dx`（軸は線形形状関数、曲げは Hermite 形状関数、分布荷重は 3 点 Gauss 積分）
- 固定端内力 `fFEF = −Q`（等価節点力の符号反転）に span 内力を重畳

**実装**：`frame::member_load`（`member_load.rs::{consistent_load_local, fixed_internal_local}`）が算定する。

**整合性**：UDL 固定端モーメント `wL²/12`、中央集中 `PL/8`、片側三角形 `wL²/30`(i)・`wL²/20`(j) をテストで照合している（CMQ の力学的閉形式）。

## 4.4 MITC4 シェル要素

せん断ロッキングを回避する混合補間法 MITC4（Mixed Interpolation of Tensorial Components）で板要素を算定する。膜、曲げ、せん断、ドリリング安定化を含む。

**算定式**:
- 膜 `Dm = E·t/(1−ν²)·[...]`、曲げ `Db = E·t³/(12(1−ν²))·[...]`、せん断 `Ds = (5/6)·G·t·I`
  （せん断補正係数 5/6）
- MITC4 せん断補間: タイング点 A(0,+1)/B(−1,0)/C(0,−1)/D(+1,0) の共変ひずみを補間し、逆ヤコビアンで
  直交座標へ射影
- 剛性は 2×2 Gauss 積分 `Bᵀ·D·B`、ドリリング安定化 `scale = γ·G·t·A`（既定 `γ = 1e-3`）

**実装**：`shell::{local_stiffness, shear_b_mitc4, add_drilling}`（`shell/mod.rs`）が算定する。剛床時は面内成分（Ux/Uy/Rz）を無効化する。

**整合性**：膜・曲げのパッチテストを機械精度で通過している（V&V #4・#5）。単純支持／固定板のたわみ収束は ±2%（ロッキングなし、V&V #6）である。文献名はコード上に明示がなく、"MITC4 spec" コメントのみである。

## 4.5 壁エレメントモデル（耐震壁）

耐震壁を壁柱（BeamElement）と剛梁変換による 4 節点 24 自由度モデルで算定する（壁エレメント置換モデル。一貫構造計算で標準的に用いられる実務的モデル化）。

**算定式**:
- 壁柱の軸剛性用断面積 `a = t·lw·rebar_factor`、`rebar_factor = 1 + (E_steel/E − 1)·ps`（壁筋考慮）
- 面内曲げ `iz = t·lw³/12·rebar_factor`、せん断形状係数は `wall_shear_shape_factor`
- 剛梁変換 `K_24 = Aᵀ·K_col·A`（並進は平均、回転はレビ・チビタ記号で ω）

**実装**：`wall::wall_panel`（`wall_panel.rs`）が算定する。側柱は面内両端ピンとし、`wall::side_column` で回転自由度を静縮約する。耐震壁成立判定は壁厚 ≥ 120 mm・開口周比 r0 ≤ 0.4 とする（`wall::misc_wall::wall_is_seismic`）。

**整合性**：せん断形状係数の ξ・η 定義、壁筋比の縦横共通近似は[原典照合リスト](../specs/原典照合リスト.md)で要照合である。壁（壁谷澤 TVLEM）の妥当性確認には実験照合が必須である（Category B、V&V #16）。

なお剛性計算用の開口低減率は `r = 1 − 1.25·r0`（`r0 = √(開口面積比)` の開口周比）
であり、複数開口は建物一律の設定（等価開口／包絡／自動判定）で単一の等価開口に
統合してから適用する。耐震壁の成立判定（壁厚 ≥ 120 mm・r0 ≤ 0.4・スリット無し）も
同じ開口情報から行う。RC規準18条の検定用低減 `r = min(γ1, γ2, γ3)` とは別式であり
混用しない（[6. 一次設計](./06_一次設計.md)参照）。

## 4.6 パネルゾーン

パネルゾーンのせん断を、フェイスモーメントから算定する。根拠は添付資料『パネルゾーンの力学』（小野瀬, 2009）である。

**算定式**:
- フェイスモーメント `b_ml = ml_b − bql·dc/2` 等（梁・柱端モーメントを柱面・梁面へ換算）
- パネルせん断 `pQc = ((b_ml + b_mr) − (cql + cqu)·db/2)/db`
- パネルせん断応力 `τ = pQc/(dc·tp)`
- 弾性せん断パネル剛性 `kp = G·tp·dc`

**実装**：`springs::panel`（`panel.rs::evaluate`）が算定する。剛域近似（方式 A、零剛性で二重計上防止）と弾性せん断パネル（方式 B）を切り替える。

**整合性**：添付資料の数値例と照合済である（ケース1 `pQc = 851.135 kN, pQb = 1702.273 kN`、V&V #7）。

## 4.7 ばね要素（免震・制振・独立せん断・節点ばね）

免震支承材、制振要素、独立せん断ばね、節点ばねを算定する。各特性は製品技術資料（メーカー公表値）と力学モデル（バイリニア・摩擦・マクスウェル要素）に基づく（Category B）。

**算定式**:
- 免震支承（積層ゴム）: 各方向独立バイリニア。摩擦支承 `Qmax = μ·N`（長期軸力）を 2 次元摩擦で射影。
  歪依存 `Qd(γ) = Qd0·CQd(γ)`、`K2(γ) = K2_0·CKd(γ)`
- マクスウェルダンパー: バネ Kd と粘性ダッシュポット `Fc = C0·sign(V)·|V|^α` の直列（後退 Euler、
  線形 α=1 は閉形式、非線形はスカラー Newton）
- 独立せん断ばね `k_shear = G·As/L`、節点ばね（6 成分独立）

**実装**：`springs::{isolator, damper, shear_spring, spring}` が算定する。詳細な設計式（等価剛性・等価減衰・製品別係数）は[10. 免震・制振](./10_免震制振.md)を参照。

**整合性**：マクスウェル要素は `K_eff = Kd·C0/(C0 + Δt·Kd)` をテストで照合している。

## 4.8 座標変換

局所座標系の剛性を全体座標系へ変換する。根拠は力学（局所-全体座標変換）である。

**算定式**：フレーム要素は `ex = (pj − pi)/L` とし、参照ベクトルを直交化して ey・ez を構成し、`K_global = Rᵀ·K_local·R` で変換する。

**実装**：`common::transform`（`transform.rs`）が算定する。シェルは規約が逆で `K_global = R·K_local·Rᵀ` とする。

**整合性**：座標変換の回帰テスト（`test_beam_to_global_transverse_uses_correct_inertia`）で検証している。
