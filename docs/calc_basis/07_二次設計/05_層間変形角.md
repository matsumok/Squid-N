# 7.5 層間変形角

建築基準法施行令 第82条の2 に基づき、各層の層間変形角が 1/200 以下（帳壁等の変形追従性が確認できれば 1/120 まで緩和）であるかを確認する。

**算定式**：\\( \text{層間変形角} = \delta/h \le 1/200 \\)。
層間変位 δ には柱の最大値（`secondary::stiffness_ratio::max_column_drift`）を用いる。

**実装**：`holding_capacity::check_story_drift` が判定する。

**整合性**：1/200（緩和 1/120）は[原典照合リスト](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md)に令82条の2 として登録している（☐）。
層間変形角には柱の最大値を、剛性率の算定には重心変位を用いる実務的取扱いとする。
なお実際の判定は `squid-n-app::summary` が `Model::stress_cfg.drift_limit_denom`（既定 200、緩和時 120 を設定）で行う（`check_holding_capacity` 系の 1/200 固定関数は参考実装）。
剛性率 Rs の算定では、層間変位が 0 の層（剛体的な層）は平均から除外し Rs を有限層の最大値以上とする（\\( K_s = 0 \\) とみなす逆転を防ぐ）。
