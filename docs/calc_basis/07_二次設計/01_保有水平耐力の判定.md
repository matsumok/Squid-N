# 7.1 保有水平耐力の判定

建築基準法施行令 第82条の3 に基づき、各層の保有水平耐力 Qu が必要保有水平耐力 Qun を上回るか（\\( Q_u \ge Q_{un} = D_s \cdot F_{es} \cdot Q_{ud} \\)）を判定する。

**算定式**

必要保有水平耐力:

\\[ Q_{un} = D_s \cdot F_{es} \cdot Q_{ud} \\]

- 保有水平耐力 `Qu` = プッシュオーバー最終点の層せん断力（機構形成時のベースシア）
- 設計用地震層せん断力 `Qud` = \\( C_0 = 1.0 \\) の Ai 分布層せん断力（`ai_distribution(..., c0=1.0)`）

**実装**：`secondary::holding_capacity`（`holding_capacity.rs::{check_holding_capacity, qud_by_story}`）が判定する。
Qu にはプッシュオーバー（[5.7](../05_構造解析/07_プッシュオーバー解析.md#57-プッシュオーバー解析漸増静的)）の層せん断を用いる。

**整合性**：doc に令82条の3 と仕様§2 を明記している。
Qud には二次設計として \\( C_0 = 1.0 \\) を適用する。
