# 7.3 部材ランク（FA〜FD）

昭和55年建設省告示第1792号と学会規準（RC 靭性、S 幅厚比）に基づき、部材ランク FA〜FD を判定する。
RC はせん断余裕度で、S は幅厚比で判定する。

**算定式**
- RC 部材ランク: \\( r = Q_{su}/Q_{mu} \\)、\\( r \ge 1.3 \\) → FA / \\( \ge 1.1 \\) → FB / \\( \ge 1.0 \\) → FC / 他 → FD
- S 部材ランク: 構造規定表の幅厚比限界値（部材種別・鋼種級別）で判定（正式版
  `s_member_rank_by_kihon`）。簡易版は \\( F = 235 \\) 基準値 9.0/11.0/13.0 に \\( \sqrt{235/F} \\) を乗じてスケーリング

**実装**：`secondary::member_rank` が部材ランクを判定し、幅厚比は `secondary::width_thickness` が算定する。

**整合性**：S 造幅厚比の部材ランク表（H形フランジ・ウェブ／円形／角形×鋼種、FA/FB/FC 限界値）は 2007年版技術基準解説書の構造規定表と照合済である（[原典照合リスト](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md)☑）。
鋼種級（400/490 級）は鋼種名の数値部（引張強さ表記）で判定し、F 値による判定は名称が解決できない場合のフォールバックとする（SS490 等 F 値と引張強さ級が食い違う鋼種への対応）。
RC せん断余裕度の境界 1.3/1.1/1.0 は要照合である（☐）。

## 7.3.1 幅厚比

部材の幅厚比を、2007年版技術基準解説書（構造規定）の幅厚比規定に基づいて算定する。

**算定式（代表最大幅厚比）**
- H形 \\( \max(b/2t_f, (h-2t_f)/t_w) \\)、箱形 \\( \max((h-2t)/t, (b-2t)/t) \\)、溝形 \\( \max(b/t_f, (h-2t_f)/t_w) \\) ほか
- 限界値表（例, H形フランジ柱400級 FA 9.5/FB 12.0/FC 15.5）

**実装**：`width_thickness::{max_width_thickness, s_member_rank_by_kihon}` が算定する。

**整合性**：限界値表は構造規定表と照合済である。
ただし簡易版の `max_width_thickness` は AIJ 精算式そのものではない旨を doc に明記している。
