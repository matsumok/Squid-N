# 7.2 構造特性係数 Ds

昭和55年建設省告示第1792号 第1 に基づき、架構形式と部材ランクに応じて構造特性係数 Ds を定める。

**算定式（Ds 値表）**
- RC ラーメン: FA 0.30 / FB 0.35 / FC 0.40 / FD 0.45
- RC 耐震壁付: 0.35 / 0.40 / 0.45 / 0.55
- S ラーメン: FA 0.25 / FB 0.30 / FC 0.35 / FD 0.40
- S ブレース付: 0.30 / 0.35 / 0.40 / 0.50

**実装**：`holding_capacity::ds_value` が値表を引き、層ごとの Ds は `secondary::member_rank::story_ds` が算定する（代表ランクは worst をとり、崩壊機構補正として層崩壊や部分崩壊では1段階不利に扱う）。

**整合性**：doc に告示1792号を明記している。
ただし実際の Ds は、S 造では筋かい水平力分担率 βu によって変動し、RC 耐震壁付では壁量によって表の行が変わる（本実装の表は代表値である）。
βu 依存と壁量変動は[原典照合リスト 優先度1](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md#優先度1ds構造特性係数-最重要形が複雑)に最重要照合項目として登録している（☐）。
