# 4. 要素の定式化・剛性

この章では、各有限要素の剛性マトリクス、質量マトリクス、幾何剛性、内力回復の定式化をまとめる。
フレーム要素はティモシェンコ梁理論、板は MITC4 シェルによる。
剛域は RC規準の剛域規定（技術基準解説書の運用）、スラブ協力幅は RC規準8条、雑壁の断面性能算入は等価断面の力学（平行軸の定理）に基づく。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。
