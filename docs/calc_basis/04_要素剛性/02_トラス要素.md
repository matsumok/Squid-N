# 4.2 トラス（ブレース）要素

軸剛性のみをもつブレースを次式で算定する（トラス要素の力学）。

\\[ K_B = E \cdot A/L \\]

軸剛性のみを持ち、曲げ・せん断・ねじりはゼロとする。

**実装**：`frame::truss`（`truss.rs`）が算定する。

## 引張専用ブレースの扱い

引張専用ブレース（`ElementKind::Brace { tension_only: true }`）は要素側では特別扱いせず、一般ブレースと同じ全剛性 \\( E\,A/L \\) のトラス要素として扱う。圧縮側の無効化は線形応力解析の active-set 反復で実現する。計算条件 `StressAnalysisCfg::tension_only_iteration` で切り替える。

- **一括解析（既定, `false`）**：引張専用ブレースも全剛性 \\( E\,A/L \\) のまま 1 回だけ解く。圧縮側でも軸力を負担する。
- **active-set 反復（真の引張専用, `true`）**：各反復で圧縮側（軸伸び δ < 0）に入った引張専用ブレースの軸剛性を残留比 1e-6 まで縮小して無効化する。無効化ブレースの節点変位から求めた軸伸びが引張へ転じれば再び active に戻す。active 集合が前反復と一致した時点で収束とみなす（`statics::linear` の `solve_tension_only_iterative`）。収束後は active な引張ブレースのみが \\( N = E\,A/L \cdot \delta \\) を負担し、圧縮ブレースの軸力はほぼ 0 となる。
