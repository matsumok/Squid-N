# 4.3 部材荷重（等価節点力・固定端内力）

部材に作用する分布荷重を節点力へ変換し、固定端内力を算定する。
根拠は力学で、Hermite 形状関数による等価節点力（consistent load）と両端固定梁の固定端内力に基づく。

**算定式**:
- 等価節点力 \\( Q = \int N^T \cdot w~dx \\)（軸は線形形状関数、曲げは Hermite 形状関数、分布荷重は 3 点 Gauss 積分）
- 固定端内力 \\( f_{FEF} = -Q \\)（等価節点力の符号反転）に span 内力を重畳

**実装**：`frame::member_load`（`member_load.rs::{consistent_load_local, fixed_internal_local}`）が算定する。

**整合性**：UDL 固定端モーメント \\( wL^2/12 \\)、中央集中 \\( PL/8 \\)、片側三角形 \\( wL^2/30 \\)(i)・\\( wL^2/20 \\)(j) をテストで照合している（CMQ の力学的閉形式）。
