# 2.3 材料強度（F値・許容応力度の基礎）

> 許容応力度の検定式そのものは[6. 一次設計](../06_一次設計/README.md)、終局耐力は[8. 部材終局耐力](../08_部材終局耐力/README.md)に
> 記載する。ここでは材料強度定数の根拠を示す。

材料強度定数は、建築基準法施行令89〜91条・96条・98条、および告示（鋼材の F 値等）に基づく。
詳細な埋込値と条番号の対応は[原典照合リスト 優先度2](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md#優先度2許容応力度特に鉄筋の径区分)を参照する。

**算定式**（主要な材料強度）:
- 鋼材 F 値（令98条／告示）: SN400/SS400 = 235（\\( t \le 40 \\)）、SN490/SM490 = 325 ほか
- 鉄筋降伏強度（令90条・JIS G3112）: SD295 = 295、SD345 = 345、USD685 = 685 ほか
- コンクリート設計基準強度 Fc（令91条）: 許容応力度は長期 \\( F_c/3 \\)、短期 \\( 2F_c/3 \\)
- ヤング係数比 n（Fc 区分）: \\( F_c \le 27 \to 15 \\) / \\( \le 36 \to 13 \\) / \\( \le 48 \to 11 \\) / \\( \le 60 \to 9 \\) / \\( \text{それ超} \to 7 \\)

**実装**: `squid_n_design_jp::material_strength`（`material_strength.rs`）が算定し、ヤング係数比 n は `rc::young_ratio_n` が算定する。
既定鉄筋強度は `squid-n-app` で SD345（主筋）・SD295（帯筋）とする。

**整合性**: ヤング係数比 n は RC規準のヤング係数比区分に、USD685 の許容応力度は大臣認定に基づく製品規格値（Category B）による（[原典照合リスト](https://github.com/hrntsm/squid-n/blob/main/specs/原典照合リスト.md)☑）。
鋼材・コンクリートの許容応力度の条番号対応は[6. 一次設計](../06_一次設計/README.md)を参照する。
