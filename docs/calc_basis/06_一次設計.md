# 6. 一次設計（許容応力度計算）

本章は、材料の許容応力度、および RC・S・SRC・CFT 部材の断面検定（曲げ・せん断・軸力・付着・接合部・座屈長さ）の根拠をまとめる。上位根拠は建築基準法施行令 第82条（許容応力度等計算）であり、部材の許容応力度は令第89〜91条・98条に、断面算定は各学会規準（RC規準・鋼構造設計規準）に依拠する。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

実装は `squid_n_design_jp` にある。材料強度は `material_strength.rs` が担い、突合の全体は[断面検定_参照実装照合](../v_and_v/断面検定_参照実装照合.md)を参照する。

## 6.1 材料の許容応力度

### 6.1.1 コンクリート

コンクリートの許容応力度を建築基準法施行令 第91条に基づいて算定する。

**算定式**

- 許容圧縮応力度 `fc = 長期 Fc/3、短期 = 長期×2`
- 許容せん断応力度 `fs = 長期 min(Fc/30, 0.49 + Fc/100)、短期 = 長期×1.5`
- 付着（異形鉄筋）`fa = 長期上端筋 min(Fc/15, 0.9 + (2/75)Fc)、その他 min(Fc/10, 1.35 + Fc/25)、短期×1.5`
- 軽量コンクリート1種・2種は許容応力度 0.9 倍

**実装**：`material_strength::{concrete_allowable_compression, concrete_allowable_shear, concrete_allowable_bond, concrete_class_factor}` が算定する。

**整合性**：doc に令91条・RC規準を明記している。せん断は圧縮の×2 と異なり、短期は×1.5 とする点に注意する。

### 6.1.2 鉄筋（径区分に注意）

鉄筋の許容応力度を建築基準法施行令 第90条（表）に基づいて算定する。異形鉄筋の許容引張応力度には径区分がある。

**算定式（許容引張、長期／短期）**

- SR235 = 155/235、SD295 = 195/295、SD345/390/490 は `D ≥ 29 → 195/`、`D < 29 → 215/`（短期は 345/390/490）
- USD685 = 215/685（径によらず）
- せん断補強筋: 長期は SR235 のみ 155、他は 195。短期は基準強度（SR235 = 235、
  SR295/SD295 = 295、SD345 = 345、SD390 = 390）。SD490 の短期はせん断のみ F = 390 に頭打ち

**実装**：`material_strength::{rebar_allowable_tension, rebar_allowable_shear}` が算定する。

**整合性**：SD345 の D29 以上の径区分（215 → 195）を実装している。USD685 の値は大臣認定に基づく製品規格値（Category B）である（[原典照合リスト](../specs/原典照合リスト.md)☑）。令90条表の径区分は[原典照合リスト 優先度2](../specs/原典照合リスト.md#優先度2許容応力度特に鉄筋の径区分)で要照合とする。

### 6.1.3 高強度せん断補強筋（pw 上限）

高強度せん断補強筋の pw 上限を、各製品の大臣認定・製品技術資料の製品別 pw 上限（Category B）に基づいて定める。

**算定式（pw 上限、短期・安全確保）**：KH785 = min(1.2%, 1.0%·Fc/27)、KH685/SPR685 = min(1.2%, 1.2%·Fc/27)、SHD685/MK785 = 1.2% 固定、ウルボン系 = 1.0〜1.2% ほか（製品別表）。w_ft は長期 195、短期は SBPD1275 = 585・他 590 とする。

**実装**：`material_strength::{high_strength_pw_cap, high_strength_w_ft}` が算定する。

**整合性**：製品別表はユーザー提供の原典図と照合済み（MK785 を 1.2% 固定へ修正、[原典照合リスト](../specs/原典照合リスト.md)☑）。

### 6.1.4 鋼材（F 値・許容応力度・座屈）

鋼材の F 値・許容応力度・座屈を、建築基準法施行令 第90条・98条／告示、および鋼構造設計規準 1973 に基づいて算定する。

**算定式**

- F 値（板厚区分）: SS400/SN400 = 235(t≤40)/215、SN490/SM490 = 325/295 ほか
- 許容引張 `ft = 長期 F/1.5、短期 F`、許容せん断 `fs = 長期 F/(1.5√3)、短期 F/√3`
- 圧縮（座屈考慮）`Λ = 1500/√(F/1.5)`、`ν = 3/2 + (2/3)(λ/Λ)²`、`λ ≤ Λ: fc = F(1 − 0.4(λ/Λ)²)/ν`、`λ > Λ: fc = (18/65)F/(λ/Λ)²`

**実装**：`material_strength::{steel_f_value, steel_ft, steel_fs, steel_fc, big_lambda}` が算定する。

**整合性**：doc に令90条・98条・鋼構造設計規準 1973 を明記している。F 値表・許容応力度は[原典照合リスト](../specs/原典照合リスト.md)で要照合とする。

## 6.2 RC 断面検定

### 6.2.1 梁（RC規準13条）

RC 梁の曲げ・せん断・付着を、日本建築学会 RC規準 13条（2010年版）に基づいて検定する。

**算定式**

- 許容曲げ `MA = min(MA_t, MA_c)`、`MA_t = at·ft·j`、`MA_c = fc·Icr/xn`（中立軸 xn を弾性・ヤング係数比 n で算定）
- せん断 `α = 4/(M/(Q·d)+1)`（1.0〜2.0）、`QAL = b·j·(α·fs + 0.5·w_ft·(pw − 0.002))`
- 地震時設計用せん断 `QD = min(QD1, QD2)`、`QD1 = QL + ΣBMy/l'`、`QD2 = QL + n·QE`

**実装**：`rc::beam`（`rc/beam.rs::beam_check`）が検定し、許容せん断と地震時 QD の共通処理を `rc::mod` が担う。

**整合性**：basis 文字列は「RC 規準13条（梁の曲げ・せん断・付着）」。pw オフセットは 0.002、pw 上限は長期 0.006・短期 0.012 とする。α の退化時（Q≈0）は下限 1.0（割増なし）とする。

### 6.2.2 柱（RC規準14条）

RC 柱の軸力＋二軸曲げ＋せん断を、RC規準 14条に基づいて検定する。

**算定式**

- 許容軸力 `NA = min(fc·Ae, ft·Ae/n)`、`Ae = A + (n−1)·As`
- N-M 相関曲線を弾性・線形ひずみ仮定で構築する（圧縮縁・圧縮鉄筋・引張鉄筋の 3 条件の最小）
- 二軸曲げは、矩形が `ratio = ratio_z + ratio_y`（線形和）、円形が `(mz/ma)² + (my/ma)²`
- せん断 `α = 4/(M/(Q·d)+1)`（1.0〜1.5）。α のせん断スパン比は加力方向ごとに
  「モーメントが最大となる検定位置」の (|M|, |Q|) を採用する（強軸: |Mz|max と
  対応 |Qy|、弱軸: |My|max と対応 |Qz|）。地震時 `ΣcMy = 2·Mu`

**実装**：`rc::column`（`rc/column.rs::column_check`）が検定する。地震時 QD は軸力を考慮する `rc_column_mu_simple` が担う。

**整合性**：basis 文字列は「RC 規準14条（柱、軸力+二軸曲げ+せん断）」。

### 6.2.3 柱梁接合部（RC規準15条）

柱梁接合部のせん断を、RC規準 15条に基づいて検定する。

**算定式**

- 許容せん断 `QAj = κA·(fs − 0.5)·bj·D`、形状係数 κA = 十字10/T字7/ト字5/L字3
- 有効幅 `bj = bb + 2·bai`、`bai = max(bi/2, D/4)`
- 設計用 `Qdj = min(Qdj1, Qdj2)`、`ξ = j/(cH(1 − D/Lb))`。`Qdj1 = ΣMy/j·(1−ξ)` の
  ΣMy は**大梁の降伏モーメント**の和（略算降伏モーメント `0.9·at·σy·d`。
  弾性解析の梁端モーメントではない）

**実装**：`rc::joint`（`rc/joint.rs::rc_joint_shear_check`）が検定する。

**整合性**：有効幅 `bai = max(bi/2, D/4)` は原典図「大きい方」の規定に従い min → max へ修正した。この規定が min 採用に比べ非安全側に働く点を doc に明記している（[原典照合リスト](../specs/原典照合リスト.md)☑）。

### 6.2.4 付着・定着（RC規準1999／1991）

RC 梁の付着・定着を、RC規準1999（既定）／1991 に基づいて検定する。

**算定式（1999方式）**：必要定着長 `ldb = σt·as/(K·fb·φ)`、`K = 0.3·(C+W)/db + 0.4`（上限 2.5、C = min(あき, 3cover, 5db)）、`fb = Fc/60 + 0.6`（上端筋 0.8 倍、短期×1.5）。

**実装**：`rc::bond`（`rc/bond.rs::rc_beam_bond_check`）が検定する。

**整合性**：doc に RC規準1999 を明記し、通し筋（カットオフ無し）の仮定を明示している。

### 6.2.5 耐震壁（RC規準18条）

耐震壁のせん断を、RC規準 18条に基づいて検定する。

**算定式**：コンクリート負担 `Q1 = r·t·l·fs`、壁筋負担 `Qw = ps·t·le·w_ft`、側柱負担 Qc、`Q2 = r·(Qw + ΣQc)`、短期許容 `Qa = max(Q1, Q2)`。開口低減は `r = min(γ1, γ2, γ3)`。

**実装**：`rc::wall`（`rc/wall.rs::rc_wall_shear_check`）が検定する。

**整合性**：doc に RC規準18条を明記している。剛性計算用の開口低減 `r = 1 − 1.25·r0`（計算編02）とは別式であり、混用しない旨を明示している（[原典照合リスト](../specs/原典照合リスト.md)・`wall_opening.rs` doc）。

## 6.3 鋼構造 断面検定

鋼構造の断面検定を、鋼構造設計規準 1973 に基づいて行う。

**算定式**

- 梁: 曲げ `σb = |Mz|/Z`、横座屈許容 `fb = max(fb1, fb2) ≤ ft`（許容曲げ応力度は
  許容引張応力度を上限とする）、`fb1 = F(2/3 − (4/15)(lb/i)²/(C·Λ²))`、
  `fb2 = 89000/(lb·h/Af)`。von Mises 合成 `max(√(σb'² + 3τ²)/ft, τ/fs)`
- 横座屈修正係数 `C = 1.75 + 1.05(M2/M1) + 0.3(M2/M1)² ≤ 2.3`。横補剛により
  座屈区間 lb が部材の部分区間となる場合は、区間端モーメント比が不明なため
  安全側の C = 1.0 とする
- 柱: `σc/fc + σbX/fbX + σbY/fbY ≤ 1.0`（座屈考慮 fc）
- ブレース: 圧縮 `σc/fc`、引張 `σt/ft`

**実装**：`steel::{beam, column, brace, section}` が検定する。断面欠損を考慮した Z'、および横座屈長さ lb の解決も実装している。

**整合性**：doc に鋼構造設計規準 1973 を明記し、basis 文字列で規準名を記録している。

### 6.3.1 柱の座屈長さ係数 K

柱の座屈長さ係数 K を、鋼構造塑性設計指針 式(6.65)〜(6.67)に基づいて算定する。水平移動非拘束（sway）を前提とする。

**算定式**：`(GA·GB·x² − 36)/(6(GA+GB)) = x/tan(x)`、`x = π/K`（K ≥ 1）、`G = Σ(EI/L)_柱/Σ(EI/L)_梁`。ピン端・梁無しは G = 10 とする。

**実装**：`steel::buckling`（`buckling.rs::sway_buckling_k`）が算定する。

**整合性**：マニュアル 04 と照合済み（チャート代表値でテスト、[原典照合リスト](../specs/原典照合リスト.md)☑）。

### 6.3.2 S 造パネルゾーン

S 造パネルゾーンを、鋼構造接合部設計指針に基づいて検定する。

**算定式**：`pMy = (Ve/κ)·√(1 − n²)·Fy/√3`、形状係数 κ（H形／角形／円形）、`Ve`（有効体積）。

**実装**：`steel::panel_zone`（`panel_zone.rs::s_panel_zone_check`）が検定する。

**整合性**：原典図と照合済み（Ve·κ → Ve/κ へ修正、[原典照合リスト](../specs/原典照合リスト.md)☑）。

### 6.3.3 冷間成形角形鋼管柱の柱梁耐力比

冷間成形角形鋼管柱の柱梁耐力比を、2008年版「冷間成形角形鋼管の柱に用いる角形鋼管設計・施工マニュアル」に基づいて算定する。

**算定式**：耐力低減 `νu`（`n ≤ 0.5: 1 − 4n²/3`、`n > 0.5: 4(1−n)/3`）、要求値 `min(1.5·ΣMpb, 1.3·Mpp)`。

**実装**：`steel::cold_formed`（`cold_formed.rs`）が算定する。存在軸力は `N = NL + 1.5·NE` とする。

**整合性**：doc に 2008年版マニュアルを明記している。

## 6.4 SRC / CFT 断面検定

SRC・CFT の断面検定を、SRC規準1987（累加強度式・併用せん断式）および構造規定（技術基準解説書）に基づいて行う。

**算定式**

- SRC 梁 曲げ `MA = sMo + rMA`（鉄骨 `sMo = sZ·s_ft`＋RC `rMA = at·ft·j`）
- SRC 梁 せん断: 鉄骨部 `sQA = tw·dw·sfs` と RC 部
  `rQA = min(b·rj·(rα·fs + 0.5·pw·wft), b·rj·(2(b′/b)·fs + pw·wft))`（pw は 0.6% 上限）を、
  弾性分担（または地震時短期の構造規定方式 sQD/rQD）と比較する
- SRC 柱 N-M 相関を累加強度式で構築する（コンクリート許容応力度は鉄骨フランジ食い込み低減後 `fc' = fc(1 − 15·s_pc)`）、二軸曲げは線形和。RC 部 N-M 曲線は引張側アンカー `(rNt, 0)` を持ち、純引張耐力近傍で rM → 0 に収束する
- SRC 柱 せん断: 長期は併用式 `QA = (1+β)·b·rj·a′·fs`（`a′ = rα`（`b′/b ≥ rα/3`）または `3b′/b`、`1 ≤ rα ≤ 2`、β は鉄骨ウェブの形式と寸法による係数）を全せん断力と比較する。短期は鉄骨部 `sQA`（強軸 `dw·tw·sfs`／弱軸 `(4/3)·bf·tf·sfs`）と RC 部 `rQAS1 = b·rj·(fs + 0.5·pw·wft)`（α を含まない）・`rQAS2 = b·rj·(2(b′/b)·fs + pw·wft)` を分担 sQD/rQD と比較する
- CFT 柱は SRC 規準を CFT 断面に適用する（相互拘束効果の強度割増しは考慮せず、非拘束・安全側とする）。せん断有効断面積は方向別（`2t(H−2t)`／`2t(B−2t)`）

**実装**：`srrc::{beam, column, panel_zone}`、`cft::mod` が検定する。SRC パネルゾーンは `cV·jδ·fs·(1+β) ≧ (h'/h)(BM1+BM2)`。

**整合性**：SRC パネルゾーン検定式・jδ は原典図と照合済み（cVe → cV、h/h' → h'/h へ修正、[原典照合リスト](../specs/原典照合リスト.md)☑）。CFT の一部簡略化（QD1 未実装等）は実装 doc に明記している。
