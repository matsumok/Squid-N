# 7. 二次設計（保有水平耐力計算）

本章では、保有水平耐力 Qu と必要保有水平耐力 Qun の比較、構造特性係数 Ds、形状係数 Fes（剛性率 Fs と偏心率 Fe）、
部材ランクと幅厚比、層間変形角について、その算定根拠を示す。これらはいずれも建築基準法施行令 第82条の2〜6 と
昭和55年建設省告示第1792号を根拠とする。

> 各項目は「導入文（何を・どの基準で）、算定式、実装、整合性」の順で記す。

実装は `squid_n_design_jp::secondary` に置く。監査結果は[P7 監査](../v_and_v/p7_review.md)と
[終局検定_参照実装照合](../v_and_v/終局検定_参照実装照合.md)を参照。

---

## 7.1 保有水平耐力の判定

建築基準法施行令 第82条の3 に基づき、各層の保有水平耐力 Qu が必要保有水平耐力 Qun を上回るか
（`Qu ≧ Qun = Ds·Fes·Qud`）を判定する。

**算定式**
- 必要保有水平耐力 `Qun = Ds·Fes·Qud`
- 保有水平耐力 `Qu` = プッシュオーバー最終点の層せん断力（機構形成時のベースシア）
- 設計用地震層せん断力 `Qud` = C0 = 1.0 の Ai 分布層せん断力（`ai_distribution(..., c0=1.0)`）

**実装**：`secondary::holding_capacity`（`holding_capacity.rs::{check_holding_capacity, qud_by_story}`）が判定する。
Qu にはプッシュオーバー（[5.7](./05_構造解析.md#57-プッシュオーバー解析漸増静的)）の層せん断を用いる。

**整合性**：doc に令82条の3 と仕様§2 を明記している。Qud には二次設計として C0 = 1.0 を適用する。

---

## 7.2 構造特性係数 Ds

昭和55年建設省告示第1792号 第1 に基づき、架構形式と部材ランクに応じて構造特性係数 Ds を定める。

**算定式（Ds 値表）**
- RC ラーメン: FA 0.30 / FB 0.35 / FC 0.40 / FD 0.45
- RC 耐震壁付: 0.35 / 0.40 / 0.45 / 0.55
- S ラーメン: FA 0.25 / FB 0.30 / FC 0.35 / FD 0.40
- S ブレース付: 0.30 / 0.35 / 0.40 / 0.50

**実装**：`holding_capacity::ds_value` が値表を引き、層ごとの Ds は `secondary::member_rank::story_ds` が算定する
（代表ランクは worst をとり、崩壊機構補正として層崩壊や部分崩壊では1段階不利に扱う）。

**整合性**：doc に告示1792号を明記している。ただし実際の Ds は、S 造では筋かい水平力分担率 βu によって変動し、
RC 耐震壁付では壁量によって表の行が変わる（本実装の表は代表値である）。βu 依存と壁量変動は
[原典照合リスト 優先度1](../specs/原典照合リスト.md#優先度1ds構造特性係数-最重要形が複雑)に最重要照合項目として登録している（☐）。

## 7.3 部材ランク（FA〜FD）

昭和55年建設省告示第1792号と学会規準（RC 靭性、S 幅厚比）に基づき、部材ランク FA〜FD を判定する。
RC はせん断余裕度で、S は幅厚比で判定する。

**算定式**
- RC 部材ランク: `r = Qsu/Qmu`、r ≥ 1.3 → FA / ≥ 1.1 → FB / ≥ 1.0 → FC / 他 → FD
- S 部材ランク: 構造規定表の幅厚比限界値（部材種別・鋼種級別）で判定（正式版
  `s_member_rank_by_kihon`）。簡易版は F = 235 基準値 9.0/11.0/13.0 に √(235/F) を乗じてスケーリング

**実装**：`secondary::member_rank` が部材ランクを判定し、幅厚比は `secondary::width_thickness` が算定する。

**整合性**：S 造幅厚比の部材ランク表（H形フランジ・ウェブ／円形／角形×鋼種、FA/FB/FC 限界値）は
2007年版技術基準解説書の構造規定表と照合済である（[原典照合リスト](../specs/原典照合リスト.md)☑）。
鋼種級（400/490 級）は鋼種名の数値部（引張強さ表記）で判定し、F 値による判定は名称が
解決できない場合のフォールバックとする（SS490 等 F 値と引張強さ級が食い違う鋼種への対応）。
RC せん断余裕度の境界 1.3/1.1/1.0 は要照合である（☐）。

### 7.3.1 幅厚比

部材の幅厚比を、2007年版技術基準解説書（構造規定）の幅厚比規定に基づいて算定する。

**算定式（代表最大幅厚比）**
- H形 `max(b/2tf, (h−2tf)/tw)`、箱形 `max((h−2t)/t, (b−2t)/t)`、溝形 `max(b/tf, (h−2tf)/tw)` ほか
- 限界値表（例, H形フランジ柱400級 FA 9.5/FB 12.0/FC 15.5）

**実装**：`width_thickness::{max_width_thickness, s_member_rank_by_kihon}` が算定する。

**整合性**：限界値表は構造規定表と照合済である。ただし簡易版の `max_width_thickness` は AIJ 精算式そのものではない旨を
doc に明記している。

## 7.4 形状係数 Fes（剛性率 Fs・偏心率 Fe）

### 7.4.1 剛性率 Rs・Fs

建築基準法施行令 第82条の6 と告示第1792号に基づき、剛性率 Rs（規定は Rs ≥ 0.6）から Fs を算定する。

**算定式**：`Ks = h/δ`、`Rs = Ks/mean(Ks)`。`Fs = Rs ≥ 0.6 → 1.0、Rs < 0.6 → 2.0 − Rs/0.6`。

**実装**：`holding_capacity::{stiffness_ratios, fs}` が算定する。剛性率用の層間変位には重心変位 δg
（`secondary::stiffness_ratio::cog_story_drifts`）を用いる。

**整合性**：Fs 式は[原典照合リスト](../specs/原典照合リスト.md)に告示1792号として登録している（☐）。

### 7.4.2 偏心率 Re・Fe

建築基準法施行令 第82条の6 と告示第1792号に基づき、偏心率 Re（規定は Re ≤ 0.15）から Fe を算定する。
剛心と弾力半径は武藤 D 値法で求める。

**算定式**
- D 値 `D = a·Kc0`、`Kc0 = 12EIc/h³`、`a = k̄/(2+k̄)`（一般階）
- 剛心 `Xs = Σ(Dy·x)/ΣDy`、偏心距離 `ex = |Xg − Xs|`
- 弾力半径 `rex = √(KR/ΣDx)`、`KR = Σ(Dx·ȳ²) + Σ(Dy·x̄²)`
- `Re = e/re`、`Fe = Re ≤ 0.15 → 1.0、Re > 0.15 → min(1.0 + 0.5(Re − 0.15)/0.15, 1.5)`

**実装**：`secondary::eccentricity`（D 値法略算）と `secondary::eccentricity_analysis`（応力解析結果からの精算層。
剛心は解析剛性、重心は長期軸力による）、および `holding_capacity::fe` が算定する。

**整合性**：doc に令82条の6、告示1792号、武藤 D 値法を明記している。Fe の上限を 1.5 で頭打ちにする扱いは
[原典照合リスト 優先度3](../specs/原典照合リスト.md#優先度3fes形状係数の頭打ち挙動)で要照合である（☐）。
偏心率は弾性剛性（弾性解析結果）に基づいて算定する（令82条の6 の運用に基づく実務的取扱い）。

### 7.4.3 形状係数 Fes

形状係数 Fes は、剛性率による Fs と偏心率による Fe の積として算定する。

**算定式**：`Fes = Fs·Fe`。

**実装**：`holding_capacity::fes` が算定する。

## 7.5 層間変形角

建築基準法施行令 第82条の2 に基づき、各層の層間変形角が 1/200 以下（帳壁等の変形追従性が確認できれば
1/120 まで緩和）であるかを確認する。

**算定式**：`層間変形角 = δ/h ≤ 1/200`。層間変位 δ には柱の最大値（`secondary::stiffness_ratio::max_column_drift`）を用いる。

**実装**：`holding_capacity::check_story_drift` が判定する。

**整合性**：1/200（緩和 1/120）は[原典照合リスト](../specs/原典照合リスト.md)に令82条の2 として登録している（☐）。
層間変形角には柱の最大値を、剛性率の算定には重心変位を用いる実務的取扱いとする。
なお実際の判定は `squid-n-app::summary` が `Model::stress_cfg.drift_limit_denom`（既定 200、
緩和時 120 を設定）で行う（`check_holding_capacity` 系の 1/200 固定関数は参考実装）。
剛性率 Rs の算定では、層間変位が 0 の層（剛体的な層）は平均から除外し Rs を有限層の
最大値以上とする（Ks=0 とみなす逆転を防ぐ）。

## 7.6 主軸の計算

地震力方向に対する架構の主軸を、水平力のなす仕事の極値条件から計算する（実務的手法）。

**算定式**：`tan2Θ = −Pᵗ(uy+vx)/Pᵗ(vy−ux)`。仕事 W の極値から主軸角 Θ を決定する。

**実装**：`secondary::principal_axis`（`principal_axis.rs`）が計算する。

**整合性**：仕事 W の極値条件からの導出を実装 doc に明記している。
