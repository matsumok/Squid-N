# 7. 二次設計（保有水平耐力計算）

保有水平耐力 Qu と必要保有水平耐力 Qun の比較、構造特性係数 Ds、形状係数 Fes（剛性率 Fs・偏心率 Fe）、
部材ランク・幅厚比、層間変形角の根拠。建築基準法施行令 第82条の2〜6 と昭和55年建設省告示第1792号を
根拠とする。

> 記載フォーマット: 📖 根拠 / ∑ 算定式 / 💻 実装 / ✓ 整合性

実装は `squid_n_design_jp::secondary`。監査結果は[P7 監査](../v_and_v/p7_review.md)、
[終局検定_RESP-D照合](../v_and_v/終局検定_RESP-D照合.md)を参照。

## 7.1 保有水平耐力の判定

- 📖 **根拠**: 建築基準法施行令 第82条の3。`Qu ≧ Qun = Ds·Fes·Qud`。
- ∑ **算定式**:
  - 必要保有水平耐力 `Qun = Ds·Fes·Qud`
  - 保有水平耐力 `Qu` = プッシュオーバー最終点の層せん断力（機構形成時のベースシア）
  - 設計用地震層せん断力 `Qud` = C0 = 1.0 の Ai 分布層せん断力（`ai_distribution(..., c0=1.0)`）
- 💻 **実装**: `secondary::holding_capacity`（`holding_capacity.rs::{check_holding_capacity, qud_by_story}`）。
  Qu はプッシュオーバー（[5.7](./05_構造解析.md#57-プッシュオーバー解析漸増静的)）の層せん断。
- ✓ **整合性**: doc に令82条の3・仕様§2 を明記。C0 = 1.0（二次設計）を Qud に適用。

---

## 7.2 構造特性係数 Ds

- 📖 **根拠**: 昭和55年建設省告示第1792号 第1。架構形式・部材ランクにより Ds を定める。
- ∑ **算定式（Ds 値表）**:
  - RC ラーメン: FA 0.30 / FB 0.35 / FC 0.40 / FD 0.45
  - RC 耐震壁付: 0.35 / 0.40 / 0.45 / 0.55
  - S ラーメン: FA 0.25 / FB 0.30 / FC 0.35 / FD 0.40
  - S ブレース付: 0.30 / 0.35 / 0.40 / 0.50
- 💻 **実装**: `holding_capacity::ds_value`、層 Ds は `secondary::member_rank::story_ds`（代表ランクは
  worst、崩壊機構補正: 層崩壊/部分崩壊は 1 段階不利）。
- ✓ **整合性**: doc に告示1792号を明記。**留意**: 実際の Ds は S 造で筋かい水平力分担率 βu により
  変動し、RC 耐震壁付は壁量で行が変わる（本実装の表は代表値）。βu 依存・壁量変動は
  [原典照合リスト 優先度1](../specs/原典照合リスト.md#優先度1ds構造特性係数-最重要形が複雑)で最重要照合項目として登録（☐）。

## 7.3 部材ランク（FA〜FD）

- 📖 **根拠**: 昭和55年建設省告示第1792号、および学会規準（RC 靭性・S 幅厚比）。RC はせん断余裕度、
  S は幅厚比で判定。
- ∑ **算定式**:
  - RC 部材ランク: `r = Qsu/Qmu`、r ≥ 1.3 → FA / ≥ 1.1 → FB / ≥ 1.0 → FC / 他 → FD
  - S 部材ランク: 構造規定表の幅厚比限界値（部材種別・鋼種級別）で判定（正式版
    `s_member_rank_by_kihon`）。簡易版は F = 235 基準値 9.0/11.0/13.0 に √(235/F) を乗じてスケーリング
- 💻 **実装**: `secondary::member_rank`、幅厚比は `secondary::width_thickness`。
- ✓ **整合性**: S 造幅厚比の部材ランク表（H形フランジ・ウェブ／円形／角形×鋼種、FA/FB/FC 限界値）は
  RESP-D マニュアル 04 の構造規定表と照合済（[原典照合リスト](../specs/原典照合リスト.md)☑）。RC せん断余裕度
  境界 1.3/1.1/1.0 は要照合（☐）。

### 7.3.1 幅厚比

- 📖 **根拠**: RESP-D マニュアル「04 断面検定 幅厚比の検討」、2007年版技術基準解説書（構造規定）。
- ∑ **算定式（代表最大幅厚比）**: H形 `max(b/2tf, (h−2tf)/tw)`、箱形 `max((h−2t)/t, (b−2t)/t)`、
  溝形 `max(b/tf, (h−2tf)/tw)` ほか。限界値表（例, H形フランジ柱400級 FA 9.5/FB 12.0/FC 15.5）。
- 💻 **実装**: `width_thickness::{max_width_thickness, s_member_rank_by_kihon}`。
- ✓ **整合性**: 限界値表は構造規定表と照合済。簡易 `max_width_thickness` は AIJ 精算式そのものではない旨を
  doc に明記。

## 7.4 形状係数 Fes（剛性率 Fs・偏心率 Fe）

### 7.4.1 剛性率 Rs・Fs

- 📖 **根拠**: 建築基準法施行令 第82条の6、告示第1792号。剛性率 Rs ≥ 0.6 が規定。
- ∑ **算定式**: `Ks = h/δ`、`Rs = Ks/mean(Ks)`。`Fs = Rs ≥ 0.6 → 1.0、Rs < 0.6 → 2.0 − Rs/0.6`。
- 💻 **実装**: `holding_capacity::{stiffness_ratios, fs}`。剛性率用の層間変位は重心変位 δg
  （`secondary::stiffness_ratio::cog_story_drifts`）。
- ✓ **整合性**: Fs 式は[原典照合リスト](../specs/原典照合リスト.md)で告示1792号として登録（☐）。

### 7.4.2 偏心率 Re・Fe

- 📖 **根拠**: 建築基準法施行令 第82条の6、告示第1792号。偏心率 Re ≤ 0.15 が規定。武藤 D 値法で
  剛心・弾力半径を算定。
- ∑ **算定式**:
  - D 値 `D = a·Kc0`、`Kc0 = 12EIc/h³`、`a = k̄/(2+k̄)`（一般階）
  - 剛心 `Xs = Σ(Dy·x)/ΣDy`、偏心距離 `ex = |Xg − Xs|`
  - 弾力半径 `rex = √(KR/ΣDx)`、`KR = Σ(Dx·ȳ²) + Σ(Dy·x̄²)`
  - `Re = e/re`、`Fe = Re ≤ 0.15 → 1.0、Re > 0.15 → min(1.0 + 0.5(Re − 0.15)/0.15, 1.5)`
- 💻 **実装**: `secondary::eccentricity`（D 値法略算）、`secondary::eccentricity_analysis`（応力解析結果
  からの精算層、剛心=解析剛性・重心=長期軸力）。`holding_capacity::fe`。
- ✓ **整合性**: doc に令82条の6・告示1792号・武藤 D 値法を明記。Fe の上限 1.5 頭打ちは
  [原典照合リスト 優先度3](../specs/原典照合リスト.md#優先度3fes形状係数の頭打ち挙動)で要照合（☐）。
  偏心率の計算は常に弾性解析結果からとする RESP-D 規定（計算編03）を実装。

### 7.4.3 形状係数 Fes

- ∑ **算定式**: `Fes = Fs·Fe`。
- 💻 **実装**: `holding_capacity::fes`。

## 7.5 層間変形角

- 📖 **根拠**: 建築基準法施行令 第82条の2。層間変形角 ≤ 1/200（帳壁等の変形追従性が確認できれば
  1/120 まで緩和）。
- ∑ **算定式**: `層間変形角 = δ/h ≤ 1/200`。層間変位 δ は柱の最大値（`secondary::stiffness_ratio::
  max_column_drift`）。
- 💻 **実装**: `holding_capacity::check_story_drift`。
- ✓ **整合性**: 1/200（緩和 1/120）は[原典照合リスト](../specs/原典照合リスト.md)で令82条の2 として登録（☐）。
  RESP-D 計算編03 に従い、層間変形角は柱の最大値、剛性率算定は重心変位を用いる。

---

## 7.6 主軸の計算

- 📖 **根拠**: RESP-D 計算編 03「応力解析 §主軸の計算」。地震力方向に対する架構の主軸。
- ∑ **算定式**: `tan2Θ = −Pᵗ(uy+vx)/Pᵗ(vy−ux)`、仕事 W の極値から主軸角 Θ を決定。
- 💻 **実装**: `secondary::principal_axis`（`principal_axis.rs`）。
- ✓ **整合性**: doc に RESP-D 計算編03「主軸の計算」を明記。
