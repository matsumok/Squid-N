# 応力解析の RESP-D マニュアル照合レポート

**照合日:** 2026-07-11
**原典:** RESP-D 操作・計算マニュアル 計算編「03. 応力解析」
**方法:** マニュアル全節（応力解析モデル/層間変形角・剛性率/偏心率/主軸の計算/
施工時解析/段階的耐力喪失解析）と実装を突き合わせ、差異を「修正済み」「新規実装」
「既知の制約（明記あり）」に分類した。

---

## 1. 今回修正した差異

### 1.1 層間変形角が「階の全節点の平均変位差」だった

- **箇所:** `squid-n-app/src/summary.rs` `compute_story_metrics()`
- **マニュアル:** 「層間変形角は、その階の**柱の層間変形角の最大値**を用いて確認します。
  ただし、斜め柱の変形角は除外します」`1/irs = max(δ1,…,δn)/iH`
  （δ: 柱頭変位−柱脚変位（加力方向の水平変位））。
- **不具合:** 階に属する全節点の水平変位の単純平均を層代表変位としており、
  柱ごとの最大値も斜め柱の除外も無かった（ねじれを持つ建物で危険側）。
- **修正:** `squid-n-design-jp/src/story_metrics.rs` を新設。
  `max_column_drift()` が鉛直柱（|方向余弦z|>0.707）ごとの柱頭・柱脚水平変位差の
  最大値を返す。柱頭・柱脚の平面位置が 1mm を超えてずれる柱は斜め柱として除外。
  柱を拾えない層のみ従来の平均変位差にフォールバック。

### 1.2 剛性率 Rs の層間変位が重心位置の変位でなかった

- **マニュアル:** 「剛性率算出時の層間変形角は `1/irs = iδg/iH`（iδg: **重心位置**の
  層間変位）」。層間変形角の確認用（柱別最大）と剛性率用（重心位置）は別定義。
- **不具合:** 確認用と同じ「全節点平均変位差」を Rs にも使い回していた。
- **修正:** `story_metrics::cog_story_drifts()` が質量重み付き平均変位（剛床の
  並進＋回転を仮定すると重心位置の変位に厳密一致）から δg を算定し、
  `stiffness_ratios()`（Rs = rs/r̄s、r̄s は相加平均）へ渡す。

### 1.3 偏心率の剛心が D 値法（略算）のみだった

- **マニュアル:** 「剛心は、その階の柱の水平方向剛性の中心として求めます。各柱の
  水平剛性は、**地震時応力解析結果のせん断力と層間変位**により計算します。
  `ki = Qi/δi`」「各階の重心は、鉛直荷重を支持する柱の**長期荷重による軸力 N**
  およびその部材の平面座標から計算します。`gx = Σ(Ni·xi)/ΣNi`」
  「偏心率の計算は常に弾性解析結果から計算されます」。
- **従来:** 剛心＝武藤 D 値法（断面・階高・梁剛比からの略算）、重心＝節点質量。
  仕様 `specs/P7_二次設計.md` §5 に「精算は将来」と明記された既知の制約だった。
- **修正:** `eccentricity::column_stiffnesses_from_analysis()`（ki=Qi/δi）と
  `center_of_gravity_from_axial()`（長期軸力重心。圧縮分を重みとし、全柱引張時は
  質量重心へフォールバック）による精算ルート
  `story_eccentricity_from_analysis()` を追加。X/Y 加力の弾性解析結果が揃って
  いる場合は精算、無い場合は従来の D 値法。ねじり剛性 KR・弾力半径 re・Re の
  式は従来実装がマニュアルと一致（変更なし）。

## 2. 新規実装（従来は完全未実装）

### 2.1 主軸の計算

- **マニュアル:** 水平力が X 軸と角度 Θ をなすときの仕事 W が極値をとる角度
  `tan2Θ = −Pᵗ(uy+vx) / Pᵗ(vy−ux)`。
- **実装:** `squid-n-design-jp/src/principal_axis.rs`。
  `principal_axis_angle()`（ベクトル直接）と
  `principal_axis_from_results()`（X/Y 加力の `StaticOnce` から）。
  回転させた剛性系から角度が復元されることをテストで確認。

### 2.2 施工時解析（施工段階解析）

- **マニュアル:** 下層から順に 1 層ずつ部材を生成し、生成ごとに固定荷重を載荷。
  生成時の節点は初期座標（変位 0）から解析を開始し、下階の鉛直変位は累積しない。
  全部材生成・固定荷重載荷後に積載荷重を載荷。
- **実装:** `squid-n-solver/src/construction.rs`
  `construction_stage_analysis(model, dead, live)`。層 elevation 昇順にステージを
  作り、未生成節点を全自由度拘束・未生成部材を除外した部分構造へ、当該ステージで
  新規生成された節点・部材に作用する固定荷重のみを増分載荷して逐次解く。
  増分の単純和が「生成前ステージは寄与しない」規則を自動的に満たす。
  `ConstructionResult::as_static_once()` で長期解析結果の置き換えが可能。

### 2.3 段階的耐力喪失解析

- **マニュアル:** せん断降伏後、耐力喪失変形角に達した部材を両端ピン・せん断力を
  負担しない部材に変更して荷重を 0 から再載荷し、包絡線で耐力劣化曲線を得る。
  直接指定（開始・終了変形角）と FEMA による非線形特性設定（大梁の C/NC 判定＋
  塑性回転角テーブル、柱は RESP-D 自身も未対応）。
- **実装:** `squid-n-solver/src/strength_loss.rs`
  `staged_strength_loss()`。`LossCriterion::DriftRange`（終了変形角超過部材の
  発生時に開始変形角超過部材をまとめて除去）と `LossCriterion::Fema`
  （`fema_plastic_rotation()`: テーブル 4 隅の双線形補間、`fema_is_c()`:
  s≤D/3 かつ Vs≥0.75V）。除去は部材専用の縮小断面（×1e-6）＋両端ピン化。
  各パスの capacity_curve を頂部変位で包絡。
- **せん断降伏判定:** プッシュオーバーにせん断降伏イベントの記録を追加
  （`pushover::compute_shear_yield_qy`、`PushoverResult::shear_yields`）。
  材端力を要素局所座標の ey・ez へ射影し、**局所 Vy↔Qy_y・Vz↔Qy_z を方向別に
  独立判定**する。Qy の算定は
  - 鋼（fy あり）: `Qy = as·fy/√3`
  - RC（`RcRect` 形状＋fc あり）: **荒川 mean 式**
    `Qsu = {0.068·pt^0.23·(Fc+18)/(M/(Qd_e)+0.12) + 0.85·√(pw·σwy) + 0.1·σ0}·b·j`
    （`squid_n_core::rc_capacity::rc_qsu_simple`。design-jp から core へ移設し
    再エクスポートで API 互換維持）。強軸＝(b, d, main_x)、弱軸＝b/d 入替＋main_y。
  - RC（形状無し・fc のみ）: 慣用値 `as·0.7√fc` へフォールバック
  耐力喪失判定は「せん断降伏後に変形角しきい値超過」を優先し、せん断降伏
  イベントが 1 件も無いモデル（as_y/as_z 未設定等）のみ曲げ降伏代用へ
  フォールバックする。
- **精緻化（第3弾）:** 荒川式の h0 は剛域控除後の内法長さ
  （`effective_clear_span`: 節点間長 −(剛域長i+j)。非正になる異常指定は
  節点間長へフォールバック）。軸力項 σ0 は各ステップの部材軸力（圧縮のみ、
  σ0=N/(b·D)、式側で 0〜0.4Fc にクランプ）から**動的に**反映する
  （`DirThreshold::RcArakawa`）。符号規約は片持ち柱の手計算検算で確定済み。
- **既知の制約（コードに明記）:** 柱の FEMA 特性は未対応（原典と同じ）。
  RcRect 以外の RC 形状（円形等）と SRC/CFT のせん断降伏は慣用値のまま。

### 2.4 長期応力解析の軸力非負担オプション

- **マニュアル:** 「長期応力解析においては、計算条件の指定により以下の部材について
  長期軸力を負担させないことも可能です。ブレース／柱、制振間柱」。
- **実装:** `Model` の計算条件（`StressAnalysisCfg`）でブレース・柱の長期軸力
  非負担を指定可能（長期系荷重ケースのみに適用、短期には適用しない）。
  SRC/CFT 等の合成断面柱も対象（複製断面で `shape` を外し、軸剛性のみ縮小）。
  制振間柱は要素種別自体が未実装のため対象外（要素実装後に追随）。

### 2.5 雑壁の剛性評価（n 倍法）

- **マニュアル:** 剛性率・偏心率算定時の雑壁剛性 `Kw' = n·Aw'·ΣKc/ΣAc`
  （ΣAc=0 の場合は Kw'=0。n は入力値）。
- **実装:** `eccentricity::misc_wall_stiffness()`・`sum_column_area()`・
  `append_misc_wall_stiffnesses()`。`MiscWall` に壁厚属性（`thickness`、
  `None` は剛性評価の対象外）、`StressAnalysisCfg` に n 係数（`misc_wall_n`、
  `None` は雑壁剛性を考慮しない）を追加し、当該層に帰属する雑壁
  （中間高さ z が層帯域内）を方向別の等価剛性要素として剛心・ねじり剛性へ
  自動配線した（`story_eccentricity` / `story_eccentricity_from_analysis` 両ルート）。

## 3. 応力解析モデル（変形自由度）の照合

マニュアルの「部材の変形と自由度」表との対応:

| 部材 | マニュアル | 実装 | 判定 |
|---|---|---|---|
| 柱・梁 | 曲げθX,θY,θZ＋せん断γY,γZ＋軸δX | Timoshenko 梁 12 自由度（せん断・軸考慮） | 整合 |
| 壁 | θZ, γY, δX | 暫定 Beam 流用（開口低減はせん断断面積へ反映） | 既知の暫定（P5.5 で TVLEM 予定） |
| ブレース | δX のみ | トラス要素 KB=EA/L | 整合 |
| 節点バネ | θY,θZ,γY,γZ,δX（θX非考慮） | `ElementKind::NodalSpring`（2節点・零長対応、局所6自由度の独立バネ剛性 `ElementData::spring`。θX は既定 0） | 整合（新規実装） |
| 仕口パネル | γY, γZ | 剛域近似／弾性せん断パネルの2モード | 簡易近似 |

## 4. 追加実装（照合後の第2弾）

- **節点バネ要素**（§3 の表参照）: `squid-n-element/src/spring.rs`。
- **せん断降伏イベントの記録と耐力喪失判定の厳密化**（§2.3 参照）。
- **雑壁の壁厚・n 係数の入力 UI**: 雑壁テーブルに壁厚[mm]（行ごと・undo対応）と
  モデル共通の n 係数入力を追加。
- **層間変形角の緩和値の計算条件化**: `StressAnalysisCfg::drift_limit_denom`
  （原則 200、緩和時 120）。判定・UI 表示・CSV ヘッダが追従する。
- **主軸角のレポート出力**: X・Y 加力の弾性解析結果が揃っている場合、
  レポート CSV に主軸角 Θ[deg] を出力（P ベクトルは
  `Analysis::seismic_nodal_force_magnitudes`）。
- **SRC/CFT 複合断面柱への長期軸力非負担の適用**（§2.4 参照）。

- 壁要素の限定自由度モデル（TVLEM）。P5.5 スコープ。
- 制振間柱要素と、その長期軸力非負担への追随。
- 節点バネの入力 UI（データモデル・要素・解析は対応済み）。
- 荒川式 Qy の RcRect 以外の RC 形状（円形等）・SRC/CFT への拡張
  （現状は慣用値 as·0.7√fc へフォールバック）。
- 荒川 mean 式の各係数の原典照合（靭性指針/技術基準解説書等。コード内
  「要・原典照合」注記を参照）。
